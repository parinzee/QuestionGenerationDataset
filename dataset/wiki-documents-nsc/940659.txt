<doc id="940659" url="https://th.wikipedia.org/wiki?curid=940659" title="บราวนิง ไฮ-พาวเวอร์">บราวนิง ไฮ-พาวเวอร์ บราวนิง ไฮ-พาวเวอร์ () เป็นปืนพกกึ่งอัตโนมัติที่มีประสิทธิภาพและทรงพลังชนิดหนึ่งของโลก ออกแบบเมื่อปี พ.ศ. 2456 โดยจอห์น บราวนิง ผู้ผลิตและออกแบบอาวุธปืนชาวอเมริกัน ผลิตครั้งแรกในปี พ.ศ. 2475 โดยบริษัทผลิตอาวุธปืนเบลเยียมเอฟเอ็น (FN) และผลิตครั้งสุดท้ายในปี พ.ศ. 2560 ไฮพาวเวอร์ บรรจุกระสุนได้ 10 นัด มีขนาดกะทัดรัด ลักษณะของปืนได้นำปืนพกรุ่นลูเกอร์ และเอ็ม 1911 มารวมในกระบอกเดียวกัน ไฮพาวเวอร์ใช้กระสุนขนาด 9×19 มิลลิเมตร และ.40 สมิตร์แอนด์เวสสัน ผลิตออกมาหลายรุ่นคุณสมบัติคุณสมบัติ. - ตัวปืนมีขนาดกะทัดรัด - ซองกระสุนบรรจุได้ 10 นัด - ป้องกันการปืนลั่นและมีความปลอดภัย - มีความแข็งแรงและทนทาน ง่ายต่อการพกพา - ระยะยิงหวังผล 50 เมตรการนำมาใช้ การนำมาใช้. ไฮพาวเวอร์ นำมาใช้ครั้งแรกในสงครามโลกครั้งที่สอง เมื่อปี พ.ศ. 2483 และเป็นปืนพกที่นิยมใช้ในการทหารมาก่อนปี พ.ศ. 2533 จึงมีการผลิตออกมาเรื่อย ๆ โดนรุ่นปัจจุบันผลิตครั้งล่าสุดใน พ.ศ. 2560 และยังเป็นปืนพกมาตรฐานในกองทัพแคนาดา โดยใช้รุ่น MK1 นอกจากนี้ยังมีหลายประเทศผลิตปืนพกที่มีลักษณะคล้ายปืนพกรุ่นนี้ เช่น ประเทศอินโดนีเซียรุ่นปืน รุ่นปืน. บราวนิง ไฮ-พาวเวอร์ มีรุ่นที่ผลิตหลายรุ่น แต่รุ่นที่เป็นที่รู้จักกันดี เช่น- พี 35 (P-35) เป็นรุ่นที่ผลิตออกมารุ่นแรก ถูกใช้ครั้งแรกในสงครามโลกครั้งที่ 2 และเป็นรุ่นที่นิยมใช้มากที่สุดในปัจจุบัน - L9A1 ออกแบบโดยสหราชอาณาจักร เพื่อนำมาใช้ในกองทัพอากาศสหราชอาณาจักรในสงครามเย็น ทั้งยังเป็นต้นแบบของปืนพกกึ่งอัตโนมัติด้วยกัน กล็อก 17 (Glock 17) - บีดีเอ (BDA) เป็นรุ่นย่อยของไฮพาวเวอร์ โดยนำแบบ FN Model 1900 มาผสมผสานกับไฮเพาว์เวอร์ประเทศที่ใช้ประเทศที่ใช้. - : ผลิตภายใต้ใบอนุญาตและใช้โดยทหารและตำรวจ กองทัพที่ใช้คือ M95 และ M02AR. - : เป็นปืนพกรุ่นทั่วไปที่ใช้ในกองทัพออสเตรเลีย - : ได้รับการคัดเลือกให้เป็นอาวุธประจำกายของกองทัพเบลเยี่ยมเมื่อปี พ.ศ. 2478 และสำนักงานตำรวจเบลเยี่ยม - : กองทัพแคนาดาใช้ปืนพกของบริษัทจอห์นอิงกลิส จากรัฐออนแทรีโอ ประเทศแคนาดา ใช้บริการเป็นปืนพกหลัก - : ใช้ปืนพกที่ผลิตโดยบริษัทจอห์นอิงกลิส จากรัฐออนแทรีโอ ประเทศแคนาดา - : ผลิตโดยไม่มีใบอนุญาตโดยบริษัทโนรินโค - : ใช้โดยกองตำรวจทหารแห่งชาติและกองทัพอากาศฝรั่งเศส ในระหว่างสงครามอินโดจีนครั้งที่หนึ่งและสงครามแอลจีเรีย - : ใช้ปืนพกที่ผลิตโดยบริษัทจอห์นอิงกลิส จากรัฐออนแทรีโอ ประเทศแคนาดา - : ผลิตภายใต้ใบอนุญาตเดียวกัน - : ผลิตภายในประเทศในฐานะปืนพกออโต 9เอ็มเอ็ม 1เอ โดยโรงงานอาวุธอินเดีย/กองทัพ Ishapore ใช้แม่พิมพ์จากบริษัทจอห์นอิงกลิส จากรัฐออนแทรีโอ ประเทศแคนาดา - : ใช้เป็นอาวุธทั่วไปในกองทัพแห่งชาติอินโดนีเซีย, โดยเฉพาะ กองบัญชาการกาเตาะ (Kopaska) กลุ่มนักประดาน้ำ และ กองบัญชาการคูซุส (Kopassus) กลุ่มกองทัพกำลังพิเศษ ผลิตภายใต้ใบอนุญาตโดยพินแดด เป็นพี1 - : ใช้โดยกองทัพเรืออิหร่าน กองกำลังพิเศษ - : ใช้โดยกองทัพไอริช แทนปืนพกเฮคเลอร์แอนด์คอช ยูเอสพี. - : ใช้โดยยาแมม ผลิตภายในประเทศ - : ส่งออกให้แก่เจ้าหน้าที่ในกองทัพจาเมกา - : ใช้แทนปืนพก กล็อก 17 - : ใช้เป็นอาวุธทั่วไปสำหรับเจ้าหน้าที่อาวุโสและกองกำลังพิเศษในกองทัพมาเลเซียและสำนักงานตำรวจแห่งชาติ - : ใช้โดยกองกำลังตำรวจนามิเบีย (NAMPOL) - : มากกว่า 300,000 กระบอกถูกสร้างขึ้นเพื่อ เวร์มัคท์ หลังจากโรงงานเอฟเอ็นถูกยึดโดยเยอรมนี ปืนพกถูกตั้งชื่อว่า พิสโทน 640 (บี) - : ใช้โดยกองกำลังพิเศษเกาหลีเหนือ - : ใช้แทนซิกซาวเออร์ พี226 - : ผลิตภายใต้ใบอนุญาตโดยบริษัทการป้องกันอุตสาหกรรมแห่งไนจีเรีย - : ใช้โดยกลุ่มกองทัพเรือพิเศษ. - : จัดส่งจากออสเตรเลีย - เคยใช้ในกองกำลังพิเศษโปแลนด์ (JW GROM) - : ตั้งแต่ปี พ.ศ. 2478 โดยกองกำลังรักษาดินแดนสาธารณรัฐ - : ใช้โดยกองทัพอากาศไทย - : ใช้โดยกองอำนวยการรักษาความปลอดภัย - : ถูกนำมาใช้โดยกองทัพบริติชในปี พ.ศ. 2497 และโดยสำนักงานความมั่นคงไอร์แลนด์เหนือในปี พ.ศ. 2541 แทนปืนพก กล็อก 17 เจ็น 4 - : ใช้โดยทีมกู้ภัย ตัวประกัน - : ใช้แทนปืนพก กล็อก 17 - : มีมีปริมาณน้อย นำเข้ามาในประเทศโดยกองทัพออสเตรเลียระหว่างสงครามเวียดนาม ยังคงสามารถพบได้ในกองทัพประชาชนเวียดนามยุติการผลิต ยุติการผลิต. ไฮเพาว์เวอร์ ถูกผลิตเป็นครั้งสุดท้ายในปี พ.ศ. 2560 จากนั้นก็ได้ขายลิขสิทธิ์และกรรมสิทธิ์ทั้งหมดให้แก่โรงงานยุทโธปกรณ์อิสโพธิ์, ประเทศอินเดีย</doc>
