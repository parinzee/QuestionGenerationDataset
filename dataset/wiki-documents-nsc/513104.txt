<doc id="513104" url="https://th.wikipedia.org/wiki?curid=513104" title="My Melody 360 องศารัก">My Melody 360 องศารัก My Melody 360 องศารัก คือ Musical Comedy ทาง โมเดิร์นไนน์ทีวี ออกอากาศเวลาเวลา 20.30 น. ทุกวันเสาร์-อาทิตย์เนื้อเรื่องตัวละครหลักตัวละครหลัก. - ครูโจ : ครูพละหนุ่มรูปหล่อ เป็นที่ชื่นชมของบรรดา ครูสาว ครูแก่ และนักเรียนมากมาย - ครูมารี : ครูสาวสวย เซ็กซี่ ที่โรงเรียนสั่งตรงจากต่างประเทศ ในอดีตครูมารีเป็นเพื่อนกับครูโจตอนสมัยมัธยม - ครูโรส : ครูสวยเซ็กซี่ หุ่นดี ชอบครูโจ - วิน : นักเรียนใหม่ รูปหล่อ หุ่นนักกีฬา เป็นที่จับจ้อง ของบรรดาสาวๆมากมาย - แพร : นักเรียนสาวแสนเฉิ่ม โก๊ะสวย แต่ซุ่มซ่ามเพราะสายตาสั้น แต่ไม่ชอบสวมแว่น - ทับทิม : เพื่อนสนิทของแพร น่ารัก อัธยาศัยดี - แจ๊ค : ลูกเศรษฐี ขี้โอ่ อ้วนหน้าตาดูไม่ได้ ทำอะไรก็ไม่เก่ง แต่เป็น ขาใหญ่ประจำโรงเรียน เพราะตกซ้ำหลายปี - เอก : ลูกคนใช้ที่บ้านของแจ๊ค ลูกน้อง จอมแสบคู่ใจของแจ็ค จัดการเรื่องแทนให้ทั้งหมด - แจน : เชียร์ลีดเดอร์สาวสวย เซ็กซี่ ดาวโรงเรียน - เจ : เด็กเงียบขรึม แต่รักเสียงเพลงเป็นชีวิตและจิตใจ ใส่หูฟังเพลงตลอดเวลา - ครูใหญ่สมเจตน์ : ครูใหญ่ประจำโรงเรียน ที่มีเรื่องราวมากมายที่ไม่ได้เปิดเผย - ครูอังคณา : ครูฝ่ายวิชาการ - เจ๊ชัย : ภารโรง ที่แอ๊บแมนเพื่อหวังทำตัวใกล้ชิดหนุ่มๆ ในโรงเรียนตลอดเวลา - เจ๊จุ๊บแจง : แม่ค้ากะเทยที่ตั้งบูธขายขนมนักแสดงหลักนักแสดงหลัก. - เจตริน วรรธนะสิน (รับบท : ครูโจ) - แคทรียา อิงลิช (รับบท : ครูมารี) - รฐา โพธิ์งาม (รับบท : ครูโรส) - พชร จิราธิวัฒน์ (รับบท : วิน) - ชนม์ทิดา อัศวเหม (รับบท : แพร) - อัปสรสิริ อินทรคูสิน (รับบท : ทับทิม) - เฉลิมพล ทิฆัมพรธีรวงศ์ (รับบท : แจ๊ค) - วริศ พรพงศา (รับบท : เอก) - เมทัล สุขขาว (รับบท : แจน) - มงคล กิจแสวง (รับบท : เจ) - ไพบูลย์เกียรติ เขียวแก้ว (รับบท : ครูใหญ่สมเจตน์) - วิยะดา โกมารกุล ณ นคร (รับบท : ครูอังคณา) - ธงธง มกจ๊ก (รับบท : เจ๊ชัย) - มัม ลาโคนิคส์ (รับบท : เจ๊จุ๊บแจง) - เกล้าแก้ว สินเทพดล (รับบท : แป้งร่ำ) - นนทนันท์ อัญชุลีประดิษฐ์ (รับบท : ก๊วยเจ๋ง) - อรรณพ ทองบริสุทธิ์ (รับบท : ตะวัน)นักแสดง/ศิลปินรับเชิญนักแสดง/ศิลปินรับเชิญ. 1. หนึ่ง ETC (รับเชิญตอนที่ 3 , 4) 2.
