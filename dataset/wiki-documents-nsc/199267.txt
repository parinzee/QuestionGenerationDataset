<doc id="199267" url="https://th.wikipedia.org/wiki?curid=199267" title="มหานครแห่งรัก">มหานครแห่งรัก มหานครแห่งรัก ( แปลว่า "ปารีส...ฉันรักเธอ") เป็นภาพยนตร์ความยาว 2 ชั่วโมง ที่ออกฉายในปี พ.ศ. 2549 ประกอบด้วยภาพยนตร์สั้นจำนวน 18 เรื่อง กำกับโดยผู้กำกับ 21 คน โดยในแต่ละเรื่องผู้กำกับมีอิสระในการเขียนบท คัดเลือกนักแสดง และเทคนิคการถ่ายทำ ภาพยนตร์ทั้ง 18 เรื่องไม่มีเนื้อหาเกี่ยวข้องกัน เพียงแต่กล่าวถึงความรัก ในแง่มุมต่างๆ ที่เกิดขึ้นในกรุงปารีส ภาพยนตร์ทั้ง 18 เรื่อง ตั้งชื่อตามหมายเลข "ตำบล" (arrondissement) ทั้ง 20 ตำบลของปารีส ประกอบด้วย- Montmartre (XVIIIe arrondissement ; เขต 17 ) — กำกับโดย Bruno Podalydès ผู้กำกับและนักเขียนบทชาวฝรั่งเศส - Quais de Seine (Ve arrondissement ; เขต 5 ) — กำกับโดย Paul Mayeda Berges นักเขียนบทชาวอเมริกัน และ Gurinder Chadha ภรรยา ผู้กำกับชาวอังกฤษเชื้อสายอินเดีย - Le Marais (IVe arrondissement ; เขต 4 ) — กำกับโดย กัส แวน แซนต์ ผู้กำกับและนักเขียนบทชาวอเมริกัน - Tuileries (Ier arrondissement ; เขต 1 ) — กำกับโดยพี่น้อง โจเอล โคเอน และอีธาน โคเอน ผู้กำกับและนักเขียนบทชาวอเมริกัน แสดงโดย สตีฟ บุสเซมี - Loin du 16e (XVIe arrondissement ; เขต 16 ) — กำกับโดย Walter Salles และ Daniela Thomas ผู้กำกับและนักเขียนบทชาวบราซิล - Porte de Choisy (XIIIe arrondissement ; เขต 13 ) — กำกับโดย คริสโตเฟอร์ ดอยล์ ผู้กำกับชาวออสเตรเลีย แสดงโดย บาร์เบต ชโรเดอร์ - Bastille (XIIe arrondissement ; เขต 12 ) — กำกับโดย Isabel Coixet ผู้กำกับและนักเขียนบทชาวสเปน แสดงโดย เซอร์จิโอ คาสเทลลิตโตและมิแรนดา ริชาร์ดสัน - Place des Victoires (IIe arrondissement ; เขต 2 ) — กำกับโดย Nobuhiro Suwa ผู้กำกับและนักเขียนบทชาวญี่ปุ่น - Tour Eiffel (VIIe arrondissement ; เขต 7 ) — กำกับโดย Sylvain Chomet นักสร้างภาพยนตร์แอนิเมชั่นชาวฝรั่งเศส - Parc Monceau (XVIIe arrondissement ; เขต 17 ) — กำกับโดย อัลฟอนโซ กวารอน ผู้กำกับและนักเขียนบทชาวเม็กซิกัน แสดงโดย นิค นอลเต - Quartier des Enfants Rouges (IIIe arrondissement ; เขต 3 ) — กำกับโดย Olivier Assayas ผู้กำกับและนักเขียนบทชาวฝรั่งเศส แสดงโดย แมกกี จิลเลนฮอล - Place des fêtes (XIXe arrondissement ; เขต 19 ) — กำกับโดย Oliver Schmitz ผู้กำกับและนักเขียนบทชาวเยอรมัน แสดงโดย จูเลียส บินอชและวิลเลม เดโฟ - Pigalle (IXe arrondissement ; เขต 9 ) — กำกับโดย Richard LaGravenese ผู้กำกับและนักเขียนบทชาวอเมริกัน แสดงโดย บ็อบ ฮอสกินส์และแฟนนี อาร์ดองต์ - Quartier de la Madeleine (VIIIe arrondissement ; เขต 8 ) — กำกับโดย Vincenzo Natali ผู้กำกับและนักเขียนบทชาวแคนาดา แสดงโดย เอไลจาห์ วูดและโอลกา คูรีเลนโก - Père-Lachaise (XXe arrondissement ; เขต 20 ) — กำกับโดย เวส คราเวน ผู้กำกับและนักเขียนบทชาวอเมริกัน แสดงโดย เอมิลี มอร์ติเมอร์ รูฟัส ซีเวล และอเล็กซานเดอร์ เพย์น - Faubourg Saint-Denis (Xe arrondissement ; เขต 10 ) — กำกับโดย ทอม ทีคแวร์ ผู้กำกับและนักเขียนบทชาวเยอรมัน แสดงโดย นาตาลี พอร์ตแมน - Quartier Latin (VIe arrondissement ; เขต 6 ) — เขียนบทโดย จีนา โรว์แลนด์ส นักแสดงชาวอเมริกัน กำกับโดย เจอราร์ด เดอปาร์ดิเออ นักแสดงชาวฝรั่งเศส และ Frédéric Auburtin ผู้กำกับชาวฝรั่งเศส แสดงโดย เบน กาซซารา - 14e arrondissement (XIVe arrondissement ; เขต 14 ) — กำกับโดย อเล็กซานเดอร์ เพย์น จากความสำเร็จของภาพยนตร์เรื่องนี้ ทีมงานสร้างได้ประกาศสร้างภาพยนตร์ภาคต่อ New York, I Love You เนื้อหากล่าวถึงความรัก ในแง่มุมต่างๆ ที่เกิดขึ้นในมหานครนิวยอร์ก สหรัฐอเมริกา กำหนดออกฉายในวันวาเลนไทน์ พ.ศ. 2552นักแสดงนักแสดง. - Nick Nolte รับบทเป็น วินเซนต์ - Rufus Sewell รับบทเป็น วิลเลี่ยม - Gaspard Ulliel รับบทเป็น Gaspard - Leila Bekhti รับบทเป็น Zarka - Li Xin รับบทเป็น Madame Li - Cyril Descours รับบทเป็น Francois - Axel Kiener รับบทเป็น Axel - Natalie Portman รับบทเป็น Natalie</doc>
