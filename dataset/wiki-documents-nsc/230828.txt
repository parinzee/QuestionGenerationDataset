<doc id="230828" url="https://th.wikipedia.org/wiki?curid=230828" title="เอฟเอ็น พี90">เอฟเอ็น พี90 พี90 คือปืนกลมือที่ออกแบบโดยเบลเยียม ชื่อย่อของอาวุธมาจากโครงการ 90 (Project 90) ซึ่งเจาะจงระบบอาวุธของทศวรรษที่ 1990 พี90 ถูกจัดว่าเป็นอาวุธป้องกันประจำบุคคลหรือพีดีดับบลิวและถูกออกบบมาให้เป็นอาวุธขนาดกะทัดรัดแต่ทรงพลังสำหรับพลขับ ผู้ทำงานด้านอาวุธ ผู้ให้การสนับสนุน หน่วยรบพิเศษ และหน่วยต่อต้านผู้ก่อการร้าย การพัฒนาเริ่มขึ้นระหว่างปีพ.ศ. 2529-2530 ที่ฟาบริก เนชั่นเนล เดอ เฮอร์สตอล (Fabrique Nationale de Herstal) พี90 มีจุดเด่นที่ความกะทัดรัดสไตล์บุลพัพ ด้ามจับแบบสองมือ และโครงสร้างที่ทำการพอลิเมอร์และอัลลอย อาวุธมีวัตกรรมใหม่หลายอย่างรวมทั้งกระสุนขนาด 5.7x28 ม.ม.ที่ออกแบบมาให้เจาะทะลุเกราะกันกระสุนได้มากกว่ากระสุนของปืนพก พี90 และแบบต่างๆ ถูกใช้โดยกองทัพและตำรวจกว่า 30 ประเทศทั่วโลกและรุ่นสำหรับการกีฬาก็เป็นที่นิยมในหมู่พลเมืองการพัฒนา การพัฒนา. พี90 ถูกสร้างขึ้นระหว่างปีพ.ศ. 2529-2530 ในเฮอร์สตอลประเทศเบลเยียม เป้าหมายของมันคือการเข้ามาแทนที่ปืนพกและคาร์บินซึ่งใช้โดยทหารและผู้รักษากฎหมาย เพราะว่าอาวุธเหล่านั้นไม่มีประสิทธิภาพพอในการจัดการเป้าหมายที่สวมเสื้อเกราะกันกระสุน ปืนถูกออกแบบให้ใช้กระสุนใหม่ขนาด 5.7x28ม.ม.ซึ่งมีการทะลุทะลวงที่ดีกว่า ระยะที่ไกล และการเคลื่อนที่ที่แบนราบมากกว่าปืนพกคาลิเบอร์ที่ใช้กระสุนนาโต้ขนาด 9x19 ม.ม.พาราเบลลลัม เริ่มแรกนั้นอาวุธได้ใช้กระสุนขนาด 5.7x28 ม.ม.และกระสุนอีกหลายชนิด ต้นแบบอันแรกที่ใช้กระสุนนี้สร้างเสร็จในเดือนตุลาคมพ.ศ. 2529 และปืนกลมือมากกว่า 3,000 กระบอกถูกผลิตในรูปแบบนี้จนถึงปีพ.ศ. 2536 ด้วยการผลิตระดับต่ำ ในขณะเดียวกันเอฟเอ็นได้ทำการแก้ไขกระสุนพร้อมความตั้งใจที่จะใช้มันในปืนพกกึ่งอัตโนมัติรุ่นเอฟเอ็น ไฟว์-เซเว่น กระสุนใหม่นี้มีชื่อว่าเอสเอส190 ที่มีเหล็กและอะลูมิเนียมเป็นตัวเจาะทะลุ หัวกระสุนอื่นๆ หลายแบบยังได้ถูกสร้างขึ้นมาสำหรับกระสุนใหม่ พี90 รุ่นที่ดัดแปลงแล้วได้ใช้กระสุนแบใหม่ซึ่งปราฎตัวในปีพ.ศ. 2536ประเทศผู้ใช้งานประเทศผู้ใช้งาน. - : กองทัพและคอมมานโดของออสเตรเลีย - : หน่วยยุทธวิธีและหน่วยคอมมานโด - : หน่วยต่อต้านผู้ก่อการร้ายและกองกำลังพิเศษ - : หน่วยตำรวจกลางพิเศษ และกองกำลังพิเศษ - : ใช้โดยตำรวจในโนวา สโคเทียและกำลังรบเฉพาะกิจร่วมที่ 2 - : กองทัพบกและนาวิกโยธินชิลี - : กองกำลังรักษาดินแดนไซปรัสได้ซื้อพี90 จำนวน 350 กระบอกเมื่อปีพ.ศ. 2543 เพื่อใช้กับหน่วยรบพิเศษ ในรายการสั่งซื้อรวมทั้งที่เก็บเสียง เลเซอร์ชี้เป้า และไฟฉายยุทธวิธี - : ใช้โดยบางหน่วยพิเศษภายในกองทัพบกของฝรั่งเศส อย่างนาวิกโยธินและนาวิกโยธินขนส่งทางอากาศ นอกจากนั้นยังใช้โดยจีไอจีเอ็น และหน่วยเรด (Recherche Assistance Intervention Dissuasion, RAID) ของตำรวจ - : ใช้โดยตำรวจกลางที่ทำหน้าที่อารักขาบุคคลสำคัญ - : ยามชายฝั่งและหน่วยต่อต้านผู้ก่อการร้าย - : กลุ่มป้องกันพิเศษ - : กองทหารพรานไอริช - : ใช้โดยหน่วยจีไอเอส (Gruppo di Intervento Speciale, GIS) เอ็นโอซีเอส (Nucleo Operativo Centrale di Sicurezza, NOCS) และคอมซูบิน (COMSUBIN) - : ใช้โดยลิพับลิกันการ์ด (Republican Guard Brigade) และหน่วยรักษาความปลอดภัย - : กองทัพเรือและหน่วยต่อต้านผู้ก่อการร้ายของมาเลเซีย[ - : กองทัพเรือและหน่วยรบพิเศษของแมกซิโก - : หน่วยจีไอเอสจีอาร์ (Groupement d'Intervention et de Sécurité de la Gendarmerie Royale) - : ใช้โดยหน่วยต่อต้านผู้ก่อการร้าย - : กองทัพบก หน่วยรบพิเศษ และหน่วยองครักษ์ของปากีสถาน - : กองทัพของปานามา - : หน่วยรบพิเศษและกองทัพเรือเปรู - : ใช้อย่างจำกัดโดยหน่วยกรอม (GROM) - : หน่วยรบพิเศษกองทัพเรือและหน่วยรบพิเศษจีโออี (Grupo de Operações Especiais) - : กองกำลังฉุกเฉินพิเศษ - : หน่วยคอมมานโดของสิงคโปร์ - : หน่วยจีอีโอของตำรวจสเปน (Grupo Especial de Operaciones) นอกจากนั้นยังใช้โดยกองกำลังติดอาวุธอย่างหน่วยอีเอดีเอ (Escuadron de Apoyo al Despliegue Aereo) และหน่วยอีซีเอพีเอซี (Escuadron de Zapadores Paracaidistas) - : หน่วยรบพิเศษของกองทัพไทย - : ใช้โดยตำรวจในเวสท์มิดแลนด์ที่สนามบินเบอร์มิงแฮมและสนามบินอื่นๆ - : ใช้โดยองค์กรลับ หน่วยป้องกัน หน่วยผู้อพยพและตรวจสัญชาติ และหน่วยงานรักษากฎหมายอื่นๆ โดยส่วนมากเป็นหน่วยสวาท</doc>
