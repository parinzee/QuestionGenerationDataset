<doc id="764070" url="https://th.wikipedia.org/wiki?curid=764070" title="มาโฮสึไค พรีเคียว!">มาโฮสึไค พรีเคียว! มาโฮสึไค พรีเคียว! หรือ มาโฮเกิร์ล พรีเคียว! (Mahou Tsukai Precure! or Maho Girls Precure!) เป็น อะนิเมะ ในซีรีส์ของ พริตตี้เคียว ลำดับที่ 13 ผลิตโดย Toei Animationเนื้อเรื่องตัวละครหลัก ตัวละครหลัก. เคียวมิราเคิล (Cure Miracle) / อาซาฮินะ มิไร (Asahina Mirai) "ปาฏิหาริย์ของสองเรา เคียวมิราเคิล!" สีประจำตัว : "สีชมพู Diamond and Alexandrite" ( ), "สีแดง Ruby" ( ), "สีน้ำเงิน Sapphire" ( ), "สีเหลือง Topaz" ( ) เด็กสาวผู้ร่าเริงและอยากรู้อยากเห็น ชื่นชอบในเรื่องราวต่างๆ ได้พบกับริโกะโดยบังเอิญ เนื่องจากริโกะช่วยเตือนเรื่องที่มิไรเผลอทำตุ๊กตาหมีหล่น "ท่าไม้ตาย" "Precure Diamond Eternal!" "Precure Ruby Passionale!" "Precure Sapphire Smartish!" "Precure Topaz Esperanza!" "Precure Linkle Pink Tourmaline!" "Precure Linkle Tanzanite!" "Precure Linkle Amethyst!" "Precure Linkle Garnet!" "Precure Linkle Peridot!" "Precure Extreme Rainbow!" "Precure Heartful Rainbow!" เคียวมาจิคัล (Cure Magical) / อิซาโยอิ ริโกะ (Izayoi Riko) "เวทมนตร์ของสองเรา เคียวมาจิคัล!" สีประจำตัว : "สีม่วง Diamond and Alexandrite" ( ), "สีแดง Ruby" ( ), "สีน้ำเงิน Sapphire" ( ), "สีเหลือง Topaz" ( ) สาวน้อยจากโลกเวทมนตร์และเป็นนักเรียนจอมเวทย์ฝึกหัด ทำให้ยังไม่สามารถควบคุมเวทมนตร์ได้อย่างเต็มที่และมักแสดงเวทมนตร์ให้คนอื่นๆ ได้เห็นจึงมักตกเป็นเป้าสายตาของคนทั่วไป "ท่าไม้ตาย" "Precure Diamond Eternal!" "Precure Ruby Passionale!" "Precure Sapphire Smartish!" "Precure Topaz Esperanza!" "Precure Linkle Aquamarine!" "Precure Linkle Peridot!" "Precure Linkle Moonstone!" "Precure Extreme Rainbow!" "Precure Heartful Rainbow!" เคียวเฟลิซ (Cure Felice) / ฮานามิ โคโตฮะ (Hanami Kotoha) / ฮาจัง (Ha-chan) "ขอส่งคำอวยพรให้กับชีวิตและกว้างไกล เคียวเฟลิซ!" สีประจำตัว : "สีเขียว" ( ), "สีชมพู" ( ) ฮาจังเป็นนางฟ้าทารกลึกลับที่อาศัยอยู่ภายในริงเคิลสมาร์ทบุ๊คและโตขึ้นเรื่อยๆ อย่างรวดเร็ว ในตอนที่ 22 เธอกลายเป็นเคียวเฟลิซ ซึ่งเป็นพริตตี้เคียวคนที่ 3 "ท่าไม้ตาย" "Precure Emerald Reincarnation!" "Precure Linkle Pink Tourmaline!" "Precure Extreme Rainbow!" "Precure Heartful Rainbow!"ผู้ช่วยของพรีเคียว ผู้ช่วยของพรีเคียว. โมฟุรุน (Mofurun) / เคียวโมฟุรุน (Cure Mofurun) (เฉพาะภาคเดอะมูฟวี่เท่านั้น) ตุ๊กตาหมีของมิไรที่ได้มาจากคุณยายคาโนโกะ มิไรมักพาโมฟุรุนไปไหนมาไหนด้วยกันเสมอ มีชีวิตขึ้นมาด้วยพลังเวทมนตร์ของมิไรและริโกะและภาคเดอะมูฟวี่ โมฟุรุนกลายเป็นพริตตี้เคียวคนที่ 4ตัวละครอื่นๆ ตัวละครอื่นๆ. ยูกิ คาโนโกะ (Yuki Kanoko) คุณยายของมิไร อาซาฮินะ ไดคิจิ (Asahina Daikichi) คุณพ่อของมิไร อาซาฮินะ เคียวโกะ (Asahina Kyoko) คุณแม่ของมิไร ริอัน (Lian) คุณพ่อของริโกะ ริเลีย (Lilia) คุณแม่ของริโกะ ริซุ (Liz) พี่สาวของริโกะ ผู้อำนวยการโรงเรียนเวทมนตร์ ผู้อำนวยการแห่งโรงเรียนเวทมนตร์ มักปรากฏตัวในฐานะจอมเวทย์ทั่วไป เป็นอาจารย์สอนในคลาสพิเศษให้กับมิไรและริโกะ เพื่อฝึกฝนให้กับทั้ง 2 ในฐานะจอมเวทย์ในตำนาน เคียวโต (Kyoto) ผลึกแห่งเวทมนตร์ (Magic Crystal) ฟรองซัว (Francois) กุสตาฟ (Gustav) โทดด์ (Todd) ฮุค (Hook) อาจารย์ไอแซก (Isaac Teacher) จุน (Jun) เคย์ (Kay) เอมิลี่ (Emily)อุปกรณ์ของพรีเคียว อุปกรณ์ของพรีเคียว. เป็นริงเคิลสโตนสีน้ำเงิน ริงเคิลสโตนโทแพซ (Linkle Stone Topaz) ปรากฏครั้งแรกในตอนที่ 11 เป็นริงเคิลสโตนสีเหลือง ริงเคิลสโตนเอเมอรัลด์ (Linkle Stone Emerald) ปรากฏครั้งแรกในตอนที่ 22 สามารถใช้ท่าไม้ตายได้หลายรูปแบบ ริงเคิลสมาร์ทบุ๊ค (Linkle Smart Book) อุปกรณ์แปลงร่างของเคียวเฟลิซ ฟลาวเวอร์เอคโค่วอนด์ (Flower Echo Wand) เรนโบว์แคเรียจ (Rainbow Carriage) มาจิควอนด์ (Magic Wands)
