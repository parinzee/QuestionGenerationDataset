<doc id="362956" url="https://th.wikipedia.org/wiki?curid=362956" title="อี เซ-ดล">อี เซ-ดล อี เซ-ดล (; ; 2 มีนาคม ค.ศ. 1983 — ) เป็นนักหมากล้อมระดับอาชีพชาวเกาหลีใต้ ที่อยู่ในระดับ 9 ดั้งประวัติ ประวัติ. เซดลเป็นหนึ่งในผู้เล่นที่แข็งแกร่งในสมาคมหมากล้อมแห่งประเทศเกาหลี โดยมีหลักฐานตามจำนวนรายการแข่งขันที่เขาเข้าร่วม ถูกจัดให้อยู่ในอันดับสามในการแข่งขัน 33 รายการ ถัดจากโช ฮุน-ฮย็อน (150) กับอี ชัง-โฮ (140) แม้กระนี้ เขาก็ยังอธิบายว่าการเล่นแบบเปิดเกมของเขาอยู่ในระดับที่อ่อนมาก เซดลยังอยู่ในอันดับที่สองในการแข่งขันระดับนานาชาติ ด้วยการแข่งขัน 11 รายการ ถัดจากอี ชางโฮ (18) ปัจจุบันเซดลสมรสแล้ว และมีลูกสาวหนึ่งคนผลงานเลื่อนขั้นผลงานระดับอาชีพผลงานระดับอาชีพ. - ค.ศ. 1997-98: ชนะ 0, แพ้ 1, เสมอ 0 - ค.ศ. 1998-00: ชนะ 0, แพ้ 6, เสมอ 0 - ค.ศ. 2000-02: ชนะ 55, แพ้ 29, เสมอ 0- ค.ศ. 2002-04: ชนะ 67, แพ้ 32, เสมอ 0 - ค.ศ. 2004-06: ชนะ 64, แพ้ 23, เสมอ 0 - ค.ศ. 2006-08: ชนะ 122, แพ้ 52, เสมอ 0- ค.ศ. 2008-09: ชนะ 72, แพ้ 23, เสมอ 0 - ค.ศ. 2009-10: ชนะ 23, แพ้ 11, เสมอ 0 - ค.ศ. 2010-11: ชนะ 13, แพ้ 0, เสมอ 0 - รวม: ชนะ 416, แพ้ 177, เสมอ 0 (อัตราชนะร้อยละ 70%)รายการแข่งขัน รายการแข่งขัน. อันดับ 3 ในจำนวนรวมทั้งหมดของรายการในเกาหลีใต้- รวม: 33 รายการ, 16 รอบรองชนะเลิศ</doc>
