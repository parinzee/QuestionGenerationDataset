<doc id="246317" url="https://th.wikipedia.org/wiki?curid=246317" title="คดีฆาตกรรมหมู่บ้านอาถรรพ์">คดีฆาตกรรมหมู่บ้านอาถรรพ์ คดีฆาตกรรมหมู่บ้านอาถรรพ์ อยู่ในคินดะอิจิ ภาค มังงะ 1 เล่ม 2 - 3 ตอนที่ 1 - 13 ตัวละครหลัก คินดะอิจิ ฮาจิเมะ ไปงานแต่งงานของ โทคิตะ วาคาบะ ที่หมู่บ้าน 6 มุมกับมิยูกิ นานาเสะ มิยูกิ ไปกับคินดะอิจิ และอาจารย์โอดะงิริ ซึซึมุ ผู้ต้องสงสัย โอดะงิริ ซึซึมุ ( อาจารย์ที่คบกันกับ วาคาบะ และขอคินดะอิจิไปงานแต่งงานของวาคาบะที่ถูกครอบครัวเรียกกลับไปแต่งงานกับ เรนโจ ฮิซาฮีโกะ ) โทคิตะ วาคาบะ ( เจ้าสาว เป็นน.รที่คบกับอาจารย์โอดะงิริ ) โทคิตะ จูโซ ( พ่อวาคาบะ เจ้าของคฤหาสถ์นาฬิกา ) คุซานางิ มิทซึโกะ ( ครอบครัววาคาบะ เจ้าของคฤหาสถ์เถาวัลย์ ) โกโต้ รัน ( ครอบครัววาคาบะ เจ้าของคฤหาสถ์หอคอย ) คาบุโตะ เรย์อิจิ ( ครอบครัววาคาบะ คฤหาสถ์เสื้อเกราะ ) อิสึชิกิ เทราโอะ ( ครอบครัววาคาบะ เจ้าของคฤหาสถ์สเทนกลาส ) คาซามัตสึริ จุนยะ ( ครอบครัววาคาบะ เจ้าของคฤหาสถ์ไก่ฟ้า ) เรนโจ ฮิซาฮีโกะ ( เจ้าบ่าว ) คาบุโตะ คิรีโกะ ( ลูกของเรย์อิจิ อยู่คฤหาสถ์เดียวกับเรย์อิจิ ) ทาราวะดะ โคทาโร่ ( ตำรวจ ) เรื่องย่อ คินดะอิจิและอาจารย์ไปงานแต่งงานของวาคาบะที่หมู่บ้าน 6 มุมบ้านเกิดของวาคาบะแต่ระหว่างพิธีคนในนั้นกลับถูกฆ่าตายทีละคนทีละคนลง...โดยฆาตกรใช้ชื่อว่า ศพรายที่ 7 ผู้ตาย ทั้งหมด ผู้รอดชีวิต ไม่มี ฆาตกรตัวจริง ( ศพรายที่ 7 ) โอดะงิริ ซึซึมุ เหตุจูงใจ เหตุจูงใจคือ ครอบครัวของตนเองถูกคนในคฤหาสถ์ทุกคนฆ่าตายยกเว้นแม่และพ่อแต่แม่บอกว่าพ่อแฝงตัวเป็นคนในคฤหาสถ์แล้วบอกว่าพ่อเป็นใครแล้วถูกแม่สั่งและสอนการฆ่าคนให้ยกเว้นพ่อห้ามฆ่า ตัวจริงของพ่อที่แม่ของโอดะงิริ ซึซึมุ คาซามัตซึริ จุนยะ ฉากสุดท้ายของ คดีฆาตกรรมหมู่บ้านอาถรรพ์ คินดะอิจิได้บอกว่าโอดะงิริคือศพรายที่ 7...แล้วโอดะงิริก็แย่งปืบของตำรวจยิงเรย์อิจิตายและรีบจับมิยูกิเป็นตัวประกันแล้ววิ่งไปตรงไร่กัญชาแล้วคินดะอิจิก็ไล่ตามไปโอดะงิริจึงยิงคินดะอิจิและคิดจะยิงตัดขั้วหัวใจคินดะอิจิแต่อยู่คาซามัตสึริก็ถือปืนยิงอาจารย์โอดางิริ ซึซึมุแล้วพูดว่า " อภัยให้ด้วยลูกพ่อ...!! " แล้วคาซามัตสึริก็เข้าไปที่โอดะงิริแล้วฆ่าตัวตายและเผาไร่กัญชาสุดท้ายจึงไม่เหลือใครสักคนในคดีนี้... - คินดะอิจิ กับ คดีฆาตกรรมปริศนา คดีฆาตกรรมหมู่บ้านอาถรรพ์ได้รับอิทธิพลจากนวนิยาย ไสย์ศาสตร์ฆาตกรรม ( ยังไม่แปลไทย )</doc>
