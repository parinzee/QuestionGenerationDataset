<doc id="189216" url="https://th.wikipedia.org/wiki?curid=189216" title="เฮคแลร์อุนด์คอค เอ็มเพ5">เฮคแลร์อุนด์คอค เอ็มเพ5 เฮคเลอร์แอนด์คอช เอ็มพี5 เป็นปืนกลมือ 9 ม.ม. ถูกพัฒนาใน ค.ศ. 1960 โดยกลุ่มวิศวกรของบริษัทผลิตอาวุธเล็ก เฮคแลร์อุนด์คอค จากเยอรมนีตะวันตก บริษัทถูกกระตุ้นโดยความสำเร็จของปืนเล็กยาว พัฒนากลุ่มของอาวุธเล็กที่ประกอบด้วยอาวุธปืน 4 อย่างด้วยกัน ที่ซึ่งรุ่นแรกนั้นใช้กระสุนแบบ 7.62x51 ม.ม. รุ่นที่สองใช้ 7.62x39 ม.ม. รุ่นที่สามใช้ 5.56x45 ม.ม. และรุ่นที่สี่ใช้ 9x19 ม.ม. ของปืนพก เอ็มพี5 (ย่อมาจาก Maschinenpistole 5) ถูกสร้างในกลุ่มที่สี่ของอาวุธปืน และรู้กันในนามเอชเค54 () งานสร้างเอ็มพี5 เริ่มขึ้นใน ค.ศ. 1964 และไม่ถึงสองปีมันก็ถูกนำไปใช้โดยกองกำลังตำรวจของเยอรมันและกองกำลังพิเศษ มันถูกใช้โดยกองกำลังติดอาวุธและผู้คุมกฎหมายในกว่า 40 ประเทศ ปืนกลมือถูกผลิตภายใต้ใบอนุญาตในหลายประเทศรวมทั้ง กรีซ อิหร่าน เม็กซิโก ปากีสถาน ซูดาน ตุรกีและสหราชอาณาจักรประเทศผู้ใช้งานประเทศผู้ใช้งาน. - : ใช้โดยกองกำลังพิเศษและตำรวจอัลบาเนีย - : ใช้โดยคอมมานโดขนส่งทางอากาศของอัลจีเรีย - : ใช้โดยกองพันที่ 4 หน่วยเอสเอเอสอาร์ กองทัพอากาศและผู้รักษากฎหมาย - : ใช้โดยกองทัพ กองกำลังพิเศษ และกองพันเคลื่อนที่เร็ว - : ใช้โดยกองกำลังพิเศษของกองทัพและตำรวจ - : ใช้โดยทีมยุทธวิธีของตำรวจ หน่วยเฉพาะกิจของกองทัพเรือ - : ใช้โดยกองพันขนส่งทางอากาศที่ 1 และ กองกำลังพิเศษของกองทัพเรือ - : ใช้โดยหน่วยพิเศษของตำรวจ - : ใช้โดยตำรวจ - : ใช้โดยกองกำลังพิเศษของกองทัพและตำรวจ - : ใช้โดยกองทัพและตำรวจ ใช้ชื่อเรียกว่า9.00 เคพี 2000 - : ใช้โดยหน่วยพิเศษในกองทัพ - : ใช้โดยตำรวจ ส่วนกองทัพบกและกองทัพเรือใช้เอ็มพี5เค - : ใช้โดยตำรวจและกองกำลังพิเศษ อาวุธถูกผลิตโดยบริษัทเฮลลินิก ดีเฟนซ์ ซิสเตมส์ - : ใช้โดยตำรวจ ผู้คุม และกองกำลังป้องกันภาคพื้นดิน - : ใช้โดยยามชายฝั่งและกองกำลังพิเศษ - : ใช้โดยกองทัพ กองกำลังพิเศษ หน่วยรักษาความปลอดภัยแห่งชาติ และหน่วยอื่น ๆ - : ใช้โดยกองกำลังติดอาวุธและหน่วยต่อต้านผู้ก่อการร้าย - : ใช้โดยกองทัพและตำรวจ - : ใช้โดยหน่วยนาวีซีลของกองทัพเรือ - : ใช้โดยกองกำลังพิเศษ - : ใช้โดยกองกำลังติดอาวุธ หน่วยพิเศษอื่น ๆ และเจ้าหน้าที่ผู้รักษากฎหมาย - : ใช้โดยกองทัพและเจ้าหน้าที่ผู้รักษากฎหมาย - : ใช้โดยกองกำลังติดอาวุธและตำรวจ - : ใช้โดยนาวิกโยธิน ตำรวจ และหน่วยต่อต้านผู้ก่อการร้าย - : ใช้โดยกองทัพและตำรวจ - : ใช้เป็นอาวุธพื้นฐานสำหรับการปะทะระยะใกล้ของกองทัพ ถูกสร้างโดยโรงงานของปากีสถานภายใต้ใบอนุญาตจากเฮคเลอร์แอนด์คอช - : ใช้โดยกองกำลังติดอาวุธ กองกำลังพิเศษ และหน่วยสวาทของตำรวจ - : ใช้โดยหน่วยต่อต้านผู้ก่อการร้ายและตำรวจ - : ใช้โดยกองกำลังพิเศษ - : ใช้โดยหน่วยคอมมานโดและตำรวจ - : ใช้โดยตำรวจและกองกำลังพิเศษ - : ใช้โดยตำรวจและกองกำลังพิเศษ - : ใช้โดยหน่วยคอมมานโดและหน่วยเฉพาะกิจ - : ใช้โดยกองทัพ มนุษย์กบของนาวิกโยธิน ยามชายฝั่ง และหน่วยสวาทและกองกำลังพิเศษ - : ใช้โดยตำรวจ และ เพชฌฆาตผู้ที่ทำหน้าที่ประหารชีวิตของกรมราชทัณฑ์ - : ใช้โดยนาวิกโยธินและหน่วยปฏิบัติการพิเศษอย่างทหารพราน หน่วยเดลต้า หน่วยซีลของกองทัพเรือและหน่วยพิเศษของกองทัพอากาศ นอกจากนั้นยังใช้โดยผู้บังคับกฎหมายและเจ้าหน้าที่พิเศษ</doc>
