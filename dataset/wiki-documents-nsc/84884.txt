<doc id="84884" url="https://th.wikipedia.org/wiki?curid=84884" title="เอเดียม">เอเดียม เอเดียม (Adium) เป็นโปรแกรมเมสเซนเจอร์บนระบบปฏิบัติการ Mac OS X สนับสนุนโพรโทคอลการพูดคุยหลายชนิด โดยพัฒนามาจากไลบรารี libpurple โปรแกรมพัฒนาขึ้นด้วย API ของ Mac OS X ที่ชื่อ Cocoa และใช้วิธีการพัฒนาแบบซอฟต์แวร์เสรี มีสัญญาอนุญาตใช้งานเป็น GPLความต้องการของระบบ ความต้องการของระบบ. Adium 1.3 ต้องการ Mac OS X 10.4 หรือใหม่กว่านั้น สำหรับผู้ใช้ Mac OS X 10.3.9 ควรที่จะใช้เวอร์ชัน 1.0.6 ในทางตรงกันข้ามผู้ใช้ Mac OS X 10.2 ก็่ควรจะต้องใช้เวอร์ชัน 0.89.1 Adium 0.88 และเวอร์ชันที่สูงกว่านั้นได้ถูกสร้างออกมาด้วย Universal binariesโพรโทคอลที่สนับสนุนโพรโทคอลที่สนับสนุน. - Bonjour - Facebook Chat - Gadu-Gadu - IRC (ยังจำกัดอยู่ที่ trunk builds เท่านั้น) - LiveJournal - Lotus Sametime - MeBeam โดยใช้ปลั๊กอิน - MySpaceIM - Novell GroupWise - OSCAR ประกอบด้วย AIM, MobileMe (เดิมคือ .Mac) และ ICQ - Skype โดยใช้ปลั๊กอิน - Tlen โดยใช้ปลั๊กอิน - Windows Live Messenger - XMPP (เดิมคือ Jabber) ประกอบด้วย Google Talk และ LiveJournal - Xfire โดยใช้ XBlaze ปลั๊กอิน - Yahoo! Messenger - Yahoo! Japan - Zephyr</doc>
