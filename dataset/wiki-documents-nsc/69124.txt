<doc id="69124" url="https://th.wikipedia.org/wiki?curid=69124" title="คาเมนไรเดอร์ V3">คาเมนไรเดอร์ V3 คาเมนไรเดอร์ V3 () หรือ มาส์คไรเดอร์ V3 เป็นผลงานชุดที่ 2 ของภาพยนตร์ฮีโร่แปลงร่างเรื่องคาเมนไรเดอร์ ของประเทศญี่ปุ่นซึ่งเป็นภาคต่อของ คาเมนไรเดอร์ภาคแรก โดยเนื้อหาจะเน้นที่ดราม่ามากกว่าภาคแรก กล่าวคือในภาคนี้แรงบันดาลใจที่จะต่อสู้กับเหล่าร้ายของตัวเอก มาจากความต้องการที่จะล้างแค้นให้ครอบครัวที่ถูกฆ่าโดยองค์กรเดสตรอน คาเมนไรเดอร์ V3 เข้ามาฉายในประเทศไทยในชื่อของ ไอ้มดเขียว V3 (ซึ่งคาดว่าชื่อคงมาจากจากชุดสีเขียวของตัวละครหลัก) และทำให้เด็กๆในสมัยนั้นนิยมกันเป็นอย่างมาก โดยเฉพาะคำพูดตอนแปลงร่างที่นักพากย์ชาวไทยได้พากย์ลงไปใหม่ว่า "บิดๆเบี้ยวๆขอแปลงเป็นมดเขียววี3" (จากเดิมคือ "เฮนชิน วีทรี") ทำให้คำพูดนี้เป็นที่นิยิมและเข้าใจกันในหมู่ชาวไทยว่าคำพูดนี้จะหมายถึง คาเมนไรเดอร์ V3 นอกเหนือจากนั้นในประเทศไทยการปรากฏตัวของคาเมนไรเดอร์ V3 ได้ก่อให้เกิดความเข้าใจผิดของคำว่า V โดยชาวไทยบางคนเข้าใจผิดว่า V หมายถึงลำดับของคาเมนไรเดอร์ เพราะคาเมนไรเดอร์ V3 เป็นคาเมนไรเดอร์คนที่ 3 พอดี ทำให้บางคนจะเข้าใจผิดเรียกคาเมนไรเดอร์คนที่ 1 (ฮอนโก ทาเคชิ) และคาเมนไรเดอร์คนที่ 2 (อิจิมอนจิ ฮายาโตะ) ว่า ไอ้มดแดง V1, ไอ้มดแดง V2 (รวมถึงคาเมนไรเดอร์คนหลังๆ เช่น คาเมนไรเดอร์สตรองเกอร์ ก็ถูกเรียกว่า V7) ซึ่งจริงๆแล้วชื่อของ V3 มาจากคำว่า Version Three (เวอร์ชันทรี) นอกจากนั้นแล้วคาเมนไรเดอร์ V3 ได้ก่อกำเนิด "ชื่อของคาเมนไรเดอร์" (ซึ่งคาเมนไรเดอร์ 2 คนแรกไม่มีชื่อเรียก) ทำให้ตัวละครในเรื่องสามารถเรียกตัวละครคาเมนไรเดอร์ V3 ด้วยชื่อย่อได้คือ V3 คาเมนไรเดอร์ V3 เป็นภาคที่ได้รับความนิยมอย่างมากจนถึงกับเกมหลายๆเกมที่เกี่ยวกับคาเมนไรเดอร์ยุคเก่า ต้องมี V3 มาเกี่ยวข้องเสมอๆ (เช่น Super Hero Sakusen: Daidaru no Yabou, Super Tokusatsu Taisen 2001, Kamen Rider: Seigi no Keifu) อีกทั้งในคาเมนไรเดอร์ภาคหลังๆ คาซามิ ชิโร่ หรือ V3 ก็มักจะปรากฏตัวมาช่วยเหลือคาเมนไรเดอร์รุ่นน้องอยู่บ่อยครั้งเนื้อเรื่อง เนื้อเรื่อง. หลักจากที่ดับเบิ้ลไรเดอร์ทำลายองค์กรเกลช็อกเกอร์ลงไปแล้ว คาซามิ ชิโร่ ตัวเอกของเรื่องเป็นนักขี่จักรยายนต์ชมรมเดียวกับฮอนโก ทาเคชิ (คาเมนไรเดอร์หมายเลข 1) และ อิจิมอนจิ ฮายาโตะ (คาเมนไรเดอร์หมายเลข 2) วันหนึ่งครอบครัวของเขา ทั้งพ่อ, แม่และน้องสาวถูกองค์กรวายร้ายเดสตรอนฆ่าจนหมด ด้วยความโกรธแค้นที่อยู่ในใจเขาได้ขอให้ทาเคชิและฮายาโตะช่วยดัดแปลงเขาเป็นมนุษย์ดัดแปลง แต่ทั้งสองปฏิเสธ จากนั้นไรเดอร์ทั้งสองได้บุกเข้าไปที่ฐานทัพของเดสตรอนแต่กลับเป็นกับดักและเกือบถูกฆ่าแต่คาซามิ ชิโร่ก็เข้ามาช่วยและสละชีวิตเพื่อช่วยไรเดอร์ทั้งสอง ไรเดอร์ทั้งสองเห็นความกล้าของชิโร่จึงดัดแปลงให้ชิโร่กลายเป็น คาเมนไรเดอร์ V3 และโผล่มาช่วยไรเดอร์ทั้งสองสู้กับ เต่าบาซูก้า มนุษย์ดัดแปลงของเดสตรอน ซึ่งเต่าบาซูก้ามีพลังทำลายล้างที่สูงมากจนไรเดอร์ทั้งสองต้องยอมสละชีวิตทำลายเต่าบาซูก้าลงและพูดฝากฝังญี่ปุ่นไว้ให้กับV3 จึงเป็นหน้าที่ของคาเมนไรเดอร์ V3 ที่จะต้องปกป้องญี่ปุ่นต่อไป (แต่ไรเดอร์ทั้งสองก็กลับมาอีกครั้งในตอนที่ 33)ตัวละครตัวละครหลักตัวละคร. ตัวละครหลัก. - คาซามิ ชิโร่ / คาเมนไรเดอร์ V3 : นักขี่จักรยานยนต์รุ่นน้องของฮอนโก ทาเคชิและอิจิมอนจิ ฮายาโตะที่ถูกองค์กรเดสตรอนฆ่าคนในครอบครัวจนหมด เขาได้รับการดัดแปลงให้เป็นคาเมนไรเดอร์ โดยคาเมนไรเดอร์รุ่นพี่ทั้งสอง ร่างกายของ V3 มีความลับ 26 อย่างอยู่ในตัว ซึ่งจะเปิดเผยขึ้นมาเรื่อยๆ ในแต่ละตอน คาซามิแปลงร่างโดยการพูดคำว่า "เฮนชิน วีทรี" (แปลงร่าง วี3) แล้วใบพัดที่เข็มขัดทั้งสองจะหมุน และแปลงร่างได้ เขายังมีมอเตอร์ไซด์ เฮอร์ริเคน ซึ่งทาจิบานะ โทเบเป็นคนสร้างขึ้นมาให้ นอกจากนั้นแล้วV3ถือเป็นไรเดอร์ที่มีท่าไม้ตายเยอะมากคนหนึ่ง ชุดแต่งกายของ V3 มีต้นแบบมาจากแมลงปอ (ที่มา: คาเม็นไรเดอร์.วิเคีย และ วิกิพีเดียภาษาอังกฤษ)- ยูกิ โจจิ / ไรเดอร์แมน: นักวิทยาศาสตร์ฝีมือดีของเดสตรอนซึ่งถูกโยโร่ยเก็นซุยหักหลังเพราะกลัวจะได้ตำแหน่งสูงกว่าตน เขาได้รับการดัดแปลงจากเพื่อนนักวิทยาศาสตร์ของเดสตรอนจนกลายมาเป็นไรเดอร์แมน บทบาทของเขาอยู่ในฐานะก้ำกึ่งระหว่างตัวเอก และตัวร้าย กล่าวคือเขาคอยช่วยเหลือV3เป็นระยะๆ แต่ก็ยังภักดีอยู่กับองค์กรเดสตรอน (แต่เกลียดโยโร่ยเก็นซุย) แต่ในช่วงท้ายของเรื่องเขายอมสละชีวิตเพื่อหยุดยั้งระเบิดโปรตอน และ V3 ได้ยอมรับเขาในฐานะ "คาเมนไรเดอร์คนที่ 4" (แต่ในภาษาไทยจะพูดว่าไอ้มดแดง V4)ตัวละครอื่นๆตัวละครอื่นๆ. - ทาจิบานะ โทเบ: ชายวัยทองที่ออกตามหาสิ่งมหัศจรรย์ โดยคราวนี้ เขาได้เป็น เป็นพี่เลี้ยงให้กับ ชิโร่ คอยดูและและฝึกฝนคาเมนไรเดอร์เกือบทุกคน ในภาคV3นี้เขาเปิดร้านขายอุปกรณ์กีฬา- ทามะ จุนโกะ: หญิงสาวที่ถูกชิโร่ช่วยไว้ และ ได้ย้ายเข้ามาอยู่ กับชิโร่ กับ ทาจิบานะ- ยุวชนไรเดอร์ : กลุ่มเด็กๆที่ทำหน้าที่สอดส่องดูแลเหตุการณ์ร้ายและแจ้งให้คาซามิทราบ - ทาคิ คัตสึยะ: สายรับประจำ (เอฟบีไอ)และเป็นที่ปรึกษา กับ ชิโร่ กับ ทาจิบานะเดสตรอนเดสตรอน. - ผู้นำของเดสตรอน - ด็อคเตอร์ จี / คานิเลเซอร์ - บารอน คิบะ / แมมอธแวมไพร์ - เจ้านิกาย ซึบาสะ / ค้างคาวซอมบี้ - จอมพลชุดเกราะ / คานิเซริงานิ - สมุนเดสตรอน</doc>
