<doc id="945731" url="https://th.wikipedia.org/wiki?curid=945731" title="คลาวด์ เอชเอ็ม">คลาวด์ เอชเอ็ม คลาวด์ เอชเอ็ม หรือ บริษัท คลาวด์ เอชเอ็ม จำกัด (อังกฤษ: Cloud HM ชื่อย่อ: CHM) เป็นกลุ่มบริษัทชั้นนำที่ให้บริการบริการระบบ Cloud Services ที่ตั้งอยู่ภายในประเทศไทย โดยก่อตั้งขึ้นเมื่อปี พ.ศ. 2557ประวัติ ประวัติ. ในปีพุทธศักราช 2557 บริษัท คลาวด์ เอชเอ็ม จำกัด (Cloud HM) ได้ก่อตั้งขึ้น เป็นบริษัทที่ให้บริการ Cloud Services ชั้นแนวหน้าในประเทศไทย โดยที่มาของชื่อบริษัท คลาวด์ เอชเอ็ม จำกัด (Cloud HM) นั้น คำว่า 'คลาวด์' (Cloud) มีที่มาจากบริการของเราที่ให้บริการ คลาวด์เซอร์วิส (Cloud Service) แก่ลูกค้า คำว่า 'เอชเอ็ม' (HM) ได้ย่อมาจาก 'Hybrid Managed' ซึ่งณ ปัจจุบัน ระบบคลาวด์จะไม่เป็นเพียงแค่การให้บริการแบบ Private หรือ Public อย่างใดอย่างหนึ่งอีกต่อไป จะมีการควบรวมการใช้งานทั้งสองรูปแบบเข้าด้วยกันในรูปแบบ 'Hybrid' ส่วนคำว่า 'Managed' โดยปกติทางบริษัท คลาวด์ เอชเอ็ม จำกัด จะเป็นผู้ให้บริการดูแลรับผิดชอบระบบไอทีให้แก่ลูกค้าทั้งหมด โดยที่ลูกค้าไม่ต้องเสียเวลาในการดูแลเอง บริษัท คลาวด์ เอชเอ็ม จำกัด นั้นมีสถานะเป็นบริษัทลูกของ ยูไนเต็ด อินฟอร์เมชั่น ไฮเวย์ (UIH) ที่อยู่ในกลุ่มเครือ เบญจจินดา โฮลดิ้ง กรุ๊ป ผู้ก่อตั้ง บริษัท โทเทิ่ล แอ็คเซ็ส คอมมูนิเคชั่น จำกัด (มหาชน) หรือ DTACบริการของ Cloud HM   บริการของ Cloud HM  . บริการที่ Cloud HM ให้บริการอยู่ มีทั้งหมด 5 บริการ1. Enterprise Cloud เป็นการให้บริการ Infrastructure as a Service (IaaS) ในรูปแบบ Virtual Machine (VM) โดยที่สามารถเพิ่ม/ลด ทรัพยากรได้ตามต้องการ โดยมุ่งเน้นที่กลุ่มลูกค้าที่เป็นกลุ่ม Enterprise และ SME เป็นหลัก 2. DevOps Cloud เป็นการให้บริการ Infrastructure as a Service (IaaS) ในรูปแบบ Virtual Machine (VM) เช่นเดียวกับบริการ Enterprise Cloud แตกต่างกันตรงที่ ระบบ DevOps Cloud จะมาในรูปแบบ Self-Service สามารถบริหารจัดการได้เองทั้งหมด ผ่านหน้าเว็บ Portal รวมไปถึงการบริหารจัดการ Network, Firewall และอื่น ๆ 3. cloudBackup เป็นการให้บริการ Backup as a Service (BaaS) ระบบเก็บข้อมูลสำรองไว้บนระบบคลาวด์ 4. cloudDisasterRecovery เป็นการให้บริการ Disaster Recovery as a Service (DRaaS) เปรียบเสมือนระบบสำรองให้แก่ระบบหลัก หากระบบหลักเสีย สามารถใช้งาน ระบบสำรองได้แทน 5. cloudShare เป็นการให้บริการระบบ Share File ที่ให้แก่องค์กร มีความปลอดภัยสูงกว่า เนื่องจากใช้เซิร์ฟเวอร์ภายในองค์กรของคุณเอง 6. LogicMonitor เป็นการให้บริการ Software as a Service (SaaS) ในการมอนิเตอร์ระบบให้แก่องค์กรรางวัลที่ได้รับ รางวัลที่ได้รับ.
