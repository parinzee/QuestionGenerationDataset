<doc id="205256" url="https://th.wikipedia.org/wiki?curid=205256" title="เค้าร่างเอกซ์เอ็มแอล">เค้าร่างเอกซ์เอ็มแอล เค้าร่างเอกซ์เอ็มแอล () คือวิธีการที่จะใช้จำแนกชนิดของเอกสารแบบเอกซ์เอ็มแอล โดยจะบอกถึงโครงสร้างและชนิดของข้อมูลในเอกสารเอกซ์เอ็มแอลแต่ละชนิด โดยตัวเค้าร่างเองมีหน้าตาเหมือนเนื้อหาเอกซ์เอ็มแอลตามปรกติเช่น XML Schema (W3C) และ RELAX NG หรือนิยามขึ้นโดยภาษาพิเศษโดยเฉพาะและเป็นส่วนหนึ่งของเอกซ์เอ็มแอลเช่น Document Type Definition (DTD) ซึ่งพัฒนาขึ้นก่อนแบบอื่นที่กล่าวมาข้างต้นโดยพัฒนามาจาก SGML โดยมีข้อจำกัดเช่นการไม่รองรับ XML Namespaceเค้าร่างเอกซ์เอ็มแอลต่าง ๆเค้าร่างเอกซ์เอ็มแอลต่าง ๆ. - Document Definition Markup Language (DDML) - Document Schema Definition Languages (DSDL) - Document Structure Description (DSD) - Document Type Definition (DTD) - Namespace Routing Language (NRL) - RELAX NG โดยเป็นการรวม RELAX และ TREX เข้าด้วยกัน - SGML - Schema for Object-Oriented XML (SOX) - Schematron - XML-Data Reduced (XDR) - XML Schema (W3C) (WXS or XSD)</doc>
