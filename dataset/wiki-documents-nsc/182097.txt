<doc id="182097" url="https://th.wikipedia.org/wiki?curid=182097" title="ไอซีเอ็มพี">ไอซีเอ็มพี เกณฑ์วิธีข้อความควบคุมบนอินเทอร์เน็ต หรือ ไอซีเอ็มพี () คือ โพรโทคอลที่ทำหน้าที่รายงานความผิดพลาดต่างๆ ใน IP packet และตรวจสอบการทำงานในชั้น Internet Layer เนื่องจากปกติข้อมูลที่ส่งใน Network จะต้องผ่าน Router มากกว่าหนึ่งตัว เพื่อให้ Router สามารถตรวจสอบปัญหาในการส่งข้อมูลไปยังปลายทาง Router จะใช้ Internet Control Message Protocol (ICMP) ไปแจ้งยัง IP ต้นทาง โดย ICMP จะถูกใช้ในการวิเคราะห์เพื่อแก้ไขปัญหา ข้อความใน ICMP จะประกอบด้วย- Echo Request and Echo Reply - Source Quench - Destination Unreachable - Time Exceeded - Fragmentation Needed</doc>
