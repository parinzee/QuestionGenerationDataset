<doc id="515178" url="https://th.wikipedia.org/wiki?curid=515178" title="มาสค์ไรเดอร์ x ซูเปอร์เซนไท x ตำรวจอวกาศ ซูเปอร์ฮีโร่ไทเซน Z">มาสค์ไรเดอร์ x ซูเปอร์เซนไท x ตำรวจอวกาศ ซูเปอร์ฮีโร่ไทเซน Z มาสค์ไรเดอร์ x ซูเปอร์เซนไท x ตำรวจอวกาศ X อุลตร้าแมน ซูเปอร์ฮีโร่ไทเซน Z () เป็นภาพยนตร์ที่นำ 4 ภาพยนตร์โทคุซัทสึซีรีส์ระหว่าง ซูเปอร์เซนไท, มาสค์ไรเดอร์ ซีรีส์, อุลตร้าแมน ซีรีส์ และเมทัลฮีโร ซีรีส์ ผลิตโดยโตเอะคัมปานี เริ่มฉายในโรงภาพยนตร์ในญี่ปุ่นเมื่อวันที่ 27 เมษายน ค.ศ. 2018เรื่องย่อ เรื่องย่อ. วายร้ายแห่งจักรวาลอย่างองค์กรชอคเกอร์ได้กลับมาอีกครั้ง คราวนี้มันกลับมาพร้อมกับพลังอันยิ่งใหญ่และขนานนามตัวเองว่า "สเปซชอคเกอร์" ซึ่งนอกจากพลังวิทยาศาสตร์อันล้ำยุคของพวกมันแล้ว สเปซชอคเกอร์ยังได้ครอบครองอำนาจเวทมนตร์สุดมหัศจรรย์เอาไว้อีกด้วย ในขณะเดียวกันสหพันธ์ตำรวจแห่งจักรวาลได้มีคำสั่งให้ "เกคิ จูมอนจิ" หรือ "เกียบัน Type G" ลงมาบนโลกเพื่อทำภารกิจจับเป็น "โซมะ ฮารูโตะ" หรือคาเมนไรเดอร์วิซเซิร์ดนั่นเอง แต่เมื่อการต่อสู้ของตำรวจอวกาศและไรเดอร์จอมเวทย์จบลง พวกเค้าก็พบว่าสเปซชอคเกอร์นั้นอยู่เบื้องหลังเรื่องราวทั้งหมด อีกด้านหนึ่งโกบัสเตอร์ก็ได้พบกับหุ่นยนต์ลูกบอลสีเหลืองที่ตกมาจากอวกาศอย่าง "ไซโครอน" ดูเหมือนว่าไซโครอนจะเป็นผู้กุมความลับของพวกสเปซชอคเกอร์เอาไว้ และในตอนนั้นเองพวกสเปซชอคเกอร์ก็ควบคุมพลังเวทมนตร์เอาไว้ไม่อยู่ ผลจากความบ้าคลั่งนั้นเองทำให้โลกกำลังจะถึงจุดจบในไม่ช้าตัวละครตัวละครฝั่งไรเดอร์ตัวละคร. ตัวละครฝั่งไรเดอร์. - มาสค์ไรเดอร์วิซเซิร์ด / โซมะ ฮารุโตะ (操真晴人) แสดงโดย ชิราอิชิ ชุนยะ (白石 隼也) - มาสค์ไรเดอร์บีสต์ / นิโต โคสุเกะ (仁藤 攻介) แสดงโดย นางาเซะ ทาซุคุ (永瀬 匡) - คาเมนไรเดอร์หมายเลข 1 พากย์โดย อินาดะ เทตสึ - คาเมนไรเดอร์แอเมซอน พากย์โดย เซกิ โทโมคาสุ - เคียวริวเรด / คิริว ไดโก (桐生 ダイゴ) แสดงโดย ริวเซย์ เรียว (竜星 涼) - เคียวริวแบล็ค / เอียน ยอรค์แลนด์ (イアン・ヨークランド) แสดงโดย ไซโต ชูสุเกะ (斉藤 秀翼) - เคียวริวบลู / อุโดะ โนบุฮารุ (有働 ノブハル) แสดงโดย คินโจ ยามาโตะ (金城 大和) - เคียวริวกรีน / ริปปูกัง โชจิ (立風館 ソウジ) แสดงโดย ชิโอโนะ อากิฮาระ (塩野 瑛久) - เคียวริวพิงค์ / เอมี่ ยูสึกิ (アミィ 結月) แสดงโดย คอนโนะ อายูริ (今野 鮎莉) - เคียวริวโกลด์ พากย์โดย มารุยามะ อัทสึชิ (丸山敦史) - เรดบัสเตอร์ / ซากุราดะ ฮิโรมุ (桜田 ヒロム) แสดงโดย ซึซึกิ คาสึฮิโระ - บลูบัสเตอร์ / อิวาซากิ ริวจิ (岩崎 リュウジ) แสดงโดย บาบะ เรียวมะ - เยลโล่บัสเตอร์ / อุซามิ โยโกะ (宇佐見 ヨーコ) แสดงโดย อาริสะ โคมิยะ - สแต๊กบัสเตอร์ / บีท J สแต๊ก (ビート・J・スタッグ) พากย์โดย นาคามุระ ยูอิจิ (中村悠一) -
