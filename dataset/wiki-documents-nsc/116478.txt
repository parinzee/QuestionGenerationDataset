<doc id="116478" url="https://th.wikipedia.org/wiki?curid=116478" title="สมการกำลังสาม">สมการกำลังสาม ในทางคณิตศาสตร์ สมการกำลังสาม คือสมการของพหุนามตัวแปรเดียวที่มีดีกรีเท่ากับ 3 รูปแบบทั่วไปของสมการกำลังสามคือ เมื่อ a ≠ 0 (ถ้า a = 0 สมการนี้จะกลายเป็นสมการกำลังสอง) โดยปกติแล้ว formula_2 คือสัมประสิทธิ์ที่เป็นจำนวนจริง ฟังก์ชันของสมการกำลังสามสามารถวาดกราฟบนระบบพิกัดคาร์ทีเซียนได้รูปเส้นโค้งคล้ายตัว S หรือ Nดิสคริมิแนนต์ ดิสคริมิแนนต์. สมการกำลังสามทุกสมการที่มีสัมประสิทธิ์เป็นจำนวนจริง จะมีรากของสมการ 3 คำตอบเสมอ ซึ่งจะต้องมีจำนวนจริงอย่างน้อยหนึ่งจำนวนที่เป็นคำตอบ ตามทฤษฎีบทค่าระหว่างกลาง (intermediate value theorem) และคำตอบเหล่านั้นอาจจะเท่ากันบางค่าก็ได้ ส่วนอีกสองจำนวนที่เหลือสามารถแยกแยะได้จากการพิจารณาดิสคริมิแนนต์ ซึ่งคำนวณจาก คำตอบของสมการจะเป็นประเภทใดประเภทหนึ่ง ดังต่อไปนี้- ถ้า Δ < 0 คำตอบของสมการจะเป็นจำนวนจริงทั้งสามค่า ที่แตกต่างกันทั้งหมด - ถ้า Δ > 0 คำตอบของสมการจะมีหนึ่งค่าที่เป็นจำนวนจริง และอีกสองจำนวนเป็นจำนวนเชิงซ้อนสังยุคซึ่งกันและกัน - ถ้า Δ = 0 คำตอบของสมการจะเป็นจำนวนจริงทั้งสามค่า ซึ่งมีสองจำนวนเป็นค่าเดียวกัน หรือ เป็นค่าเดียวกันทั้งสามจำนวน อย่างใดอย่างหนึ่งสูตรกำลังสาม สูตรกำลังสาม. ถ้าหาก formula_4 เป็นคำตอบของสมการกำลังสามแล้ว เราจะสามารถแยกตัวประกอบของพหุนามกำลังสามได้ดังนี้ กำหนดให้ คำตอบของสมการทั้งสามค่าสามารถคำนวณได้จากสูตร เมื่อ i คือหน่วยจินตภาพที่นิยามโดย i = −1</doc>
