<doc id="3150" url="https://th.wikipedia.org/wiki?curid=3150" title="โพรโทคอล">โพรโทคอล โพรโทคอล () หรือศัพท์บัญญัติว่า เกณฑ์วิธี คือข้อกำหนดซึ่งประกอบด้วยกฎต่าง ๆ สำหรับรูปแบบการสื่อสารเฉพาะรูปแบบใดรูปแบบหนึ่ง เพื่อให้การติดต่อสื่อสารในระบบเครือข่าย ทำงานได้ด้วยกันทั้งระบบ คล้ายกับมนุษย์สามารถใช้ภาษาอังกฤษเป็นภาษากลางในการสื่อสารถึงกันได้ชุดของโพรโทคอล ชุดของโพรโทคอล. ชุดโพรโทคอลต่อไปนี้ คือชุดโพรโทคอลสำคัญ ซึ่งเป็นใช้เป็นต้นแบบ ในการใช้งานต่างๆ แบ่งได้เป็น 2 มาตรฐานดังนี้ คือ:1. มาตรฐานเปิด- Internet protocol 2. มาตรฐานปิด- AppleTa - DECnet - IPX/SPX - SMB - Systems Network Architecture (SNA) - Distributed Systems Architecture (DSA)โพรโทคอลที่สำคัญโพรโทคอลที่สำคัญ. - HTTP (Hyper Text Transfer Protocol) - POP3 (Post Office Protocol 3). - SMTP (Simple Mail Transfer Protocol). - FTP (File Transfer Protocol). - IP (Internet Protocol). - DHCP (Dynamic Host Configuration Protocol). - IMAP (Internet Message Access Protocol</doc>
