<doc id="576410" url="https://th.wikipedia.org/wiki?curid=576410" title="รถไฟใต้ดินฮัมบูร์ก">รถไฟใต้ดินฮัมบูร์ก รถไฟใต้ดินฮัมบูร์ก () เป็นเส้นทางรถไฟใต้ดินที่ให้บริการในเขตเมืองฮัมบูร์กและปริมณฑล ซึ่งเชื่อมต่อกับรถไฟเร็วเขตเมืองฮัมบูร์ก ดำเนินการโดย Hamburger Hochbahn ในส่วนของ Hamburger Verkehrsverbund (HVV) เปิดให้บริการครั้งแรกเมื่อเดือนกุมภาพันธ์ ค.ศ. 1912 จำนวน 4 เส้นทาง 91 สถานี ระยะทางรวมการพัฒนาโครงข่ายเส้นทาง เส้นทาง. ประกอบด้วยสี่เส้นทาง ดังนี้:สถานีรถไฟฟ้าปัจจุบันรถไฟฟ้า. ปัจจุบัน. - HHA Type DT3 (ตั้งแต่ปี ค.ศ. 1968), some trains are current - HHA Type DT3-E (ตั้งแต่ปี ค.ศ. 1968 ส่วนที่ซ่อมแซมใหม่ตั้งแต่ปี ค.ศ. 1994) - HHA Type DT4 (ตั้งแต่ปี ค.ศ. 1988) - HHA Type DT5 (ตั้งแต่ปี ค.ศ. 2012)อดีตอดีต. - HHA Type A/B/T/TU1/TU2 (Type T ตั้งแต่ปลายคริสต์ทศวรรษ 1950) (ตั้งแต่ปี ค.ศ. 1912 จนถึงปี ค.ศ. 1970) - HHA Type DT1 (ตั้งแต่ปี ค.ศ. 1958 จนถึงปี ค.ศ. 1991) - HHA Type DT2 (ตั้งแต่ปี ค.ศ. 1962 จนถึงปี ค.ศ. 2012) - HHA Type DT3 (ส่วนที่ไม่ได้ซ่อมแซมใหม่) (ตั้งแต่ปี ค.ศ. 1968 จนถึงปี ค.ศ. 2003)</doc>
