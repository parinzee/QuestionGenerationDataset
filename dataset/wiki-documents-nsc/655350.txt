<doc id="655350" url="https://th.wikipedia.org/wiki?curid=655350" title="รายชื่อฮอลโลว์ในเทพมรณะ">รายชื่อฮอลโลว์ในเทพมรณะ = ฮอลโลว์ = ฮอลโลว์ (虚 (ホロウ),Hollow ; ความว่างเปล่า) 1. กิลเลียน (最下大虚 (ギリアン),Gillian) เมนอสที่พวกอิจิโกะเรียกกันว่า "เมนอสกรังเด้" นั้นแท้จริงแล้วคือพวก "กิลเลียน" เป็นเมนอสชั้นต่ำที่สุด ร่างใหญ่ เคลื่อนไหวช้า มีปัญญาราวกับสัตว์ป่าเท่านั้น 2. แอดจูคาส (中級大虚 (アジューカス),Adjuchas )เมนอสที่มีขนาดเล็กกว่ากิลเลียน มักเป็นผู้นำฝูงของกิลเลียน และมีขนาดเล็กกว่า ฉลาดกว่า 3. วาสโทรเด้ (最上大虚 (ヴァストローデ),Vasto Lorde) เมนอสชั้นสูงสุด มีขนาดเท่ากับมนุษย์ มีจำนวนน้อย รวมทั้งหมดในฮูเอโก้ มุนเด้ มีอยู่ไม่กี่ตนเท่านั้น ว่ากันว่า ความสามารถของวาสโทรเด้นั้นมีมากกว่าหัวหน้าหน่วยของหน่วยของยมทูตเสียอีก และฮิตสึกายะ โทชิโร่เคยพูดไว้ว่าถ้ามีวาสโทรเด้ อยู่กับไอเซ็นตั้งแต่10ตนขึ้นไปโซลโซไซตี้อวสานแน่ = ตัวละครฮอลโลว์ = = ฮูเอโกมุนโด้ = ฮูเอโกมุนโด้ (虚圏 (ウェコムンド), Hueco Mundo) : ภาษาสเปนแปลว่า โลกที่ว่างเปล่า ภาษาญี่ปุ่นแปลว่า ขอบเขตว่างเปล่า ภูมิทัศน์ของ ฮูเอโกมุนโด้ เป็นทะเลทรายสีขาวที่ดูเหมือนไม่มีที่สิ้นสุดด้วยความหลากหลายของเนินทรายที่เกลื่อนไปด้วยก้อนหิน ใน ฮูเอโกมุนโด้ เต็มไปด้วยอณูวิญญาณความเข้มข้นสูงของจึงช่วยให้ ฮอลโลว์ ขนาดเล็กได้รับสารอาหาร สถานที่สำคัญ ใน ฮูเอโกมุนโด้ มีดังนี้อารันคาร์ อารันคาร์. อารันคาร์ (破面 (アランカル), arrancar) : ภาษาสเปนแปลว่า ฉีกออก ภาษาญี่ปุ่นแปลว่า หน้ากากที่ถูกฉีกออก สิ่งที่แตกต่างกันของอารันคาร์ มีดังนี้ 1.รูกลวง อารันคาร์แต่ละคนจะมีรูกลวง เหมือนตอนยังเป็นฮอลโลว์ อยู่ทุกคนแต่ตำแหน่งของรูกลวงต่างกัน เช่น อุคิโอล่าอยู่ตรงอก แต่นอยโทร่าอยู่ตรงตา 2.รูปร่าง อารันคาร์บางคนมีชิ้นส่วนของตอนเป็นฮอลโลว์แตกต่างกัน บางคนมีเหลือน้อยมาก 3.เอสติกม่า ลวดลายบนใบหน้าของอารันคาร์ เป็นไปได้ว่าเป็นส่วนที่เหลือของหน้ากากและเป็นเอกลักษณ์ของแต่ละตน บางตนลวดลายอาจจะปรากฏเฉพาะตอนรีเซอร์เรคชั่นพลัง และ ความสามารถ พลัง และ ความสามารถ. อารันคาร์จะมีพลังเฉพาะตัวอยู่แล้ว ตามพลังของแต่ละคนจะใช้ซีโร่ได้ทุกคน 1.บาร่า (虚弾 (バラ), Bara,Bala) : ภาษาสเปนแปลว่า กระสุน ภาษาญี่ปุ่นแปลว่า กระสุนฮอลโลว์ 2.ซีโร่ (虚閃 (セロ), Sero,Cero): ภาษาสเปนแปลว่า ศูนย์ ภาษาญี่ปุ่นแปลว่า ลำแสงฮอลโลว์ 3.เดสโคเลอร์ (解空(デスコレール), Desukorēru, Descorrer) 4.ฮิเอโร่ (鋼皮 (イエロ),Hierro) ภาษาสเปนแปลว่า เหล็ก ภาษาญี่ปุ่นแปลว่า ผิวเหล็กกล้า 5.การคืนสภาพขั้นสูงสุด (超速再生, Chousoku Saisei) 6.เพสควิส(探査回路 (ペスキサ), Pesquisa) ภาษาสเปนแปลว่า การสืบสวน ภาษาญี่ปุ่นแปลว่า วงจรสืบหา 7.รีเซอร์เรคชั่น (帰刃 (レスレクシオン), Resurrección) ภาษาสเปนแปลว่า การฟื้นคืน ภาษาญี่ปุ่นแปลว่า ดาบหวนคืน 8.โซนีดโด้ (響転 (ソニード), Sonīdo) การเคลื่อนที่ที่เพิ่มความเร็วของอารันคาร์ เที่ยบเท่า ก้าวพริบตา ของยมทูต 9.คาจา เนกาเชี่ยน (反膜の匪 (カハ・ネガシオン), Caja Negación) เป็นการสร้างกล่องดำเล็ก แล้วใส่เข้าไปในรูกลวงของอารันคาร์ใต้บังคับบัญชา ทำให้อารันคาร์ที่ โดนจะเข้าไปอยู่ในอีกมิตินึงซึ่งไม่มีทางออกและจะอยู่ในมิตินั้นไปตลอดกาล พลังนี้สามารถใช้กับเอสปาด้าได้แต่จะขังไว้ในมิตินั้นได้ไม่นานก็หลุดมาแล้ว เดิมทีเอสปาด้าเอาไว้ใช้ลงโทษลูกน้อง 10.กรังเรย์ ซีโร่ (王虚の閃光 (グラン・レイ・セロ), Gran Rey Cero) เป็นพลังซีโร่ขนาดใหญ่ ที่มีพลังต่างจากซีโร่ธรรมดามากเอสปาด้า เอสปาด้า. เอสปาด้า (十刃 (エスパーダ), Esupāda,The Espada (ตัวอักษรสเปนแปลว่า ดาบ ตัวอักษรญี่ปุ่นแปลว่า ดาบทั้งสิบ )- ยามี่ ริยัลโก้ ( ヤミー リヤルゴ Yammy Riyalgo)- ชื่อเต็ม ยามี่ ริยัลโก้ - เอสปาด้าหมายเลข 10 (เมื่อปลดปล่อยดาบฟันวิญญาณแล้วจะเป็น เอสปาด้าหมายเลข 0) - ชื่อดาบฟันวิญญาณ อีร่า (憤獣 (イーラ),Ira ; ตัวอักษรสเปนแปลว่า พิโรธ ตัวอักษรญี่ปุ่นแปลว่า สัตว์ร้ายพิโรธ) - คำปลดปล่อย จงขยี้ให้แหลก (ญี่ปุ่น: ブチ切れろ, buchikirero) - ลักษณะ เคยไปโลกมนุษย์กับอุลคิโอร่า และเขาก็ดูดกลืนวิญญาณของคนมากมายไปด้วย ถูกอุราฮาร่าเล่นงานประจำ ภายหลังเขากลับมาอีกครั้งโดยมีขนาดตัวใหญ่กว่าเดิมและประกาศว่า ตนคือเอสปาด้าหมายเลข 0 สาเหตุการตาย"ความโกรธเกรี้ยว" (เช่นการสู้อย่างบ้าคลั่ง) - ความสามารถ เขาจะมีรูปร่างขนาดใหญ่มาก พลังทั่วไปของเขา การต่อย ฮิเอโร่ หรือ ซีโร่ จะเพิ่มสูงขึ้นมาก- โคโยเต้ สตาร์ค ( コヨーテ・スターク Coyote Starrk)- ชื่อเต็ม พริเมร่า เอสปาด้า โคโยเต้ สตาร์ค ลิลิเน็ต จินเจอร์แบ็ค - เอสปาด้าหมายเลข 1 - ชื่อดาบฟันวิญญาณ ลอส โลบอส (群狼 (ロス・ロボス) , Los Lobos ; ตัวอักษรสเปนแปลว่า หมาป่า ตัวอักษรญี่ปุ่นแปลว่า ฝูงหมาป่า) - คำปลดปล่อย จงไล่ต้อนให้แตกกระจาย (ญี่ปุ่น: 散らせ, kechirase) - ลักษณะ เมื่อปลดปล่อยดาบฟันวิญญาณ เขาจะมีปืนพกสองกระบอกถืออยู่ ปืนเหล่านี้คือ ลิลิเน๊ต ปืนพกทั้งสองเหมือนร่างกายเธอ เธอสามารถเห็นได้พูดผ่านปืนทั้งสองเช่นเดียวกับความรู้สึกเก็บความรู้สึก ทั้งหมด ส่วนหนึ่งของหน้ากาก ของลิลิเน็ตปรากฏบนหัวของ คโยเต้ สตาร์ค ยื่นออกมาจากด้านหลังของศีรษะของเขาไปรอบ ๆ เพื่อปิดตาซ้ายของเขา ชุดของเขาจะครอบคลุมในเสื้อขนสัตว์สีเทาเรียงราย ขาของเขาจะครอบคลุมในชนิดของกางเกงรัดรูปสีดำและมีขนสัตว์ประดับอยู่ ปืนพกสองกระบอกของเขาสามารถยิง ซีโร่ (zero) ออกมาได้ โดยสามารถยิงได้ถึง 1,000 นัด ต่อการยิงเพียงครั้งเดียว เขาถูก เคียวราคุ ฆ่าตาย - ความสามารถ 1.ซีโร่ เมทราเจ็ตต้า(แสงฮอลโลว์กระสุนไร้ขีดจำกัด) (無限装弾虚閃 (セロ・メトラジェッタ),Cero Metralleta) เป็นแสงซีโร่จำนวนมากจากการยิงเพียงครั้งเดียว 2.โคมิโร่ (ดาบแสง)(コルミージョ, Colmillo) สร้างดาบแสงออกมา สามารถใช้ 2 ดาบได้ 3.ฝูงหมาป่า(Wolves) สร้างหมาป่าแสงจำนวนมาก ที่เมื่อพวกมันกัดสิ่งใดก็ตาม มันจะระเบิดอย่างรุนแรง- บารากัน ลุยเซนบาร์น ( バラガン・ルイゼンバーン Baraggan Louisenbairn)- ชื่อเต็ม เซกุนโด้ เอสปาด้า บารากัน ลุยเซนบาร์น - เอสปาด้าหมายเลข 2 - ชื่อดาบฟันวิญญาณ อาโรกานเต้ (髑髏大帝 (アロガンテ) ,Arrogante ; ตัวอักษรสเปนแปลว่า เย่อหยิ่ง ตัวอักษรญี่ปุ่นแปลว่า ราชันหัวกะโหลก - คำปลดปล่อย จงเปื่อยสลาย (ญี่ปุ่น: 朽ちろ, kuchiro) - ลักษณะ เป็นชายแก่มีแผลผ่าทแยงมุมที่หน้าเขา ชอบที่จะให้ลูกน้องของตนพูดกับตนเปรียบเสมือนว่าตนเป็นราชาผู้ยิ่งใหญ่ ดาบของบารากันมีลักษณะเป็นขวานขนาดใหญ่ โดยหยิบออกมาจากที่บัลลังค์ของตนเอง และเวลาปลดปล่อยจะมีออร่าสีดำห่อหุ้มตัวกลายเป็นราชาหัวกะโหลก สุดท้ายโดนฮัตส์จากไวเซิร์ดจัดการ สาเหตุการตาย"ความแก่ชรา" - ความสามารถ 1. เรสปิร่า (ลมหายใจมรณะ)(死の息吹 (レスピラ), Respira) เป็นพิษสีดำอมม่วง สิ่งใดที่โดนจะเปิ่อยสล่ยและสูญสิ้น 2.กรังด์ ไกด้า (ขวานสูญสื้น)(滅亡の斧 (グラン·カイダ) Guran Kaida) เป็นขวานขนาดใหญืสีดำ ที่สามารถปล่อย เรสปิร่า ได้ด้วยการควงขวาน- เทียร์ ฮาริเบล ( ティア・ハリベル Tier Harribel)- ชื่อเต็ม เทรส เอสปาด้า เทีย ฮาริเบลล์ - เอสปาด้าหมายเลข 3 - ชื่อดาบฟันวิญญาณ ทีบูรอน (皇鮫后 (ティブロン), Tiburon ; ตัวอักษรสเปนแปลว่า ฉลาม ตัวอักษรญี่ปุ่นแปลว่า ฉลามจักรพรรดิ) - คำปลดปล่อย จงจู่โจม (ญี่ปุ่น: 討て, ute) - ลักษณะ เป็นผู้หญิงผิวสีคล้ำ ดูท่าทางสงบเงียบ ไม่ประมาทต่อศัตรู และยังเป็นเอสปาด้าตนเดียวที่เป็นผู้หญิงในตอนนี้ เมื่อปลดปล่อยแล้วจะมีลักษณะไม่ต่างจากเดิมมากนัก หน้ากากที่ปากจะหายไป และมีดาบขนาดใหญ่อยู่ที่มือสามารถควบคุมกระแสน้ำได้อย่างอิสระและเวลาปล่อย ซีโร่เธอจะตวัดดาบและมีซีโร่ออกมา นับว่าเป็นเอสปาด้าหญิงที่มีฝีมือมาก สุดท้ายถูกหักหลังด้วยฝีมือของไอเซ็น และกลายเป็นผู้ปกครองฮูเอโก้มุนโด้ในเวลาต่อมา สาเหตุการตาย"การเสียสละ" - ความสามารถ 1.ออร่า อาซุล (ปืนใหญ่คลื่นสีคราม) (波蒼砲 (オーラ·アズール) ,Ola Azul; ภาษาสเปนแปลว่า "บลูเวฟ", ภาษาญี่ปุ่นแปลว่า "ปืนบลูเวฟ") : เธอสามารถรวบรวมคลื่นน้ำสีเหลืองที่ดาบของเธอแล้วปล่อยออกมา 2.ลา โกต้า(พิรุณสังหาร) (豹鉤 (ガラ・デ・ラ・パンテラ,Garra de la Pantera ): ยิงกระสุนน้ำที่มีความรุนแรงใส่ศัตรู 3.อิร์ เวียนโด้ (กระแสสมุทรแผดเผา) (灼海流 (イルビエンド),Hirviendo ) : เป็นการทำให้ สิ่งที่ประกอบด้วย น้ำ เช่น น้ำแข็ง เดือด จนกลายเป็นสถานะของเหลว แล้วมาล้อมรอบตัวเธอไว้ 4.คาส เคด้า (สะบั่นวารีกัมปนาท)(断瀑 (カスケーダ) ,Cascada ; ภาษาสเปนแปลว่า "น้ำตก" ภาษาญี่ปุ่นแปลว่า "ตัดน้ำตก") :เป็นการรวบรวมน้ำปริมาณมหาศาลแล้วโจมตีใส่ศัตรูด้วยความรุนแรง- เนเลียล โท โอเดลชูเวงก์(ネリエル・トゥ・オーデルシュヴァンク Nelliel Tu Odelschwanck)- อดีตเอสปาด้าหมายเลข 3 - ชื่อดาบฟันวิญญาณ กามูซ่า (羚騎士 (ガミューサ), Gamuza ; ตัวอักษรสเปนแปลว่า แพะ ตัวอักษรญี่ปุ่นแปลว่า อัศวินแอนทิโลป ) - คำปลดปล่อย จงขับขาน (ญี่ปุ่น: 謳え, utae) - ลักษณะ เป็นอาร์รันคาร์เด็ก
