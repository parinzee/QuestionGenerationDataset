<doc id="153332" url="https://th.wikipedia.org/wiki?curid=153332" title="ซิงเกิลตัน">ซิงเกิลตัน ซิงเกิลตัน () ในทางคณิตศาสตร์หมายถึง เซตที่มีสมาชิกตัวเดียวและเพียงหนึ่งเดียว ตัวอย่าง เซต { 0 } เป็นซิงเกิลตัน มีสมาชิกตัวเดียวคือ 0คุณสมบัติ คุณสมบัติ. เซตของเซตซึ่งมีเซตตัวเดียวเป็นสมาชิกก็ถือว่าเป็นซิงเกิลตัน เช่น { {1,2,3} } มีสมาชิกเพียงตัวเดียวคือ {1,2,3} ถึงแม้สมาชิกนี้จะไม่ใช่ซิงเกิลตันก็ตาม ในทฤษฎีเซตเชิงสัจพจน์ การมีขึ้นของซิงเกิลตันนั้นเป็นผลมาจากสัจพจน์ของเซตว่าง (axiom of empty set) และสัจพจน์ของการจับคู่ (axiom of pairing) กล่าวคือเมื่อเซตว่าง { } จับคู่กับเซตว่าง { } จะให้ผลเป็น { { } } ซึ่งเป็นซิงเกิลตัน แต่ตัวเซตว่างเองนั้นไม่ใช่ ถ้าให้ A เป็นเซตใดๆ และ S เป็นซิงเกิลตันใดๆ ดังนั้นจะมีฟังก์ชันเพียงหนึ่งเดียวจาก A ไปยัง S เนื่องจากฟังก์ชันดังกล่าวจะให้ผลเป็นสมาชิกตัวเดียวใน S ไม่ว่าจะส่งค่าสมาชิกใดมาจาก A</doc>
