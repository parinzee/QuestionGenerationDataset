<doc id="57047" url="https://th.wikipedia.org/wiki?curid=57047" title="เชลล์">เชลล์ เชลล์ () ในทางคอมพิวเตอร์ หมายถึง โปรแกรมที่โต้ตอบผู้ใช้ และมักหมายถึงระหว่างผู้ใช้กับระบบปฏิบัติการ สามารถแบ่งได้เป็น แบบชุดคำสั่ง หรือคอมมานด์ไลน์ (command line interface, CLI) และแบบกราฟิก (graphic user interface, GUI) แบบคำสั่งและแบบกราฟิกต่างก็มีข้อดีข้อเสียแตกต่างกัน แบบกราฟิกจะใช้ง่ายและเหมาะกับงานประเภทตกแต่งภาพและวิดีโอ ส่วนแบบคำสั่งเหมาะกับงานบางอย่าง โดยสามารถระบุต้องเจาะจงโดยละเอียดเช่นย้ายไฟล์โดยใช้ไวลด์การ์ด * เป็นต้น เชลล์ที่ใช้ในไมโครซอฟท์วินโดวส์รุ่นหลัง ๆ จะเป็นวินโดวส์เอกซ์พลอเรอร์ ในขณะที่ในรุ่นเก่า ๆ จะเป็นโปรแกรมเมเนเยอร์ สำหรับยูนิกซ์ เชลล์มักหมายถึงเชลล์ยูนิกซ์ ที่ใช้สั่งทางคอมมานด์ไลน์ ตัวอย่างยูนิกซ์เชลล์ที่นิยม เช่น บอร์นเชลล์ คอร์นเชลล์ และ C เชลล์ตัวอย่างเชลล์แบบกราฟิกตัวอย่างเชลล์. แบบกราฟิก. - สำหรับไมโครซอฟท์วินโดวส์ ใช้ วินโดวส์เอกซ์พลอเรอร์ เป็นเชลล์มาตรฐาน แต่มีบริษัทผลิตซอฟต์แวร์มาเป็นทางเลือกแทนวินโดวส์เอกซ์พลอเรอร์ ตัวอย่างเชลล์ทางเลือกเช่น- Geoshell - Litestep - Secure Desktop - สำหรับแมคโอเอส ใช้ แมคอินทอชไฟเดอร์ - สำหรับ X Window System มีทั้ง X window manager โดด ๆ หรือชุดเดสก์ทอปโดยสมบูรณ์เช่น- กโนม - KDE - CDEแบบชุดคำสั่งแบบชุดคำสั่ง. - command.com ในดอส - เชลล์สำหรับภาษาโปรแกรม เช่น ภาษาเพิร์ล ภาษาไพทอน ซื่งสามารถพิมพ์คำสั่งให้ทำงานได้ที่ละคำสั่ง - สำหรับยูนิกซ์และระบบปฏิบัติการใกล้เคียง เช่น ลินุกซ์ บีเอสดี สามารถดูเพิ่มที่บทความ เชลล์ยูนิกซ์</doc>
