<doc id="7510" url="https://th.wikipedia.org/wiki?curid=7510" title="ทฤษฎีเซต">ทฤษฎีเซต ทฤษฎีเซต คือทฤษฎีทางคณิตศาสตร์ที่เกี่ยวกับเรื่องเซต ซึ่งใช้นำเสนอการรวบรวมวัตถุนามธรรม ทฤษฎีเซตเป็นแนวความคิดของการรวบรวมวัตถุในชีวิตประจำวัน และใช้สอนในโรงเรียนประถมศึกษาซึ่งบ่อยครั้งใช้แผนภาพเวนน์เป็นสื่อช่วยสอน ทฤษฎีเซตใช้ภาษาในการอธิบายวัตถุทางคณิตศาสตร์เป็นธรรมเนียมการสอนคณิตศาสตร์สมัยใหม่ ทฤษฎีเซตเป็นหนึ่งในรากฐานทางคณิตศาสตร์ที่ยอมรับกันโดยทั่วไป เหมือนเช่นตรรกศาสตร์และแคลคูลัสภาคแสดง ซึ่งทำให้สามารถสร้างวัตถุทางคณิตศาสตร์ขึ้นมาใหม่โดยใช้ "เซต" และ "ความเป็นสมาชิกของเซต" เป็นตัวนิยาม ทฤษฎีเซตเองนั้นก็เป็นสาขาหนึ่งของคณิตศาสตร์ และยังคงเป็นสาขาที่สำคัญอยู่สำหรับการวิจัยตัวอย่างทฤษฎีเซตตัวอย่างทฤษฎีเซต. - เซต เป็นการแนะนำเข้าสู่ ทฤษฎีเซตมูลฐาน (elementary set theory) - ทฤษฎีเซตสามัญ (naive set theory) - ทฤษฎีเซตเชิงสัจพจน์ (axiomatic set theory) - ทฤษฎีเซตแซร์เมโล (Zermelo set theory) - เซตอย่างหยาบ (rough set) - ทฤษฎีเซตแซร์เมโล-แฟรงเคิล (Zermelo–Fraenkel set theory) - ทฤษฎีเซตฟอนนอยมันน์-แบร์ไนส์-เกอเดิล (Von Neumann–Bernays–Gödel set theory)</doc>
