<doc id="52803" url="https://th.wikipedia.org/wiki?curid=52803" title="เต็มพิกัดสลัดจอมลุย">เต็มพิกัดสลัดจอมลุย เต็มพิกัดสลัดจอมลุย () เป็นการ์ตูนญี่ปุ่น เรื่องและภาพโดย ฮิเดยูกิ โยเนฮาระ เป็นเรื่องราวเกี่ยวกับการผจญภัยของโจรสลัด ที่ตามหาอารยธรรมในตำนานที่มีการกล่าวถึงสัตว์ปีศาจที่มอบพลังให้กับเจ้าของ และเครื่องมือที่สามารถทำให้เรือแล่นสวนกระแสน้ำได้ การ์ตูนเรื่องนี้ได้ลิขสิทธิ์จัดพิมพ์ในประเทศไทยโดย สำนักพิมพ์วิบูลย์กิจ ความยาว 29 เล่มจบเนื้อเรื่อง เนื้อเรื่อง. จอห์น บาธ เด็กชายที่ครั้งหนึ่งเคย อาศัยอยู่ในเรือโจรสลัด เรดสเคล 1ใน 3โจรสลัดที่ยิ่งใหญ่แห่งยุค บัดนี้ เขาได้ออกตามหาอารยธรรมในอดีต นามว่า "ฟอลคอน"พร้อมกับ ดาบใส ของเขา อีกครั้ง ไปเลยโจรสลัดสวีทมาดอนน่า จอห์น บาธ ผู้ที่ใครๆเรียกว่าเครซี่ (บ้า) บาธ เพราะมีลูกเรือน้อยแต่ เก่งกาจ กว่าเรือลำใหญ่ๆบางลำเสียอีก เขาต้องการตามหาอารยธรรมฟอลคอนในตำนาน ที่สูญหายไป พร้อมกับยุคสมัย การค้นหาทำให้เขาได้เพื่อนร่วมเดินทางมามากมาย รวมไปถึงชีวิตความรักในวัยรุ่นของเข้า กับเรล่า (กัปตันเรดสเคลในปัจจุบัน ใช้นามว่าสเพอโต้) ลูกสาวของกัปตันเรดสเคลคนก่อน ในที่สุดการตามหาก็มาถึงจุดสุดท้าย กุญแจที่จะเปิดประตูสายรุ้ง ซึ่งเป็นประตูที่จะนำไปสู่หินพลังขับ ในตำนานฟอลคอน ที่โจรสลัด รวมไปถึงประเทศต่างๆ ต้องการ หลังจากประตูถูก เปิดออก มลทินแห่งยุคสมัย ฟอลคอน ได้ตื่นขึ้นอีกครั้ง "ไลท์ไวซ์" มันรวบรวมสัตว์ปีศาจ*ทั้งหมดเข้ามาที่เกาะแห่งหนึ่ง (เป็นเกาะที่มีสัญลักษณ์ฟอลคอน มันขึ้นมาจากผิวน้ำตอนที่ แสงที่ออกมาจากสายรุ้ง พุ่งไป) เพื่อทำลายโลก ซางิบุรุษผู้เคยจัดการมันได้ในครั้งที่แล้ว ไม่สามารถต้านทานมันได้อีก เหล่าโจรสลัด จึงต้องร่วมมือกัน โดยมีเพียงประเทศเดียว ที่เข้าช่วยเหลือห่างๆ คือประเทศราอาจิโนวา ซึ่งมีลูกเรือคนหนึ่งของบาธ เป็นกษัตริย์ บาธ และ โคโคได้บุกเข้าไปจนถึงไลท์ไวซ์ และปราบมันได้สำเร็จ เหตุการณ์จบลง บาธ แต่งงานกับ เรล่า คาเฟลลูกของเรล่า ได้ขึ้นเป็นกัปตันเรดสเคลคนต่อไป โคโค ตัวเอกของเรื่องได้ออกเรือ พร้อมกับเพื่อนๆ ที่เจอระหว่างเดินทางเป็นสวีทมาดอนน่า ในนาม "โจรสลัดทวินซอร์ด"- สัตว์ปีศาจ - สัตว์ที่โอลด์บลัดใช้ แปลงร่างได้ 3ขั้น ขั้นสัตว์ธรรมดา ขั้นสัตว์ปีศาจ ขั้นชีวะอาวุธ - โอลด์บลัดชาย - ใช้ในการแปลงสัตว์ปีศาจให้เป็นชีวะอาวุธในขั้นสุดท้าย - โอลด์บลัดหญิง - ใช้ในการแปลงร่างสัตว์ปีศาจในขั้นที่ 1 (โดยการเรียกชื่อ และรู้จักชื่อของสัตว์ปีศาจทุกตัวตั้งแต่แรกเห็น) + มีความสามารถในการค้นหาสัตว์ปีศาจ - สัตว์ยักษ์ - สิ่งที่โอลด์บลัดควบคุม นอกจากตัวใหญ่ บางตัวยิงลำแสงได้ตัวละครเรือสวีทมาดอนน่าตัวละคร. เรือสวีทมาดอนน่า. - จอห์น บาธ : โจรสลัดเจ้าของเรือ สวีทมาดอนน่า โจรสลัดผมสีเงินซึ่งสามารถใช้ดาบคู่ได้อย่างคล่องแคล่ว แต่หากไร้ซึ่งดาบคู่ก็จะเป็นเพียงคนปกติ ต่อมาได้แต่งงานกับ เรล่า - โคโค : เด็กกำพร้าในหมู่บ้านแห่งหนึ่ง ที่จริงเป็นลูกของลัคโลค ต่อมา จอห์น บาธ ชักชวนให้ขึ้นเรือเพื่อร่วมผจญภัย และได้เรียนวิชาดาบจากเด็ทและเป็นกัปตันโจรสลัดทวินซอร์ด - โนซ : เป็นผู้มีความรู้เรื่องตำนานฟอลคอน สามารถอ่านภาษาฟอลคอนได้ เป็นหมอผู้เก่งกาจ - เด็ท : โจรสลัดลูกเรือของบาธ มีฝีมือดาบเดี่ยว โดยอดีตเคยเป็นนักฆ่าอับดับ2ฉายามัจจุราช - บัคจี้ : อดีตทาสที่มีฝีมือพนันไม่เป็นรองใครและยังมีความสามารถในการปลอมตัวอีกด้วย - ฮัลค์ : จอมพลังของสวีทมาดอนน่า มีเป้าหมายจะข้ามเส้นทางหฤโหด มันถอดเสื้อ - มิกัล : หัวขโมยที่มีความไวเป็นเลิศ ตัวเบามาก - พีท : ว่าที่กษัตริย์ของประเทศราอาจิโนวา ขึ้นเรือของบาธเพราะอยากเห็นโลกภายนอกด้วยมุมมองของคนธรรมดา - เซาลุส : สัตว์ยักษ์สองหัว ปีศาจแห่งช่องแคบ เป็นสัตว์เลี้ยงของโคโคคอยตามโคโคไปตลอดสุดท้ายถูกคุโอฆ่าตาย - มิลูก้า : โอลด์บลัดหญิง แอบรักโคโค - อาริส : สัตว์ปีศาจ กระรอกที่ร่างสุดท้ายเป็นดาบยิงลำแสงเรือเรดสเคลเรือเรดสเคล. - กัปตันสพอโด้ (เรล่า) : กัปตันของเรดสเคล เคยเป็นคู่หมั้นของบาธ - คาเฟล : ลูกของกัปตันสพอโด้กับลัค โลค เพื่อนเล่นของโคโค ขึ้นเรือเรดสเคลไปผจญภัย - บลู บาวส์ :เรือโครเบิร์ด เรือโครเบิร์ด. อดีตเคยยิ่งใหญ่ที่สุดในน่านน้ำ แต่กัปตันคนปัจจุบันดันไม่เอาไหน- กัปตันกราฟ : กัปตันสุดเพี้ยนของโครเบิร์ด ถึงแม้จะมีการตัดสินใจที่ไม่เอาไหน ขี้ขลาด และชอบเอาแต่หนี แต่ก็ซื้อใจลูกเรือได้ด้วยความเข้มแข็งและจริงใจ เป็นคนที่รู้จักความหมายของจิตวิญญาณโจรสลัดไม่แพ้ใคร กราฟคือคนที่ยอมสละเรือโครเบิร์ด ซึ่งเป็นมรดกตกทอดของพ่อ เพื่อช่วยผลักดันสวีท มาดอนน่าไปจนถึงที่สุด ในการต่อสู้ครั้งสุดท้าย - รองกัปตัน สตอร์เบอร์รี่ : รองกัปตันที่เพี้ยนพอๆกับกัปตันเลยทีเดียว - รองกัปตัน พีซ : รั่วพอกันเรือดาร์กเนสเกลเรือดาร์กเนสเกล. - คามาล่า : โจรสลัดหญิงที่ว่องไว และยิงปืนแม่น มีฉายาว่าปีศาจต่อมาแต่งงานกับฮัลค์จอมพลังแห่งสวีทมาดอนน่า มีบุตรด้วยกัน 5 คน - เจค : ผู้มีฝีมือเชิงดาบ ฝีมือเป็นรองเพียงคามาล่า หลังการต่อสู้ครั้งสุดท้าย โจรสลัดดาร์กเนสเกลยุบตัวลง จึงเข้าร่วมกับทีมค้นหาหินพลังขับที่นำโดยโนส หมอประจำเรือสวีท มาดอนน่า - คาโต : ดาบยักษ์ ภายหลังสละชีวิตเพื่อปกป้องคามาล่า จากลำแสงของสัตว์ปีศาจที่ควบคุมโดยราอู หนึ่งในสมาชิกของฮากุมุ - ชิด : เก็บมีดบินสำหรับซัดปาไว้ใต้ผ้าคลุม เข้าร่วมทีมค้นหาหินพลังขับร่วมกับเจค - โอนิมัล : เปรียบเสมือนน้องชายของ คาโต เข้าร่วมกลุ่มเพื่อแก้แค้นฮากุมุที่สังหารรุ่นพี่ของตน เป็นกำลังเสริมคอยช่วยเหลือพวก สวีท มาดอนน่า ในการต่อสู้ครั้งสุดท้าย หลังจบเหตุการณ์ ได้ขอติดตามคามาล่าและฮัลค์ไปใช้ชีวิตชาวเลที่เกาะบ้านเกิดของฮัลค์กลุ่มโจรสลัดฮากุมุ กลุ่มโจรสลัดฮากุมุ. กลุ่มโจรสลัดที่คิดจะใช้สัตว์ปีศาจครองโลก- ลัค โลค : พี่ของบาธ พ่อของโคโคและคาเฟล เป็นโอลด์บลัดพิเศษ ที่สามารถคลายผลึกสัตว์ปิศาจทุกขั้นด้วยตัวคนเดียวโดยไม่จำเป็นต้องใช้โอลด์บลัดหญิง ในการปลดผนึก - คลาวฟังก์ : รองกัปตัน อดีตเป็นนักฆ่า มีเด็ทเป็นเป้าหมาย - โบนี่ : สาวสวยที่เป็นคู่หูของคลาวฟังก์คอยคลายผลึกสัตว์ปิศาจให้คลาวฟังก์ - วิแลม : สัตว์ปีศาจงู ของคลาวฟังก์และโบนี่อื่นๆอื่นๆ. - บานิส บลาบลัม : ผู้ใช้ขวานคู่เป็นอาวุธ เคยเป็นสุดยอดนักรบของประเทศหนึ่ง - ซางิ : นักฆ่าอันดับหนึ่งของโลก มีฉายาว่าดาบพิฆาตไกล,นักมายากล เป็นพระเจ้าของชาวฟอลคอน - ออร์ก้า : ลูกศิษย์ของซางิ มีฝีมือดาบสูสีกับบาธ - ไลทไวซ์ : มลทินของชาวฟอลคอน สิ่งที่ชั่วร้ายที่สุดรายชื่อตอน</doc>
