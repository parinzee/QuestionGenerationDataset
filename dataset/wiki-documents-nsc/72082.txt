<doc id="72082" url="https://th.wikipedia.org/wiki?curid=72082" title="อุปกรณ์รับแยกสัญญาณ">อุปกรณ์รับแยกสัญญาณ วงจรดีมัลติเพลกซ์หรือเรียกอีกวงจรหนึ่งว่าวงจรกระจายข้อมูล(Data distributers) เป็นวงจรที่มีเส้นทางข้อมูล Input เพียง 1 เส้นทาง และมี Output มากกว่า 1 เส้นทาง ตัวอย่างเช่น Input 1 Output 4, Input 1 Output 8 และ Input 1 Output 16 เป็นต้น การควบคุมให้ข้อมูล Input เพียง Input เดียวออกไปที่ Output ตำแหน่งใดๆ สามารถทำได้โดยการที่ควบคุมตำแหน่งข้อมูลด้วยขา Select ข้อมูลที่รับออกมาจาก Demultiplexer ก็จะเป็นข้อมูลที่ถูกต้อง ตัวอย่างเช่น Demultiplexer ที่มี Input 1 Output 2 โดยมีตารางค่าความจริงและฟังก์ชันดังนี้  จากฟังก์ชันเราสามารถเขียนวงจร Logic ได้ดังนี้  จากรูป ข้อมูลที่ส่งเข้ามาเป็น Din ถ้าตัวเลือก S เป็น Logic 0 ANDgate ตัวล่างไปรับ Logic 0 ทำให้ Output D2 เป็น Logic0 ในขณะที่ ANDgate ตัวบนได้รับ Logic1 จาก Inverter gate เมื่อ AND กับ Din ก็จะได้ Output ที่ D1 เป็น Din ด้วย(Din คือข้อมูลทาง Digital) ในทำนองเดียวกัน ถ้า S เป็น Logic1 ก็จะปรากฏที่ D2 โดยที่ D1 จะเป็น Logic0 ถ้าเราต้องการที่จะทำ Demultiplexer ที่มี Input 2 Output 4 ก็สามารถสร้างขึ้นได้จากวงจร Input 1 Output 2 ได้ดังนี้  ซึ่งวงจรด้านล่างนี้ก็มีการทำงานคล้ายกับวงจรด้านบน เพียงแต่วงจรด้าล่างนั้นเป็นวงจรที่มี Input 1 ขาแต่มี Output จำนวน n ขา ขึ้นอยู่ที่ Select ว่า Input ที่เข้ามานั้นต้องออกขา Output ขาไหนซึ่งขึ้นอยู่กับค่า S </doc>
