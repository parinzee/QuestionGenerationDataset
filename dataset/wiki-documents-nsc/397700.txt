<doc id="397700" url="https://th.wikipedia.org/wiki?curid=397700" title="แผงควบคุม (วินโดวส์)">แผงควบคุม (วินโดวส์) แผงควบคุม (control panel) เป็นออปชันหนึ่งของ ไมโครซอฟท์ วินโดวส์ ใช้สำหรับปรับแต่งหรือตั้งค่าระบบประวัติ ประวัติ. ใน Windows 1.0 แผงควบคุมเป็นแค่ส่วนหนึ่งของเมนูบาร์ ส่วนในWindows 3.1 มันก็ถูกนำมาเป็นไอคอนแต่การใส่เครื่องมือในแต่ละออปชันยังไม่คอยเหมาะสม จนใน Windows 95 มันก็กลายเป็นโฟลเดอร์พิเศษจนถึงปัจจุบันรายชื่อออปชันในแผงควบคุมรายชื่อออปชันในแผงควบคุม. - Accessibility Option ปรับแต่งค่าให้ผู้พิการใช้คอมพิวเตอร์ง่ายขึ้น - Add/remove program ใช้ถอดถอนหรือติดตั้งโปรแกรม - Add hardware ใช้เพิ่มฮาร์ดแวร์ด้วยมือ ในกรณีที่ วินโดวส์ ไม่สามารถตรวจพบฮาร์ดแวร์ได้เอง - Administrative option เก็บเครื่องมือในการดูแลระบบของแอดมิน - Automatic Updates - Date and time ใช้ตั้งค่าเวลาและวันที่ - Display ใช้ตั้งค่าการแสดงผลบนหน้าจอ - Folder Options ใช้ตั้งค่าการใช้งานไฟล์และโฟลเดอร์ - Fonts ใช้สำหรับติดตั้งและเก็บฟอนต์ - Game Controllers - Internet Options ใช้ตั้งค่าการใช้งานของอินเทอร์เน็ตเอกซ์พลอเรอร์ - Keyboard ใช้ตั้งค่าการใช้คีย์บอร์ด - Mouse ใช้ตั้งค่าการใช้เมาส์ - Network Connections - Phone and Modem Options - Power Options - Printers and Faxes - Regional and Language Settings - Sounds and Audio Devices - System - Taskbar and Start Menu - User Accountsปัญหา</doc>
