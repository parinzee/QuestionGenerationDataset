<doc id="291669" url="https://th.wikipedia.org/wiki?curid=291669" title="ฟุตบอลโลก 1986">ฟุตบอลโลก 1986 ฟุตบอลโลก 1986 เป็นการแข่งขันฟุตบอลโลกครั้งที่ 13 ที่จัดขึ้นที่ประเทศเม็กซิโก ระหว่างวันที่ 13 มิถุนายน ถึง 13 กรกฎาคม ค.ศ. 1986 เดิมทีประเทศโคลอมเบียได้รับการเลือกให้เป็นเจ้าภาพการแข่งขัน แต่เนื่องจากเหตุผลด้านการเงินบางประการทำให้ไม่สามารถเป็นเจ้าภาพได้ และได้ถอนตัวไปในปี ค.ศ. 1982 และเม็กซิโกได้รับเลือกเป็นเจ้าภาพเมื่อเดือนพฤษภาคม ค.ศ. 1983 ประเทศผู้ชนะการแข่งขันครั้งนี้คือทีมอาร์เจนตินา (เป็นครั้งที่ 2 หลังจากชนะในปี 1978) ชนะทีมเยอรมนีตะวันตกไป 3-2 ในรอบตัดสิน ที่จัดขึ้นที่สนาม Estadio Azteca ในเม็กซิโกซิตีรายชื่อประเทศที่เข้ารอบรายชื่อประเทศที่เข้ารอบ. - เอเอฟซี (2) - &nbsp; - &nbsp; - ซีเอเอฟ (2) - &nbsp; - &nbsp; - โอเอฟซี (0) - ไม่ผ่านรอบคัดเลือก- คอนคาแคฟ (2) - &nbsp; - &nbsp;(เจ้าภาพ) - คอนเมบอล (4) - &nbsp; - &nbsp; - &nbsp; - &nbsp;- ยูฟ่า (14) - &nbsp; - &nbsp; - &nbsp; - &nbsp; - &nbsp; - &nbsp; - &nbsp; - &nbsp; - &nbsp; - &nbsp; - &nbsp; - &nbsp; - &nbsp; - &nbsp;</doc>
