<doc id="178742" url="https://th.wikipedia.org/wiki?curid=178742" title="แชร์พอยต์">แชร์พอยต์ แชร์พอยท์ (Microsoft SharePoint) คือชุดผลิตภัณฑ์ของบริษัทไมโครซอฟท์ สำหรับการประสานงานร่วมกันและบริหารเอกสารผ่านเว็บเบราว์เซอร์ รวมทั้งมีความสามารถในการสร้างอย่าง Wikis และ Blogs โดยทั่วไป คำว่าแชร์พอยต์จะหมายถึง 2 ผลิตภัณฑ์ ได้แก่- Windows SharePoint Services (WSS) - Microsoft Office SharePoint Server (MOSS) แชร์พอยต์ถูกสร้างด้วยเทคโนโลยีเอเอสพีดอตเน็ต 2.0 ทำงานบน IIS และฐานข้อมูล SQLกลุ่มผลิตภัณฑ์แชร์พอยต์Windows Share Point Services (WSS) กลุ่มผลิตภัณฑ์แชร์พอยต์. Windows Share Point Services (WSS). Windows Share Point Services (WSS) เป็นโปรแกรมที่แถมฟรีมากับ Windows Server มึความสามารถในการประสานงาน รองรับการทำงานผ่านเบราว์เซอร์Microsoft Office SharePoint Server 2007 (MOSS 2007) Microsoft Office SharePoint Server 2007 (MOSS 2007). Microsoft Office SharePoint Server 2007 (MOSS 2007) เป็นโปรแกรมที่พัฒนาเพิ่มจาก WSS โดยสามารถจัดการข้อมูลได้มีประสิทธิภาพมากกว่า ในด้านค้นหาข้อมูล จัดเก็บเอกสาร และความคุมข้อมูล รวมทั้งมีข้อมูลในการวิเคราะห์ธุรกิจMicrosoft SharePoint Designer (SPD) Microsoft SharePoint Designer (SPD). Microsoft SharePoint Designer คือ เครื่องมือที่ใช้ในการออกแบบ ปรับแต่งหน้าจอ และ สร้างลำดับการทำงานภายในแชร์พอยต์ซึ่งสามารถทำงานร่วมกับวิชวลสตูดิโอ 2008</doc>
