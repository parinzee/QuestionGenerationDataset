<doc id="98662" url="https://th.wikipedia.org/wiki?curid=98662" title="คำนิยมเพชรพระอุมา">คำนิยมเพชรพระอุมา คำนิยมเพชรพระอุมา เป็นการแสดงความคิดเห็นของผู้ทรงเกียรติและทรงคุณวุฒิหลายท่าน ที่มีต่อนวนิยายเรื่องเพชรพระอุมาจากสำนักพิมพ์ ณ บ้านวรรณกรรม ซึ่งการแสดงความนิยมต่อเพชรพระอุมานี้ เป็นการการันตีถึงผลงานการประพันธ์ของพนมเทียนและความเป็นสุดยอดของวรรณกรรมชิ้นเอกของเมืองไทย ซึ่งผู้ทรงเกียรติและทรงคุณวุฒิที่ได้ให้เกียรติมาให้คำนิยมต่อเพชรพระอุมา มาจากหลายอาชีพและหน้าที่การงาน เช่น ฯพณฯ นายชวน หลีกภัย ฯพณฯ นายมีชัย ฤชุพันธุ์ นายเสนาะ เทียนทอง ดร.พิจิตต รัตตกุล ในแต่ละตอนของเพชรพระอุมา ต่างมีคำนิยมที่แสดงให้เห็นถึงความสามารถในการประพันธ์ของพนมเทียน ความสนุนสนานของการดำเนินเรื่อง ความรู้ในด้านต่าง ๆ ของการเดินป่า การใช้ภาษาในการตัดพ้อหรือพร่ำพรรณา ที่พนมเทียนสามารถท่ายถอดให้แก่ตัวละครทุกตัวในเพชรพระอุมา ให้มีชีวิตชีวาโลดแล่นจนเป็นที่รู้จักและผูกพันกับนักอ่านทุกรุ่นทุกสมัย ซึ่งคำนิยมเพชรพระอุมาจากผู้ทรงเกียรติและทรงคุณวุฒิหลายท่าน ได้แก่คำนิยมเพชรพระอุมาคำนิยมภาคแรกตอนไพรมหากาฬคำนิยมเพชรพระอุมา. คำนิยมภาคแรก. ตอนไพรมหากาฬ. - เพชรพระอุมา ตอนไพรมหากาฬ เล่ม 1- เพชรพระอุมา ตอนไพรมหากาฬ เล่ม 2- เพชรพระอุมา ตอนไพรมหากาฬ เล่ม 3- เพชรพระอุมา ตอนไพรมหากาฬ เล่ม 4ตอนดงมรณะตอนดงมรณะ. - เพชรพระอุมา ตอนดงมรณะ เล่ม 1- เพชรพระอุมา ตอนดงมรณะ เล่ม 2- เพชรพระอุมา ตอนดงมรณะ เล่ม 3- เพชรพระอุมา ตอนดงมรณะ เล่ม 4ตอนจอมผีดิบมันตรัยตอนจอมผีดิบมันตรัย. - เพชรพระอุมา ตอนจอมผีดิบมันตรัย เล่ม 1- เพชรพระอุมา ตอนจอมผีดิบมันตรัย เล่ม 2- เพชรพระอุมา ตอนจอมผีดิบมันตรัย เล่ม 3- เพชรพระอุมา ตอนจอมผีดิบมันตรัย เล่ม 4ตอนอาถรรพณ์นิทรานครตอนอาถรรพณ์นิทรานคร. - เพชรพระอุมา ตอนอาถรรพณ์นิทรานคร เล่ม 1- เพชรพระอุมา ตอนอาถรรพณ์นิทรานคร เล่ม 2- เพชรพระอุมา ตอนอาถรรพณ์นิทรานคร เล่ม 3- เพชรพระอุมา ตอนอาถรรพณ์นิทรานคร เล่ม 4ตอนป่าโลกล้านปีตอนป่าโลกล้านปี. - เพชรพระอุมา ตอนป่าโลกล้านปี เล่ม 1- เพชรพระอุมา ตอนป่าโลกล้านปี เล่ม 2- เพชรพระอุมา ตอนป่าโลกล้านปี เล่ม 3- เพชรพระอุมา ตอนป่าโลกล้านปี เล่ม 4ตอนแงซายจอมจักราตอนแงซายจอมจักรา. - เพชรพระอุมา ตอนแงซายจอมจักรา เล่ม 1- เพชรพระอุมา ตอนแงซายจอมจักรา เล่ม 2- เพชรพระอุมา ตอนแงซายจอมจักรา เล่ม 3- เพชรพระอุมา ตอนแงซายจอมจักรา เล่ม 4คำนิยมภาคสมบูรณ์ตอนจอมพรานคำนิยมภาคสมบูรณ์. ตอนจอมพราน. - เพชรพระอุมา ตอนจอมพราน เล่ม 1- เพชรพระอุมา ตอนจอมพราน เล่ม 2- เพชรพระอุมา ตอนจอมพราน เล่ม 3- เพชรพระอุมา ตอนจอมพราน เล่ม 4ตอนไอ้งาดำตอนไอ้งาดำ. - เพชรพระอุมา ตอนไอ้งาดำ เล่ม 1- เพชรพระอุมา ตอนไอ้งาดำ เล่ม 2- เพชรพระอุมา ตอนไอ้งาดำ เล่ม 3- เพชรพระอุมา ตอนไอ้งาดำ เล่ม 4ตอนจิตรางคนางค์ตอนจิตรางคนางค์. - เพชรพระอุมา ตอนจิตรางคนางค์ เล่ม 1- เพชรพระอุมา ตอนจิตรางคนางค์ เล่ม 2- เพชรพระอุมา ตอนจิตรางคนางค์ เล่ม 3- เพชรพระอุมา ตอนจิตรางคนางค์ เล่ม 4ตอนนาคเทวีตอนนาคเทวี. - เพชรพระอุมา ตอนนาคเทวี เล่ม 1- เพชรพระอุมา ตอนนาคเทวี เล่ม 2- เพชรพระอุมา ตอนนาคเทวี เล่ม 3- เพชรพระอุมา ตอนนาคเทวี เล่ม 4ตอนแต่ปางบรรพ์ตอนแต่ปางบรรพ์. - เพชรพระอุมา ตอนแต่ปางบรรพ์ เล่ม 1- เพชรพระอุมา ตอนแต่ปางบรรพ์ เล่ม 2- เพชรพระอุมา ตอนแต่ปางบรรพ์ เล่ม 3- เพชรพระอุมา ตอนแต่ปางบรรพ์ เล่ม 4ตอนมงกุฎไพรตอนมงกุฎไพร. - เพชรพระอุมา ตอนมงกุฎไพร เล่ม 1- เพชรพระอุมา ตอนมงกุฎไพร เล่ม 2- เพชรพระอุมา ตอนมงกุฎไพร เล่ม 3- เพชรพระอุมา ตอนมงกุฎไพร เล่ม 4</doc>
