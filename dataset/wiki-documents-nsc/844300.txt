<doc id="844300" url="https://th.wikipedia.org/wiki?curid=844300" title="Mind Memory 1.44 พื้นที่รัก">Mind Memory 1.44 พื้นที่รัก Mind Memory 1.44 พื้นที่รัก เป็นภาพยนตร์ไทย แนวโรแมนติก สร้างโดย บริษัท เฟฮู อินฟินิตี้ จำกัด กำกับโดย สุพจน์ คูณดี,มนัสนันท์ กุดหอม นำแสดงโดย เจมส์ มาร์,ฮัม อึนจอง, พัสกร พลบูรณ์,ณัฐวุฒิ เจนมานะ,จุฑามาศ วิชัย กำหนดออกฉายในวันที่ 23 กุมภาพันธ์ พ.ศ. 2560เรื่องย่อ เรื่องย่อ. ภัทร์ (เจมส์ มาร์) โปรแกรมเมอร์อัจฉริยะ ผู้ที่ชอบอยู่ในโลกของตัวเอง ซึ่งมีเพื่อนสนิท พีท (ณัฐวุฒิ เจนมานะ) และแพรว (พัสกร พลบูรณ์) โลกของภัทร์ได้เปลี่ยนไปเมื่อได้เจอกับมินจี (ฮัม อึนจอง) สาวชาวเกาหลี มินจีเป็นผู้หญิงที่เก่งมีความอ่อนโยน ทั้งคู่ได้ร่วมกันพัฒนาโปรแกรมเพื่อช่วยเหลือคนชราและใช้ฐานข้อมูลมาจาก Mind ซึ่งเป็นแอพพิเคชั่นที่ภัทร์รักและหวงมากที่สุด ความสัมพันธ์ของทั้งคู่ค่อยๆ เริ่มก่อตัวขึ้น ความทรงจำต่าง ๆ ที่ถูกบันทึกด้วยหัวใจของคนสองคน จนกระทั่งได้มีคนทำให้ความสัมพันธ์ของทั้งคู่เปลี่ยนไปความรักและความเข้าใจจะช่วยให้พวกเขาผ่านพ้นเรื่องราวต่าง ๆ ไปได้หรือไม่นักแสดงนักแสดง. - เจมส์ มาร์ รับบท ภัทร์ - ฮัม อึนจอง รับบท คิม มินจี - ณัฐวุฒิ เจนมานะ รับบท พีท - พัสกร พลบูรณ์ รับบท แพรว - จุฑามาศ วิชัย รับบท จีน่าเพลงประกอบภาพยนตร์เพลงประกอบภาพยนตร์. - เพลง ความทรงจำ ขับร้องโดย คณิตกุล เนตรบุตรการสร้างและการตอบรับ การสร้างและการตอบรับ. เดิมภาพยนตร์จะใช้ชื่อว่า "Micro Love" แต่ได้เปลี่ยนชื่อมาเป็น Mind Memory 1.44 พื้นที่รัก ในที่สุด เป็นภาพยนตร์เรื่องแรกจากการสร้างของเฟฮู อินฟินิตี้ ของไทยร่วมทุนกับบริษัทสร้างภาพยนตร์ของเกาหลีใต้ โดยมีนักแสดงระดับซูเปอร์สตาร์ของเกาหลีใต้มาร่วมนำแสดง คือ ฮัม อึนจอง นักร้องแห่งวงที-อารา เกิร์ลกรุปชื่อดังของเกาหลีใต้ แต่ได้มีปัญหาในเรื่องของการเข้าฉาย ได้มีการเลื่อนการฉายออกไปจากเดิมที่กำหนดไว้ที่ปี พ.ศ. 2559 เป็นต้นปี พ.ศ. 2560 และได้มีกระแสข่าวว่ามีการถอดนักแสดงคนหนึ่งออก คือ ซอ จียอน และได้เปลี่ยนเป็นจุฑามาศ วิชัย แทน เมื่อเข้าฉายได้เรต น 13+ แต่ทว่าก็มีเสียงวิจารณ์ว่ากระแสตอบรับไม่ดี จนถึงขนาดอาจเรียกได้ว่าเงียบเชียบ แต่นักแสดงนำคือ เจมส์ มาร์ ที่มีผลงานภาพยนตร์เป็นเรื่องแรกในชีวิตก็ไม่สนกระแสวิจารณ์อันนี้</doc>
