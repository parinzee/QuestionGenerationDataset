<doc id="701972" url="https://th.wikipedia.org/wiki?curid=701972" title="เขตปกครองคณะสงฆ์จังหวัดราชบุรี">เขตปกครองคณะสงฆ์จังหวัดราชบุรี การปกครองคณะสงฆ์ระดับจังหวัดเป็นไปตาม พรบ.คณะสงฆ์ พ.ศ. ๒๕๐๕ ฉบับแก้ไข พ.ศ. ๒๕๓๔ ที่จัดให้มีการปกครองคณะสงฆ์ระดับต่าง ๆ ลดหลั่นกันไปจนกระทั่งถึงระดับจังหวัด โดยจังหวัดราชบุรี มีการปกครองคณะสงฆ์อยู่ภายใต้ เขตปกครองคณะสงฆ์หนกลาง เขตปกครองคณะสงฆ์มหานิกาย ภาค ๑๕ โดยจัดแบ่งการปครองภายในเป็น ๑๐ อำเภอ (๑๐ เจ้าคณะเภอ) ๖๙ ตำบล (๖๙ เจ้าคณะตำบล) พระธรรมปัญญาภรณ์ (ไพบูลย์ ป.ธ.๗) เจ้าอาวาสวัดมหาธาตุ ฯ พระอารามหลวง อำเภอเมือง จังหวัดราชบุรี เป็นเจ้าคณะจังหวัดราชบุรี และมีพระมหาปราโมทย์ ปโมทิโต ป.ธ.๙ วัดหลวงพ่อสดธรรมกายาราม เป็นรองเจ้าคณะจังหวัดเขตปกครองคณะสงฆ์ในเขตจังหวัดราชบุรี เขตปกครองคณะสงฆ์ในเขตจังหวัดราชบุรี. -พระสังฆาธิการจังหวัดราชบุรี-- เจ้าคณะจังหวัดราชบุรี (จจ.) : พระธรรมปัญญาภรณ์ (มหาไพบูลย์ ชินวํโส) ป.ธ.๗ จล.วัดมหาธาตุวรวิหาร. เลขานุการ : พระมหาสุนทร สุนฺทโร ป.ธ.๗. วัดมหาธาตุวรวิหาร.- รองเจ้าคณะจังวัดราชบุรี : พระปิฎกโกศล (มหาปราโมทย์ ปโมทิโต ป.ธ.๙) วัดหลวงพ่อสดธรรมกายาราม. เลขานุการ : พระมหาเกรียงศักดิ์ ภูริวฑฺฒโก ป.ธ.๙ วัดหลวงพ่อสดธรรมกายาราม. -อำเภอเมืองราชบุรี-- เจ้าคณะอำเภอเมืองราชบุรี (จอ.) : พระราชวัลภาจารย์ (ดาวเรือง) วัดหนองหอย. เลขานุการ : พระครูสิริราชบุรานุวัตร (มหาเดชา) ป.ธ.๕ วัดหนองหอย- รองเจ้าคณะอำเภอเมืองราชบุรี (รจอ.) รูปที่ ๑ : พระครูศรีธรรมวาที (มหาศักดา อคฺคปญฺโญ) วัดเทพอาวาส. - รองเจ้าคณะอำเภอเมืองราชบุรี (รจอ.) รูปที่ ๒ : พระมหาสุนทร สุนฺทโร ป.ธ.๗. วัดมหาธาตุวรวิหาร.- จต.หน้าเมืองเขต ๑ : พระครูบริรักษ์คามเขต (สุกิจ วรกิจโจ) วัดเขาเหลือ. - จต.หน้าเมืองเขต ๒ : พระครูไพบูลธรรมวงศ์ (ไพฑูรย์ วํสวณฺโณ) วัดดอนตะโก. - จต.ท่าราบ : พระครูโสภณวิภูษิต (บรรจบ สีลภูสิโต) วัดบางลี่เจริญธรรม - จต.เจดีย์หัก-โคกหม้อ : พระครูพิศาลจริยานุกิจ (มหาพรชัย ฐิตสาโร) ป.ธ.๔ วัดช่องลมพระอารามหลวง. - จต.พงสวาย-บางป่า : พระครูวาทีธรรมวัฒน์ (มหาบุญเลิศ ฐิตวฑฺฒโน ป.ธ.๔) วัดโพธิ์ราษฎร์บูรณะ. - จต.สามเรือน : พระครูสุวรรณวิหารการ (เดชา จนฺทโชโต) วัดพิกุลทอง - จต.คุ้งกระถิน : พระครูสุคันธสีลาภรณ์ (สันติ คนฺธสีโล) วัดศาลเจ้า. - จต.คุ้งน้ำวน : พระครูธรรมสารสุนทร (พิศิษฏ์ ญาณจิตฺโต) วัดท่าสุวรรณ. - จต.คูบัว : พระครูพิพัฒน์ศาสนการ (เฉลิม จนฺทธมฺโม) วัดแคทราย. - จต.อ่างทอง : พระมหาสำรวย ปญฺญาสาโร ป.ธ.๖ วัดอ่างทอง. - จต.ดอนตะโก : พระครูกิตติสังวร (อนุวัตร กิตฺติสํวโร) วัดอรุณรัตนคีรี. - จต.เกาะพลับพลา-เจดีย์หัก :พระมหาธนพัฒน์ กนฺตธมฺโม ป.ธ.๓ วัดเจติยาราม. - จต.เขาแร้ง-เกาะพลับพลา : พระครูสถิตญาณคุณ (จรัญ พลญาโณ) วัดเขาถ้ำกรวย. - จต.หินกอง : พระครูประโชติธรรมวิมล (ประเสริฐ นาควโร) วัดหนองกระทุ่ม. - จต.น้ำพุ-ห้วยไผ่ : พระสมุห์ประดิษฐ์ ปญฺญาวฑฺฒโก วัดหนองแช่เสา- รจต.น้ำพุ-ห้วยไผ่ : พระอธิการสุชาติ อตฺตสาโร วัดน้ำพุไชยสิทธิ์ -อำเภอดำเนินสะดวก-- เจ้าคณะอำเภอดำเนินสะดวก (จอ.) : พระมหาประกอบ โชติปุญฺโญ ป.ธ.๗ จร.วัดโชติทายการาม. เลขานุการ : พระสมุห์ สมกิจ วรธมฺโม วัดโชติทายการาม.- จต.ดำเนินสะดวก : พระปลัดประชุม ฌานสมฺปนฺโน จร.วัดพิทักษ์เทพาวาส - จต.บัวงาม : พระมหาประกอบ นิติธโร ป.ธ.๗ วัดบัวงามพระอารามหลวง - จต.ท่านัด : พระครูอมรวิสุทธาภรณ์ (มงคล สิริธมฺโม) จร.วัดอมรญาติสมาคม - จต.สี่หมื่น-แพงพวย : พระครูนิวิฐธรรมาภรณ์ (นิวัฒน์ อตฺตคุตฺโต) จร.วัดท่าเรือ - จต.บ้านไร่ : พระครูศิริวรรณวิวัฒน์ (วรรณะ วณฺโณ) จร.วัดหลักหกรัตนาราม -อำเภอโพธาราม-- เจ้าคณะอำเภอโพธาราม พระครูวิสุทธานันทคุณ (สุรศักดิ์ วิสุทฺธาจาโร) วัดเขาช่องพราน. เลขานุการ : พระครูวิรุฬฆ์ธรรมวงศ์ (ภาณุพงศ์ พลญาโณ) วัดเขาช่องพราน.- ที่ปรึกษาเจ้าคณะอำเภอโพธาราม : พระมงคลชัยสิทธิ์ (เขียน สิริมังคโล ป.ธ.๓) วัดโบสถ์. - รองเจ้าคณะอำเภอโพธาราม : พระครูคุณสารานุกูล วัดโบสถ์. เลขานุการ : พระดิเรก อติเรกสุโภ วัดนครทิพย์. - รองเจ้าคณะอำเภอโพธาราม : พระมหาสมคิด อตฺถสิทฺโธ วัดหนองโพ. เลขานุการ : พระมหาสมบูรณ์ จนฺทโชโต วัดพระศรีอารย์.- เจ้าคณะตำบลโพธาราม-บ้านฆ้อง : พระมหาพิสิษฐ์ โสภิโต ป.ธ.๖ วัดโพธิ์ไพโรจน์ - เจ้าคณะตำบลบ้านสิงห์-หนองโพ : พระครูโสภณกิตติคุณ (ด้วง สวาคโต) วัดบ้านฆ้อง - เจ้าคณะตำบลดอนทราย : พระครูสุนทรพิพิธการ (ฉัฏฐ์ อิฏฺฐผโล) วัดหนองอ้อ. - เจ้าคณะตำบลบ้านเลือก : พระครูสาธกธรรมกิจ (สมจิตร วิจิตฺตธมฺโม) วัดบ้านเลือก. - เจ้าคณะตำบลตลองตาคต-ดอนกระเบื้อง : พระครูเวฬุวนาภิรมย์ (ปทีป กิตฺติสาโร) วัดป่าไผ่. - เจ้าคณะตำบลเจ็ดเสมียน-คลองข่อย : พระครูพิศาลสุทธากร (ไพศาล สุทฺธจิตฺโต) วัดไทรอารีรักษ์ - เจ้าคณะตำบลสร้อยฟ้า-ชำแระ : พระครูพิทักษ์ศิลปาคม (นุชิต วชิรวุฑฺโฒ) วัดขนอน. - เจ้าคณะตำบลเตาปูน : พระครูวรวิสุทธิวัตร (มหาสวัสดิ์ อธิวโร ป.ธ.๓) วัดเขาพระ. - เจ้าคณะตำบลท่าชุมพล : พระครูพัฒนวิหารธรรม (สวัสดิ์ ปวฑฺฒโน) วัดม่วง. - เจ้าคณะตำบลบางโตนด : พระครูพิทักษ์บุญวิมล (สำลี สิริปญฺโญ) วัดสมถะ. - เจ้าคณะตำบลนางแก้ว : พระครูสุวัฒน์รัตนคุณ (มานิตย์ คุณธมฺโม) วัดนางแก้ว. - เจ้าคณะตำบลธรรมเสน : พระครูพลธรรมาภิรัต (นิพนธ์ พลวโร) วัดเขาแหลม. - เจ้าคณะตำบลเขาชะงุ้ม : พระครูภาวนาคุณาธาร (สุชิน คุณธาโร) วัดเขาส้ม. - เจ้าคณะตำบลหนองกวาง : พระครูพิหารปทุมรักษ์ (สถิตย์ อนาวิโล) วัดหนองใยบัว. -อำเภอบ้านโป่ง -- เจ้าคณะอำเภอบ้านโป่ง : พระครูศรีธรรมนาท วัดท่าผา เลขานุการเจ้าคณะอำเภอบ้านโป่ง : พระมหาทวีศักดิ์ คุณธมฺโม (ดร.) ป.ธ.7 วัดบ้านโป่ง- รองเจ้าคณะอำเภอบ้านโป่ง : พระครูสุธีปริยัติวิธาน วัดบ้านโป่ง เลขานุการรองเจ้าคณะอำเภอบ้านโป่ง : พระสมคิด จินฺตสโม ประโยค1-2 วัดบ้านโป่ง- จต.กรับใหญ่ พระครูสันติธรรมาภิรัต วัดอ้ออีเขียว เลขานุการ : พระครูสังฆรักษ์ทรงพรรณ ชยทตฺโต วัดอ้ออีเขียว - จต.บ้านโป่ง พระครูประสิทธิ์สุตกิจ วัดอุทุมพรทาราม เลขานุการ : พระปลัดประทุม สีลปทุโม วัดโคกหม้อ - จต.หนองกบ พระครูไพศาลพัฒนาภิรัติ วัดประชารังสรรค์ เลขานุการ : พระครูสมุห์ดิเรก สุจิตฺโต วัดหนองกลางด่าน - จต.หนองอ้อ - สวนกล้วย พระมหาอดุลณัฎฐ์ ยุตฺตธมฺโม เลขานุการ : - - จต.นครชุมน์ พระครูนครเขมกิจ วัดนครชุมน์ เลขานุการ : พระวรวิทย์ วรสทฺโธ วัดบ้านโป่ง - จต.บ้านม่วง - คุ้งพยอม พระครูศรีเมตตานุศาสก์ วัดโพธิ์โสภาราม เลขานุการ : รองพระอธิการสุวัฒน์ ถิรสํวโร วัดตาลปากลัด - จต.หนองปลาหมอ พระครูสุพัฒน์วรกิจ วัดชมพูพล เลขานุการ : พระสมควร คุณสาโร วัดชมภูพล - จต.เขาขลุง เขต 1 พระครูอุทัยคีรีรักษ์ วัดโป่งยอ เลขานุการ : พระศุภวิชญ์ ธมฺมวิสุทฺโธ ประโยค 1-2 วัดบ้านโป่ง - จต.เขาขลุง เขต 2 พระครูวิหารการโกวิท วัดสระสี่มุม เลขานุการ : พระดำรงศักดิ์ สุภาจาโร วัดมาบแค - จต.ท่าผา : พระครูชินวรานุวัตร วัดบ้านฆ้องน้อย เลขานุการ : พระมหาวรรณรัตน์ รตนวณฺโณ ป.ธ.3 วัดท่าผา - จต.เบิกไพร - ลาดบัวขาว : พระครูเกษมปัญญาพิสุทธิ์ (หลอด เขมจาโร) วัดปลักแรด เลขานุการ : -อำเภอจอมบึง-- เจ้าคณะอำเภอจอมบึง พระครูวิบูลกิจสุนทร (เมธี ) วัดวาปีสุทธาวาส. เลขานุการ : พระปลัด ณรงค์ศักดิ์ ปภากโร วัดวาปีสุทธาวาส- รองเจ้าคณะอำเภอจอมบึง พระครูสุพจน์บุญญากร (สุมนต์ ) วัดรางเฆ่. เลขานุการ : พระสุมิตร ปภสฺสโร วัดรางเฆ่- จต.จอมบึง-เบิกไพร : พระครูประโชติวิสุทธิธรรม (ชูชาติ โชติกาโร) วัดจอมบึง. - จต.แก้มอ้น-เบิกไพร : พระครูรัตนกิตติวัฒน์ (ประยูร ) วัดหนองปรือใหญ่พลายงาม. - จต.รางบัว : พระครูสุชาตธรรมาภรณ์ (ชาญ ยสชาโต) วัดหนองนกกระเรียน. - จต.ปากช่อง เขต ๑ : พระครูโอภาสอรัญวัตร (อรุณ ) วัดนาสมอ. - จต.ปากช่อง เขต ๒ : พระครูภาวนาโชติคุณ (กุ้ยไฮ้ ) วัดถ้ำสิงโตทอง. - จต.ด่านทับตะโก : พระครูอุดมพัฒนโส๓ณ (บัว ทตฺตมโน) วัดเขาแดน. -อำเภอสวนผึ้ง-- เจ้าคณะอำเภอสวนผึ้ง พระครูจริยธรรมานุรักษ์ (วัลลภ จริยธฺมโม) วัดหนองหมี. เลขานุการ : พระปัญญา ปญฺญาวชิโร วัดหนองหมี.- จต.ป่าหว่าย-ท่าเคย : พระอธิการธานิทร์ ถาวโร วัดทุ่งแหลม - จต.ท่าเคย : พระครูพิทักษ์อรัญเขต (ตี๋ อิสิญาโณ) วัดวังน้ำเขียว - จต.สวนผึ้ง-ตะนาวศรี : พระครูวิบูลภัทรกิจ (อุดม โอภาโส) วัดหนองหมี -อำเภอปากท่อ-- เจ้าคณะอำเภอปากท่อ พระครูสุภัทรญาณวินิฐ (บุญยัง ) วัดปากท่อ. เลขานุการ : พระปลัด จติชัย มหาวายาโม วัดปากท่อ.- รองเจ้าคณะอำเภอปากท่อ พระครูพิพัฒน์จริยาภรณ์ (สาโชติ ) วัดหนองกระทุ่ม.- เจ้าคณะตำบลปากท่อ-ดอนทราย พระครูโสภณธรรมาภิบาล (เล็ก ธมฺมปาโล) วัดหนองระกำ. - เจ้าคณะตำบลวัดยางงาม พระครูศรีศาสนกิจจากร (อวยชัย ป.ธ.๖) วัดยางงาม. - เจ้าคณะตำบลอ่างหิน พระครูบุญญาภิรักษ์ (อนุชัย ) วัดเขาถ่านธรรมเสนานี. - เจ้าคณะตำบลบ่อกระดาน พระครูพัฒนวิสิทธิ์ (สมชาย ) วัดสุขวราราม. - เจ้าคณะตำบลวังมะนาว พระครูสิริภัททาภรณ์ (เพียว ) วัดห้วยยางโทน. - เจ้าคณะตำบลหนองกระทุ่ม พระครูวิโรจน์ญาณธารี (ไสว ญาณจารี) วัดหนองน้ำใส. - เจ้าคณะตำบลทุ่งหลวง พระครูผาสุกวิริยคุณ (สุชิน ) วัดสนามสุทธาวาส. -อำเภอบางแพ-- เจ้าคณะอำเภอบางแพ พระครูโสภณอาจารคุณ (เซียน ป.ธ.๔) วัดกลางวังเย็น. เลขานุการ : พระครูโสภณอาจารวัตร (สมยศ ) วัดกลางวังเย็น.- เจ้าคณะตำบลบางแพ-ดอนใหญ่ พระมหาสละ ฉนฺทโก ป.ธ.๗ วัดลำน้ำ. - เจ้าคณะตำบลบางแพ-หัวโพ พระครูสังฆการโกวิท (น้ำวัง ) วัดหัวโพ. - เจ้าคณะตำบลวังเย็น พระครูประสาทวรคุณ (สมศักดิ์ ) วัดบางแพใต้. - เจ้าคณะตำบลโพหัก-ดอนคา พระครูสุนทรปิยาจาร (ละมูล ) วัดดอนสาลี. - เจ้าคณะตำบลวัดแก้ว พระครูปัญญาธิการ (บุญช่วย ป.ธ.๓) วัดทำนบ. -อำเภอวัดเพลง-- เจ้าคณะอำเภอวัดเพลง พระครูโสภิตอาจารวัตร (วิวัฒน์ ) วัดคลองขนอน. เลขานุการ : พระปลัดโชติ จิรโสภโณ วัดแจ้งเจริญ.- ที่ปรึกษาเจ้าคณะอำเภอวัดเพลง (๒๕๕๙) พระครูศรีธรรมาภรณ์ (สุรนารถ สีลสํวโร ป.ธ.๔) วัดแจ้งเจริญ.- เจ้าคณะตำบลวัดเพลง พระปลัดชิน สุนทโร วัดแจ้งเจริญ. -อำเภอบ้านคา-- เจ้าคณะอำเภอบ้านคา พระครูอรัญเขตคณารักษ์ (นิพนธ์ ) วัดโป่งกระทิงล่าง. เลขานุการ : พระสมุห์สัมพันธ์ ฐานจาโร วัดโป่งกระทิงล่าง.- เจ้าคณะตำบลบ้านคา-บ้านบึง เจ้าอธิการองอาจ อธิปุญฺโญ วัดเก่าต้นมะค่า. - เจ้าคณะตำบลหนองพันจันทร์ พระมหาราเชน ธมฺมรโส ป.ธ.๓ วัดหนองพันจันทร์.ทำเนียบรายนามเจ้าคณะจังหวัดราชบุรีวัดในเขตปกครองจังหวัดราชบุรีทำเนียบรายนามเจ้าคณะจังหวัดราชบุรี. วัดในเขตปกครองจังหวัดราชบุรี. - รายชื่อวัดในจังหวัดราชบุรี</doc>
