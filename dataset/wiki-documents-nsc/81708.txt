<doc id="81708" url="https://th.wikipedia.org/wiki?curid=81708" title="โพลีคีไทด์">โพลีคีไทด์ สารประกอบโพลีคีไทด์ () เป็นกลุ่มของสารประกอบเคมีที่เกิดขึ้นเองในธรรมชาติ (natural product) สามารถพบได้ทั้งใน พืช รา และ แบคทีเรีย สารประกอบโพลีคีไทด์นั้นเป็นสารประกอบที่มีหมู่คาร์บอนนิลหลายหมู่ในโมเลกุล ได้รับความสนใจค้นคว้าเนื่องจากสารประกอบจำพวกนี้มีคุณสมบัติทางชีวภาพหลายประการ อย่างเช่น มีคุณสมบัติในการฆ่าเชื้อโรค และต่อต้านเนื้องอก สารประกอบโพลีคีไทด์ที่พบในธรรมชาติสามารถแบ่งได้เป็น 1. สารประกอบจำพวก อะโรมาติก (aromatics) 2. สารประกอบจำพวก แมคโครีด (macrolides) 3. ลารประกอบจำพวก โพลีอีเธอร์ (polyethers) และ 4. สารประกอบจำพวก โพลีอีนน์ (polyenes)กระบวนการสังเคราะห์ในธรรมชาติ กระบวนการสังเคราะห์ในธรรมชาติ. สารประกอบโพลีคีไทด์ที่พบในธรรมชาตสังเคราะห์ขึ้นจากอนุพันธ์ของโคเอมไซม์เอ (coenzyme A, CoA) อย่างเช่น มาโลนิลโคเอ (malonyl-CoA) เมทิลมาโลนิลโคเอ (methylmalonyl-CoA) โดยปฏิกิริยาการควบแน่นแบบเคเซนน์ (Claisen condensation) โดยเอมไซม์ที่ถูกเรียกว่า โพลีคีไทด์ซินเทส (Polyketide synthase, PKS)  อนุพันธ์ของโคเอมไซม์เอที่ถูกใช้ในการสังเคราะห์สารประกอบโพลีคีไทด์นั้นถูกเปลี่ยนมาจากสารประกอบต่างๆภายในเซลล์ เช่น คาร์โบไฮเดรต,กรดไขมัน,กรดคาร์บอกซิล,และกรดอะมิโนโพลีคีไทด์ซินเทส โพลีคีไทด์ซินเทส. โพลีคีไทด์ซินเทส ที่พบในธรรมชาตสามารถแบ่งออกได้เป็นสามประเภทได้แก่1. พวกที่เป็นโปรตีนขนาดใหญ่และมีหลาย active site ในหนึ่งโปรตีน ที่ถูกเรียกว่า Type I หรือ Modular PKS เอนไซม์ในกลุ่มนี้ประกอบไปด้วยหลาย module โดยแต่ล่ะ module จะเร่งปฏิกิริยาต่างๆตั้งโดยเริ่มจากการเพิ่มหน่วยถัดไปเข้าสู่สายโพลีคีไทด์และทำการเปลี่ยนแปลงโครงสร้างต่างก่อนที่ส่งต่อสายโพลีคีไทด์ไปยัง module ต่อไป ในแต่ล่ะ module จะประกอบด้วย domain ต่างๆ โดนแต่ล่ะ domain จะทำหน้าที่ต่างๆในการสังเคราะห์สารประกอบโพลีคีไทด์ เช่น เร่งปฏิกิริยาการควบแน่นแบบเคเซนน์ (KS domain) ,เร่งปฏิกิริยาการรีดักชันของหมู่คาร์บอนิล (KR domain) ,เร่งปฏิกิริยาการเติ่มไฮโดรเจนของพันธะคู่ (ER domain) ,เร่งปฏิกิริยาการสูญเสียน้ำ (DH domain) ,เติมสารตั้งต้นเข้าสู่ module (AT domain) ,และส่งต่อสายโพลีคีไทด์ (ACP domain) เอนไซม์ในกลุ่มนี้สามารถแบ่งได้เป็นสองประเภทคือกลุ่มที่แต่ล่ะ module มีการทำงานเป็นเส้นตรง (Linear Modular PKS) เช่นที่ใน Erythromycin biosynthesis กับกลุ่มที่แต่ล่ะ module มีการทำงานซ้ำไปซ้ำมา (Iterative Modular PKS) ตัวอย่างของกลุ่มนี้ได้แก่ Aflatoxin biosynthesis 2. พวกที่เป็นโปรตีนขนาดเล็กที่มีเพียงหนึ่ง active site ในแต่ล่ะโปรตีน โดยแต่ล่ะโปรตีนต้องมาทำงานร่วมกันในการสังเคราะห์สารประกอบโพลีคีไทด์ กลุ่มนี้ถูกเรียกว่า Type II หรือ Iterative PKS หน่วยพันธุกรรมที่ถอดรหัสเอมไซม์เหล่านี้มักพบว่าอยู่รวมกันเป็น cluster ซึ่งทำให้ง่ายต่อการศึกษา เอมไซม์ในกลุ่มนี้ประกอบด้วย Acyl-carrier protein (ACP) , Ketosynthase (KS) , Chain Length Factor (CLF) , Ketoreducatse (KR) , Aromatase (ARO) , และ Cyclase (CYC) โดยปกติสายโพลีคีไทด์ถูกสังเคราะห์จากมาโลนิลโคเอโดย ACP, KS, และ CLF เอมไซม์ทั้งสามนี้ถูกเรียกว่า minimal PKS เพราะเป็นกลุ่มเอมไซม์ที่จำเป็นในการสังเคราะห์ เมื่อสายโพลีคีไทด์ถูกสังเคราะห์เสร็จสิ้นแล้ว KR, ARO, และ CYC ทำการเปลี่ยนแปลงโครงสร้างของสายโพลีคีไทด์ 3. พวกที่มีหนึ่ง active site ในโปรตีนแต่ตัวสารประกอบต่อกับ โคเอมไซม์เอ ในระหว่างการสังเคราะห์</doc>
