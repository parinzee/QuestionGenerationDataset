<doc id="207371" url="https://th.wikipedia.org/wiki?curid=207371" title="จอฟ้ามรณะ">จอฟ้ามรณะ จอฟ้ามรณะ คือสถานการณ์ที่ระบบปฏิบัติการคอมพิวเตอร์เกิดขัดข้อง ไม่สามารถทำงานต่อไปได้ โดยเฉพาะในระบบปฏิบัติการวินโดวส์ (Windows) ซึ่งเป็นการพยายามติดต่อกับผู้ใช้โดยจะแสดงผลเป็นหน้าจอสีน้ำเงิน (หรือสีฟ้าเช่นในเอกซ์บอกซ์) พร้อมกับรายละเอียดทางเทคนิคในจุดที่ทำให้ระบบไม่สามารถทำงานต่อไปได้ และเลขรหัสเปรียบเทียบเพื่อนำไปใช้ค้นหาในเว็บไซต์ส่วนช่วยเหลือของบริษัทผู้ผลิต การแสดงจอฟ้ามรณะจะทำให้คอมพิวเตอร์หยุดทำงานทันที เพื่อป้องกันความเสียหายที่อาจเกิดขึ้นต่อระบบ สาเหตุหลักของการเกิดจอฟ้ามรณะในวินโดวส์ตระกูลเอ็นที คือไดรเวอร์สำหรับอุปกรณ์ภายนอกไม่มีประสิทธิภาพ หรือฮาร์ดแวร์ทำงานผิดปกติ ส่วนในวินโดวส์ตระกูล 9x การใช้งานไฟล์ไดนามิกลิงก์ไลบรารี (.dll) ที่เข้ากันไม่ได้กับระบบ หรือบั๊กที่เกิดขึ้นในเคอร์เนล ก็สามารถทำให้เกิดจอฟ้ามรณะได้ นอกจากนั้นจอฟ้ามรณะสามารถเกิดขึ้นได้จากความผิดปกติทางกายภาพ เช่นความผิดปกติของหน่วยความจำ ปัญหาการจ่ายไฟฟ้า ส่วนประกอบที่มีความร้อนสูงเกินกำหนด (Overheat) หรือฮาร์ดแวร์ที่ทำงานเกินกว่าขีดจำกัดที่กำหนดไว้รูปแบบ รูปแบบ. โดยตั้งแต่ Windows Server 2012 ลงมานั้น จะพื้นหลังสีกรมท่า ตัวหนังสือสีเทา พร้อมแสดงข้อมูลที่เกิดข้อผิดพลาดด้วย ส่วน Windows 8 ขึ้นไปนั้น จะเปลี่ยนสีพื้นหลัง จากสีกรมท่า เป็นสีโทนฟ้าเข้มแทน ส่วนตัวหนังสือนั้น ช่วง Windows 95, 98 และ ME นั้น จะใช้รูปแบบตัวหนังสือล้วน โดยมีขนาด 80×25 พิกเซล , ตระกูล Windows NT ใช้ 80×50 พิกเซล บนขนาดหน้าจอ 720×400 , Windows XP ใช้ฟอนต์ Lucida Console ก่อนเป็น Windows 7 ที่จะใช้ฟอนต์ Consolas แทน ส่วน Windows 8 ขึ้นไปนั้น ใช้ฟอนต์ Segoe UI และเรนเดอร์หน้าจอแบบ Native ขนาด 640×480การเปลี่ยนแปลง การเปลี่ยนแปลง. ใน Windows 10 รุ่นทดสอบตั้งแต่ 14316 ขึ้นไป มีการเพิ่ม QR Code เพื่อดูปัญหาและวิธีการแก้ไข ผ่านโทรศัพท์มือถือ ใน Windows 10 รุ่นทดสอบตั้งแต่ 15002 ขึ้นไป จะเปลี่ยนจาก จอฟ้ามรณะ เป็น จอเขียวมรณะ ซึ่งจะรวมถึงรุ่นที่จะปล่อยในรุ่นต่อ ๆ ไปด้วย (มีผลเฉพาะผู้ที่ใช้ในรุ่น Insider Preview เท่านั้น ไม่มมีการเอาไปใช้กับรุ่นเสถียร)</doc>
