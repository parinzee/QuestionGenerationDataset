<doc id="212595" url="https://th.wikipedia.org/wiki?curid=212595" title="พิมพิไล ไชยโย">พิมพิไล ไชยโย พิมพิไล ไชยโย หรือ ต้อย เกิดปี พ.ศ. 2511 รองนางสาวไทย อันดับ 2 ประจำปี 2531 รุ่นเดียวกับ ปุ๋ย ภรณ์ทิพย์ นาคหิรัญกนก และภัสสร บุณยเกียรติประวัติ ประวัติ. พิมพิไล ไชยโย เกิดที่อำเภอเชียงกลาง จังหวัดน่าน ได้รับตำแหน่งนางสาวเชียงใหม่ ปี 2531 นางสาวลำปาง ปี 2531 และรองนางสาวไทย ปี 2531 พิมพิไลได้ตำแหน่งรองนางสาวไทย อันดับ 2 เมื่อวันที่ 2 เมษายน 2531 โดยจัดการประกวดที่ห้องบางกอกคอนเวนชั่นเซ็นเตอร์ เซ็นทรัลลาดพร้าว โดยผู้ชนะตำแหน่งนางสาวไทยคือ ปุ๋ย ภรณ์ทิพย์ นาคหิรัญกนกและได้เป็นตัวแทนสาวไทยไปประกวดนางงามนานาชาติ ปี 1989 ที่ ญี่ปุ่น หลังจากได้ตำแหน่งแล้ว ได้เข้าสู่วงการบันเทิง เป็นทั้งดาราและนักแสดง เล่นภาพยนตร์หลายเรื่อง ในช่วงหลังได้หันมาถ่ายแบบ และถ่ายภาพยนตร์แนวเซ็กซี่ รวมทั้งออกเทป ร่วมกับนางแบบแนวเซ็กซี่อีก 2 คนในชุด "สามบอมบ์" ซึ่งประกอบด้วย ฉวีวรรณ บุญปรก สินาภรณ์ พิไลลักษณ์ และพิมพิไล ไชยโยภาพยนตร์ภาพยนตร์. - 2533 กระสือซิ่ง ( คู่กับ สุชีพ ชัยสิทธิ์ ) - 2533 หยุดตัณหาซาดิสต์ (คู่กับ สุชีพ ชัยสิทธิ์ ) - 2533 ท่านผีเพี้ยน (คู่กับ สุชีพ ชัยสิทธิ์ ) - 2533 ตัณหาน้ำมันพราย (คู่กับ ธนิต พงษ์มนูญ ) - 2533 เช้งกะเด๊ะส์ (คู่กับ เกียรติ กิจเจริญ ) - 2533 ผีสำออย (คู่กับ บิณฑ์ บรรลือฤทธิ์ และ ต้อยเปา ทิฟฟานี่ ) - 2533 ก็หัวใจมันอยาก (คู่กับ สุชีพ ชัยสิทธิ์ ) - 2533 เพลิงอารมณ์ (คู่กับ ลิขิต เอกมงคล ) - 2533 พยัคฆ์สาวเห่าไฟ ( คู่กับ สามารถ พยัคฆ์อรุณ )ละครละคร. - 2535 คุณนาย ช่อง 3</doc>
