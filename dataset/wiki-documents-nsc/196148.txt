<doc id="196148" url="https://th.wikipedia.org/wiki?curid=196148" title="เดอะโกร๋น ก๊วน กวน ผี">เดอะโกร๋น ก๊วน กวน ผี ภาพยนตร์เรื่อง เดอะโกร๋น ก๊วน กวน ผี เคยออกฉายแล้วเมื่อวันที่ 20 เดือน กุมภาพันธ์ ปี พ.ศ. 2547 กำกับภาพยนตร์โดย บิณฑ์ บรรลือฤทธิ์ กำกับภาพโดย ปรีชา ชื่นชม เป็นภาพยนตร์แนว ผี/ตลกนักแสดงนักแสดง. - เทพ โพธิ์งาม ... มัคนายกหนูแดง - เพ็ญเพชร เพ็ญกุล ... กระทอก - ญดา โชติชูตระกูล ... ปิปิ๊ - สิริมา อภิรัตนพันธุ์ ... กาหลง - ค่อม ชวนชื่น ... สัปเหร่อชอบ - กิตติพงษ์ วงศ์ทอง ... เณรน้อย - เณรแอ ... เณรแอ - หยอง ลูกหยี ... แตงไท - เด๋อ เชิญยิ้ม ... สีดา - ไส เชิญยิ้ม ... ไหมดี - ไมเคิล เชิญยิ้ม ... ไมเคิล - ยาว สุโขทัย ... พระแอบ - พวง สุโขทัย ... พระแนบ - โป๊งเหน่ง เชิญยิ้ม ... พระหลบ - ชาลี ดอกกระโดน ... พระช่อน - ดอน จมูกบาน ... กำนันโด่ง - ปีเตอร์ สารีวงษ์ ... แดน - พราวนภางค์ สาครบุตร ... ดวงตา - เปขณางค์ ยอดมณี ... ลูกหมี - ศิรินรัตต์ ป้องเมือง ... ดาว - ดม ชวนชื่น ... ตาเก่ง - จันทนา ศิริผล ... ยายนุ่ม - สุพรรณี ผึ่งประเสริฐ ... นางนุ้ย - เอกวี ภักดีวงษ์ ... ผู้ใหญ่ฝอย - ชาติ พิมพ์สุวรรณ ... ตาตุ๋ย - ส่องแสง สร้อยสว่าง ... เด็กวัด 1 - ภูมิ เดชดวง ... เด็กวัด 2 - พิริยะ วันประเสริฐ ... เด็กวัด 3 - สราวุธ อินทาราม ... เด็กวัด 4 - ไชยา มิตรชัย ... ลิเก - แอน มิตรชัย ... ลิเก - เด๋อ ดอกสะเดา ... ลูกน้องหมอผี - สายัณห์ ดอกสะเดา ... หมอผีสายัณห์ - เหี่ยวฟ้า ... ผีคนแก่ - พล พันลาว ... หมอผีอีสาน - ผักกะเฉด เชิญยิ้ม ... หมอผีใต้</doc>
