<doc id="118715" url="https://th.wikipedia.org/wiki?curid=118715" title="โค้ดอิกไนเทอร์">โค้ดอิกไนเทอร์ โค้ดอิกไนเทอร์ () เป็น โปรแกรมขอบข่ายงานสำหรับพัฒนาบนเว็บ สำหรับ ผู้ที่ต้องการสร้างเว็บไซต์โดยใช้ภาษา PHP จุดมุ่งหมายของโปรแกรม มีเพื่อให้การพัฒนาซอฟต์แวร์รวดเร็วกว่าการเขียนโปรแกรมเองทั้งหมด โดยมี ไลบรารีสำหรับงานหลักๆให้มาด้วย รวมไปถึงการติดต่ออย่างง่ายๆ และโครงสร้างที่สมเหตุสมผลสำหรับการเรียกใช้งานไลบรารี โค้ดอิกไนเทอร์ช่วยให้ผู้ใช้มุ่งเน้นไปที่งานพัฒนา โดยลดปริมาณการเขียนโค้ดในแต่ละงานความสามารถของโปรแกรม ความสามารถของโปรแกรม. คล้ายกับ , โค้ดอิกไนเทอร์ช่วยให้การติดต่อกับฐานข้อมูล ง่ายขึ้นด้วยแอ็กทิฟ เรคอร์ด และสนับสนุนการใช้- สนับสนุนการใช้งาน PHP เวอร์ชัน 4 - มีขนาดเล็กมาก - รองรับการใช้งานฐานข้อมูลหลายชนิด - มีการตรวจสอบ แบบฟอร์ม และ ข้อมูล - ความปลอดภัย และ - สนับสนุน Email Sending Class สนับสนุน Attachments, อิเมล์แบบ HTML หรือ Text , โพรโทคอลหลายชนิด (sendmail, SMTP, และ Mail) - ไลบรารีสำหรับปรับแต่งรูปภาพ (cropping, resizing, rotating และอื่นๆ) สนับสนุน , , และ - สนับสนุน File Uploading Class - สนับสนุน Class - สนับสนุน - สนับสนุน Pagination - สนับสนุน Data - สนับสนุน - สนับสนุน Full Page - สนับสนุน Error Logging - สนับสนุน - สนับสนุน Scaffolding - สนับสนุน Calendaring Class - สนับสนุน Class - สนับสนุน Encoding Class - สนับสนุน Template Engine Class - สนับสนุน Class - สนับสนุน Library - สนับสนุน Class - สนับสนุน - สนับสนุน Flexible URI Routing - สนับสนุน Support for Hooks, Class Extensions, and Plugins - สนับสนุน Large library of "helper" functions</doc>
