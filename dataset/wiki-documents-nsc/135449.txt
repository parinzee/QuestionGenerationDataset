<doc id="135449" url="https://th.wikipedia.org/wiki?curid=135449" title="แลมป์">แลมป์ แลมป์ (LAMP) เป็นอักษรย่อของชุดซอฟต์แวร์เสรีสำหรับการทำเว็บไซต์ โดยตัวย่อต้นฉบับนั้นย่อมาจาก- Linux - ระบบปฏิบัติการ - Apache - เว็บเซิร์ฟเวอร์ - MySQL - ระบบจัดการฐานข้อมูล (DBMS) - PHP - ภาษาโปรแกรม (ในบางครั้งอาจหมายถึง Perl หรือ Python) ถึงแม้เทคโนโลยีแต่ละอย่างในแลมป์ จะไม่ถูกออกแบบให้ทำงานร่วมกันได้ตั้งแต่แรก แต่ LAMP ก็เป็นที่นิยมในวงกว้างเนื่องจากค่าใช้จ่ายของตัวซอฟต์แวร์มีราคาต่ำมาก (เนื่องจากเป็นซอฟต์แวร์เสรี) และชุดแลมป์สามารถหาได้ง่าย (มากับลินุกซ์ดิสทริบิวชันแทบทุกยี่ห้อ) คนที่คิดคำว่า แลมป์ เป็นครั้งแรกคือ Michael Kunze ซึ่งเขียนลงในนิตยสารคอมพิวเตอร์ภาษาเยอรมันตั้งแต่ ค.ศ. 1998 โดยบทความนั้นแสดงการใช้งานซอฟต์แวร์เสรีร่วมกัน เพื่อทดแทนซอฟต์แวร์เชิงพาณิชย์ราคาแพงชุดดัดแปลงแวมป์ ชุดดัดแปลง. แวมป์. แวมป์ (WAMP) คือ ชุดของ software ที่ประกอบด้วย Windows, Apache, MySQL และ PHP(หรือจะเป็น Perl, Python) อาจจะเรียกว่า WAMP stack หรือ WAMP Suite เป็น software ที่ช่วยในการลง Apache, MySQL และ PHP บนระบบปฏิบัติการ Windows</doc>
