<doc id="626374" url="https://th.wikipedia.org/wiki?curid=626374" title="แมคโอเอสเท็น พูม่า">แมคโอเอสเท็น พูม่า Mac OS X เวอร์ชั่น 10.1 (รหัสโปรเจกต์: Puma) เป็นระบบปฏิบัติการของคอมพิวเตอร์แมคอินทอชโดยบริษัท Apple ถูกปล่อยตัวออกตลาดเมื่อเดือนกันยายน 2544 รุ่นนี้เป็นก้าวที่สำคัญตั้งแต่ Mac OS X v10.0 และที่สำคัญ ยังเป็น Mac OS X รุ่นแรกที่คนเริ่มใช้งานอย่างจริงจัง Mac OS X v10.1 ได้ทำการปรับปรุงการเปิดใช้โปรแกรมปรับขนาดหน้าต่างให้เร็วขึ้น Dock ขยับได้ และ ส่วนประสานผู้ใช้สามารถปรับแต่งได้ การใช้งานสามมิติสามารถทำได้ดี รองรับเครื่องพิมพ์ กล้องถ่ายรูป เครื่องเล่นเพลงได้นับร้อย Mac OS X v10.1 มีทั้งหมด 5 รุ่น ได้แก่ 10.1 (5G64) 10.1.1 (5M28) 10.1.2 (5P48) 10.1.3 (5Q45) 10.1.4 (5Q125) และ 10.1.5 (5S60) Mac OS X v10.1 ต้องใช้ความต้องการเหมือน Mac OS X v10.0 และสามารถอัพเกรดฟรีสำหรับเจ้าของ Mac OS X v10.0 สำหรับผู้ที่ยังไม่มีสามารถซื้อได้ในราคา $20 ที่ Apple Store</doc>
