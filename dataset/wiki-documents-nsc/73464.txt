<doc id="73464" url="https://th.wikipedia.org/wiki?curid=73464" title="One-hot Finite State Machine">One-hot Finite State Machine One-hot Finite State Machine ถ้าสร้าง counter โดยออกแบบ flip-flop n ตัว state machine ที่ได้จะมี 2^n ตัว เช่นใช้ flip-flop 3 ตัว bก็แสดงว่ามี 3 บิต เพราะฉะนั้นก็จะมี 2^3 = 8 state แต่ว่า one hot คือ แต่ละ state จะมี 1 แค่ตัวเดียว นอกนั้นเป็น 0 หมด อย่างในกรณีนี้ จะได้แค่ ซึ่งได้แค่ 3 state จึงต้องเพิ่ม flip-flop เข้าไปอีกเพื่อที่จะให้ได้ 8 state ซึ่งดูเหมือนยุ่งยากแต่มีข้อดีคือ เราไม่ต้องแปลง output จากเดิม ถ้า output เป็น 101 เราต้องนำ output ไปใส่ decoder แปลงออกาอีกแต่ถ้าใช้แบบ one hot เราจะสามารถนำ output ออกมาใช้ได้เลย</doc>
