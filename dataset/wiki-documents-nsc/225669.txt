<doc id="225669" url="https://th.wikipedia.org/wiki?curid=225669" title="แม่นาคพระโขนง เดอะมิวสิคัล">แม่นาคพระโขนง เดอะมิวสิคัล แม่นาคพระโขนง เดอะมิวสิคัล สร้างขึ้นโดย บริษัท ซีเนริโอ จำกัด กำกับการแสดงโดย ถกลเกียรติ วีรวรรณ ที่นำตำนาน แม่นาคพระโขนง มาทำในรูปแบบของ ละครเพลง โดยเปิดทำการแสดง ณ เมืองไทยรัชดาลัย เธียเตอร์ ตั้งแต่ 20 พฤษภาคม - 28 มิถุนายน, 16 กรกฎาคม - 26 กรกฎาคม พ.ศ. 2552 รวมทั้งหมด 51 รอบการแสดง และนำกลับมาแสดงอีกครั้งโดยมีการตีความใหม่และเปลี่ยนนักแสดงนำทั้งหมด ยกเว้น นัท มีเรีย ตั้งแต่วันที่ 16 พฤษภาคม พ.ศ. 2561 เป็นต้นไปนักแสดงนักแสดงนำนักแสดง. นักแสดงนำ. - นัท มีเรีย แสดงเป็น แม่นาค : หญิงสาวผู้ซื่อสัตย์และจงรักภักดีในความรัก ความผูกพันที่เธอมีต่อพ่อมาก-สามี ทำให้เธอไม่อาจตัดใจไปสู่สุขคติได้ วิญญาณยังคงวนเวียนเพื่อรอคอยสามี น่าเสียดายที่ความหวัง ของเธอไม่อาจเป็นจริงได้แม่นาคจึง พยายามทำทุกวิถีทางเพื่อให้เธอได้ครองคู่กับพ่อมาก โดยไม่สนใจ ว่าจะต้องใช้วิธีใด - อาณัตพล ศิริชุมแสง (2552)/ภพธร สุนทรญาณกิจ (2561) แสดงเป็น พ่อมาก : ชายหนุ่มผู้มั่นคงในรัก ความรู้สึกของเขาที่มีต่อแม่นาคคือความจงรักภักดี เทิดทูน เท่าที่ผู้ชายคนหนึ่งจะบูชาภรรยาของตนเองได้ เมื่อทราบว่าแม่นาคเสียชีวิต พ่อมากเต็มไปด้วยความสับสน แน่นอนว่าเขาเลือกเชื่อภรรยามากกว่าคนอื่น และในท้ายที่สุด พ่อมากก็ต้องยอมรับความจริงอย่างเจ็บปวด - ปนัดดา เรืองวุฒิ (2552)/กุลมาศ ลิมปวุฒิวรานนท์ (2561) แสดงเป็น สร้อย : หญิงสาวชาวบ้าน เพื่อนสนิทของแม่นาค ผู้คอยดูแลและให้กำลังใจเธอมาโดยตลอดและแอบรักพ่อมากมาตั้งแต่เด็ก - สุดา ชื่นบาน (2552)/อัญชุลีอร บัวแก้ว (2561) แสดงเป็น ยายปริก : หมอตำแยผู้ทำคลอดให้แม่นาค - ติ๊ก ชีโร่ (2552)/ธชย ประทุมวรรณ (2561) แสดงเป็น อาจารย์คง : หมอผีผู้มากด้วยอาคม สามารถปราบแม่นาคให้สิ้นฤทธิ์ แม้ว่าจะต้องงัดหลายกลยุทธ์ก็ตาม - ภคชนก์ โวอ่อนศรี (2552) แสดงเป็น ยอด : เพื่อนพ่อมาก - ธงธง มกจ๊ก (2552)/จารุวัฒน์ เชี่ยวอร่าม (2561) แสดงเป็น โอ่ง : เพื่อนพ่อมาก - อู๊ด เป็นต่อ (2552) แสดงเป็น เตี้ย : เพื่อนพ่อมาก - วราวุธ โพธิ์ยิ้ม (2561) แสดงเป็น เงาะ : เพื่อนพ่อมากนักแสดงแทนนักแสดงแทน. - ปนัดดา เรืองวุฒิ, ศุภิสรา สุขสอน แสดงเป็น แม่นาค - ยอดนิยม นิยมสุขล, มฤษฏ์ เดชไหรศักดิ์ แสดงเป็น พ่อมาก - ศุภิสรา สุขสอน, ธิฆัมพร ล้ำเลิศ แสดงเป็น สร้อย - เทพธนะ ปาลกะวงศ์ ณ อยุธยา แสดงเป็น อาจารย์คง - พัฒน์ชนก เอี่ยมสะอาด แสดงเป็น ยายปริก - มฤษฏ์ เดชไหรศักดิ์ แสดงเป็น ยอด - กฤษณะ ประจงกร, ปฐมกฤษณ์ สุดสระ แสดงเป็น โอ่ง - มฤษฏ์ เดชไหรศักดิ์ แสดงเป็น เตี้ย - สุปรียา พุทธน แสดงเป็น แป้นนักแสดงสมทบ หมู่มวลนักแสดงสมทบ หมู่มวล. - กฤตินท์ เกียวติเมธา - กฤษณ สุนทรนาค - กฤษณะ ประจงกร - เก่ง จริยศาสตร์ - ดวงธิดา เลี้ยงสกุล - ทัชชา จำเนียรไวย - เทพธนะ ปาลกะวงศ์ ณ อยุธยา - ธิฆัมพร ล้ำเลิศ - นรินทร์ ภูวนเจริญ - นวพรรณ ชื่อประดิษฐ์ - ปฐมกฤษณ์ สุดสระ - เปมิศา วภักดิ์เพชร - พงศ์ปณต พัฒนธนทรัพย์ - พัฒน์ชนก เอี่ยมสะอาด - พัลลภ นามโสม - เพ็ญพิชา เพ็ญวรรณ - ภาสกรณ์ รุ่งเรืองเดชาภัทร์ - มฤษฏ์ เดชไหรศักดิ์ - ยอดนิยม นิยมสุขล - ศุภิสรา สุขสอน - สุปรียา พุทธน - อรยา เจริญกิจวาณิชย์ - อริญากร บวรวิศรุติทีมงานทีมงาน. - ประพันธ์คำร้อง โดย วิเชียร ตันติพิมลพันธ์ - ประพันธ์ดนตรี โดย สราวุธ เลิศปัญญานุช - บทละคร โดย ศุภกร เหรียญสุวรรณ , พฤกษ์ เอมะรุจิ , พิมพ์มาดา พัฒนอลงกรณ์ - กำกับดนตรี โดย พัชรพงศ์ จันทาพูน - กำกับศิลป์ โดย ศศวัต บุษยพันธ์ - กำกับลีลา โดย สุธีศักดิ์ ภักดีเทวา - ออกแบบฉาก โดย ธรณิศร ปิติกุล , วีระชัย วรรัตน์ชัยกุล , สมบัติ สมเกียรติเจริญ , อัตถสิทธิ์ พิพิธเมธานนท์ - ออกแบบเครื่องแต่งกาย โดย อุดมเวทย์ คงภักดี - ดูแลการร้อง โดย สกานดา บุณยธรรมิก , ธานี พูนสุวรรณ - กำกับแสง โดย ปรากรม์ กาญจนศรีสุขกุล - กำกับเสียง โดย ประพล สุขสันต์สมภพ - ช่วยกำกับการแสดง โดย กิตติ เชี่ยววงศ์กุล - ดำเนินการผลิต โดย อัจฉรา ชูสกุล , ชยากร สุทินศักดิ์ - ควบคุมการผลิต โดย ฤดีภัญ แสง-ชูโต , สุธาสินี บุศราพันธ์ - อำนวยการสร้างและกำกับการแสดง โดย ถกลเกียรติ วีรวรรณลำดับฉากละครเวที แม่นาคพระโขนงเดอะมิวสิคัล ลำดับฉากละครเวที แม่นาคพระโขนงเดอะมิวสิคัล. เหตุการณ์ทั้งหมดเกิดขึ้นที่บางพระโขนง เมื่อนานมาแล้ว องก์ 1- 1. เบิกโรง - นาค ( ลานวัด ) - 2. ลอยกระทง ณ บางพระโขนง - ชาวบ้าน,มาก,นาค ( ลานวัด ) - 3. อ้อมกอด - นาค,มาก ( บ้านพ่อมากและแม่นาค ) - 4. ทำคลอด - นาค,โอ่ง,ยอด,เตี้ย,สร้อย,ปริก,แป้น ( บ้านพ่อมากและแม่นาค ) - 5. อโหสิกรรม - สร้อย ( ป่าช้า ) - 6. ฤทธิ์อาจารย์คง - อาจารย์คง,นาค ( ป่าช้า ) - 7. ฉันจะรอเธอ - นาค ( ป่าช้า ) - 8. ไม่เชื่ออย่าลบหลู่ - ชาวบ้าน ( ตลาดบางพระโขนง ) - 9. อ้อมกอด(Reprise) - นาค,มาก ( บ้านพ่อมากและแม่นาค ) - 10. เรื่องราวในคืนนั้น - ปริก,ชาวบ้าน ( ตลาดบางพระโขนง ) - 11.จงปล่อยพี่ไป - นาค,มาก ( บ้านพ่อมากและแม่นาค ) - 12. อย่าหนีฉันเลย - นาค,มาก,ชาวบ้าน,ปริก ( ลานชุมชนบางพระโขนง ) องค์ 2- 13. ปีศาจร้าย - ชาวบ้าน,ปริก ( วัด ) - 14. รู้ทั้งรู้ - สร้อย ( วัด ) - 15. วิธีหนีผี - เตี้ย,โอ่ง ( ลานชุมชนบางพระโขนง ) - 16. ฉันขอโทษ - นาค ( ลานชุมชนบางพระโขนง ) - 17. รู้ทั้งรู้(Reprise) - สร้อย,มาก,นาค ( ลานชุมชนบางพระโขนง ) - 18. เพื่อนทรยศ - นาค,สร้อย ( ลานชุมชนบางพระโขนง ) - 19. ช่างน่าสงสาร - ชาวบ้าน ( ลานชุมชนบางพระโขนง ) - 20. เมียพี่ตายไปแล้ว - มาก,นาค ( ลานชุมชนบางพระโขนง ) - 21. ต่างคนต่างไป - มาก ( ลานชุมชนบางพระโขนง ) - 22. ตามหมอผีมากำจัดมัน - ชาวบ้าน,มาก ( ลานชุมชนบางพระโขนง ) - 23. ถึงเวลาทวงแค้น - อาจารย์คง ( ลานชุมชนบางพระโขนง ) - 24. พิธีปราบผี - อาจารย์คง,นาค,มาก ( ลานชุมชนบางพระโขนงและปรัมพิธี ) - 25. อ้อมกอดครั้งสุดท้าย - นาค,มาก ( ลานชุมชนบางพระโขนง ) - 26. ยามนึกถึงเรื่องของฉัน - นาค ( ลานชุมชนบางพระโขนงและเชิงตะกอนเผาศพ )รายชื่อบทเพลงประกอบละครเวที แม่นาคพระโขนงเดอะมิวสิคัล รายชื่อบทเพลงประกอบละครเวที แม่นาคพระโขนงเดอะมิวสิคัล. องค์ 1- 1. เบิกโรง - นาค - 2. ลอยกระทง ณ บางพระโขนง - ชาวบ้าน,มาก,นาค - 3. อ้อมกอด - นาค,มาก - 4. ทำคลอด - นาค,โอ่ง,ยอด,เตี้ย,สร้อย,ปริก,แป้น - 5. อโหสิกรรม - สร้อย - 6. ฤทธิ์อาจารย์คง - อาจารย์คง,นาค - 7. ฉันจะรอเธอ - นาค - 8. ไม่เชื่ออย่าลบหลู่ - ชาวบ้าน - 9. อ้อมกอด (Reprise) - นาค,มาก - 10. เรื่องราวในคืนนั้น - ปริก,ชาวบ้าน - 11. อย่าหนีฉันเลย - นาค,มาก,ชาวบ้าน องค์ 2- 12. ปีศาจร้าย - ชาวบ้าน,ปริก - 13. รู้ทั้งรู้ - สร้อย - 14. วิธีหนีผี - เตี้ย,โอ่ง - 15. รู้ทั้งรู้ (Reprise) - สร้อย,มาก,นาค - 16. เพื่อนทรยศ - นาค,สร้อย - 17. ต่างคนต่างไป - มาก - 18. พิธีปราบผี - อาจารย์คง,นาค,มาก - 19. อ้อมกอดครั้งสุดท้าย - นาค,มาก - 20. ยามนึกถึงเรื่องของฉัน -
