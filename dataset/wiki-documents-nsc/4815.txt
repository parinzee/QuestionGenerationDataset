<doc id="4815" url="https://th.wikipedia.org/wiki?curid=4815" title="ลิมิตของฟังก์ชัน">ลิมิตของฟังก์ชัน ในวิชาคณิตศาสตร์ ลิมิตของฟังก์ชัน เป็นแนวคิดพื้นฐานของ คณิตวิเคราะห์ (ภาคทฤษฎีของแคลคูลัส) ถ้าเราพูดว่า ฟังก์ชัน f มีลิมิต L ที่จุด p หมายความว่า ผลลัพธ์ของ f จะเข้าใกล้ L ที่จุดใกล้จุด p สำหรับนิยามอย่างเป็นทางการนั้น มีการกำหนดขึ้นครั้งแรก ช่วงปลายของคริสต์ศตวรรษที่ 19 มีรายละเอียดอยู่ข้างล่าง ดูที่ ข่ายลำดับ (topology) สำหรับนัยทั่วไปของแนวคิดของลิมิตประวัติ ประวัติ. ดูที่ คณิตวิเคราะห์นิยามเป็นทางการฟังก์ชันบนปริภูมิอิงระยะทาง นิยามเป็นทางการ. ฟังก์ชันบนปริภูมิอิงระยะทาง. กำหนดให้ f : (M,d) -> (N,d) เป็นการส่งค่าระหว่าง (เป็นฟังก์ชันที่นิยามบน) ปริภูมิอิงระยะทาง สองปริภูมิ, และกำหนดให้ p &isin;M และ L &isin;N, เราจะกล่าวว่า "ลิมิตของ f ที่ p คือ L" และเขียนว่า:formula_1ก็ต่อเมื่อ สำหรับทุกค่าของ &epsilon; > 0 จะมี &delta; > 0 ที่ สำหรับทุกๆ x &isin;M และ d(x, p) < &delta; แล้ว, d(f(x), L) < &epsilon;ฟังก์ชันค่าจริง ฟังก์ชันค่าจริง. เซตของจำนวนจริงหรือเส้นจำนวนจริง โดยทั่วไปสามารถมองเป็นปริภูมิอิงระยะทางได้ โดยมี formula_2. เช่นเดียวกับ เส้นจำนวนจริงขยาย (เส้นจำนวนจริงที่เพิ่ม +∞ และ -∞ เข้าไปด้วย) ก็สามารถมองเป็นปริภูมิอิงระยะทางได้ โดยมี formula_3ลิมิตของฟังก์ชันค่าจริงที่จุดใดจุดหนึ่ง ลิมิตของฟังก์ชันค่าจริงที่จุดใดจุดหนึ่ง. ให้ f เป็นฟังก์ชันค่าจริง แล้วเราจะเขียน formula_4 ก็ต่อเมื่อ สำหรับทุกค่าของ &epsilon; > 0 (ไม่ว่าจะเล็กเท่าใด) จะต้องมี &delta; > 0 อย่างน้อยหนึ่งค่า ที่ สำหรับทุกค่าของจำนวนจริง x ที่ 0 < |x-p| < &delta;, |f(x)-L| < &epsilon; ซึ่งเป็นกรณีพิเศษของฟังก์ชันบนปริภูมิอิงระยะทาง ที่มีทั้ง M และ N เป็นเซตของจำนวนจริง และ d(x,y) = |x-y|. หรือเราจะเขียน formula_5 ก็ต่อเมื่อ สำหรับทุกค่าของ R > 0 (ไม่ว่าจะใหญ่เท่าใด) จะต้องมี &delta;
