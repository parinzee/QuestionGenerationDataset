<doc id="182571" url="https://th.wikipedia.org/wiki?curid=182571" title="J2EE">J2EE J2EE (Java 2 Platform Enterprise Edition) เป็นแพล็ตฟอร์มที่ออกแบบสำหรับเครื่องคอมพิวเตอร์ระดับ เมนเฟรมของวิสาหกิจขนาดใหญ่ ซันไมโครซิสเตม (Sun Microsystems) ได้ออกแบบ สามารถพัฒนาโปรแกรมประยุกต์ง่ายขึ้นโดย J2EE และลดส่วนของการเขียนโค้ดและให้บางส่วนถูก ควบคุมอย่างอัตโนมัติ J2EE ได้รวมส่วนประกอบจำนวนมากของแพล็ตฟอร์ม J2SE (Java 2 Standard Edition) เช่น- Java Development Kit (JDK) เพื่อให้ระบบปฏิบัติการสนับสนุนการทำงานของภาษาจาวา - สนับสนุน Common Object Request Broker Architecture (CORBA) รุ่นก่อนที่จะเปลี่ยนมาเป็น Enterprise JavaBeans (EJB) ในปัจจุบัน - Java Database Connectivity 2.0 (JDBC) เทียบได้กับ Open Database Connectivity (ODBC) ได้รับการรวมเป็นการอินเตอร์มาตรฐานกับฐานข้อมูลของ Java</doc>
