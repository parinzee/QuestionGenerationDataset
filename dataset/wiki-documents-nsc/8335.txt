<doc id="8335" url="https://th.wikipedia.org/wiki?curid=8335" title="โรงเรียนนักสืบ Q">โรงเรียนนักสืบ Q โรงเรียนนักสืบ Q () () เป็นการ์ตูนญี่ปุ่นแนวสืบสวนสอบสวน แต่งเรื่องโดย เซย์มารุ อามางิ และวาดภาพโดย ฟุมิยะ ซาโต้ ซึ่งเคยมีผลงานร่วมกันมาก่อนในเรื่อง คินดะอิจิกับคดีฆาตกรรมปริศนา การ์ตูนนักสืบที่ได้รับความนิยมอย่างสูงอีกเรื่องหนึ่ง ต่อมาได้มีการนำไปสร้างเป็นการ์ตูนแอนิเมชันทางโทรทัศน์โดยมีเนื้อหาบางส่วนที่ดัดแปลงไปจากฉบับหนังสือการ์ตูน อีกทั้งยังมีการสร้างเป็นละครโทรทัศน์แล้วเนื้อเรื่อง เนื้อเรื่อง. "คิว" เด็กหนุ่มวัย 13 ปี ผู้ใฝ่ฝันอยากเป็นนักสืบอันดับหนึ่งของโลก ได้สมัครเข้าสอบ DDS (Dan Detective School) โรงเรียนนักสืบชื่อดัง เขาต้องฝ่าฝันอุปสรรคต่างๆ มากมายจนสุดท้ายได้กลายเป็นส่วนหนึ่งของ Q Class (Qualified Class) ชั้นเรียนพิเศษที่ทำการสอนโดยตรง โดย ดัน โมริฮิโกะ ยอดนักสืบอันดับหนึ่งของญี่ปุ่น คิวและสมาชิกคนอื่นอีก 4 คนที่เป็นทั้งเพื่อนและคู่แข่งต้องหาทางไขปริศนาต่างๆ รวมถึงต้องขับเคี่ยวกับ "มฤตยู" องค์กรลับที่วางแผนฆาตกรรมได้สมบูรณ์แบบจนมีแต่นักสืบผู้ชาญฉลาดเท่านั้นที่มองแผนนั้นออกตัวละครQ Classตัวละคร. Q Class. - คิวหรือเร็นโจ คิว(キュウ,Renjo Q) - อายุ:14 - ส่วนสูง:161 ซม. - น้ำหนัก:53 กก. - กรุ๊ปเลือด O คิวหรือเร็นโจ คิว เด็กชายผู้เฉลี่ยวฉลาด เขาเป็นเด็กหนุ่มที่มุ่งมั่นในการจะก้าวไปเป็นนักสืบ เพื่อที่จะได้เป็นนักสืบ เขาจึงไปสอบเข้าโรงเรียนนักสืบ ดูภายนอก เขาอาจจะเอ๋อๆ สักหน่อย แต่ความจริงแล้ว เขามีสติปัญญาหลักแหลม และพร้อมแก้ไขปัญหาต่างๆ ได้อย่างรู้จริง ซึ่งเค้าบังเอิญได้เจอกับนักสืบคนหนึ่งในตอนเด็กๆ เค้าจึงตั้งเป้าหมายไว้ว่า จะเป็นนักสืบเหมือนกับนักสืบคนที่เค้าเจอตอนนั้น โดยเค้าไม่รู้เลยว่านักสืบที่เค้าเจอนั้นคือพ่อของตนเอง- มินามิ เมงุมิ (Minami megumi) - อายุ:10 - ส่วนสูง:142ซม. - น้ำหนัก:38 กก. - กรุ๊ปเลือด B โปรแกรมเมอร์อัจฉริยะ แม้จะเป็นเพียงเด็กประถมแต่ก็สามารถสร้างเกมส์ขายได้อย่างโด่งดัง เป็นนักสืบที่อาศัยคอมพิวเตอร์ในการสืบหาข้อมูล ด้วยความสามารถในการใช้คอมพิวเตอร์ทำให้คาซึมะเป็นฐานข้อมูลสำคัญในการสืบคดีต่างๆ ด้วยอายุที่ยังน้อยอยู่อาจารย์ดันจึงนิยมให้คาซึมะเป็นฝ่ายประสานงานข้อมูลอยู่ที่โรงเรียน ส่วนอีก 4 คนนั้นให้ออกปฏิบัติหน้าที่ทางด้านนอก และส่งข้อมูลกลับมาให้คาซึมะ นอกจากนี้คาซึมะยังเชี่ยวชาญทางด้านการถอดรหัสลับมากจนถือว่าไม่มีใครเทียบได้ทีเดียวA ClassA Class. - ยูทากะ ซาบุโร่มารุ (Yutaka Saburomaru) : ซาบุโร่มารุเป็นนักศึกษามหาวิทยาลัยโทได ด้วยความสามารถ IQ 180 แต่กลับไม่มีความสามารถด้านการสืบสวนสอบสวนเลย สาเหตุที่ได้เข้ามาอยู่ใน A Class เนื่องจากได้ช่วยอาจารย์ดัน โมริฮิโกะ ในการสอบเข้าชั้น Q Class แต่หลังจากนั้นกลับไม่มีผลงานอีกเลย ภายหลังผลการเรียนตกลงทำให้ต้องย้ายไปอยู่ชั้น B Class- ยูคิฮิระ ซากุระโกะ (Yukihara Sakurako) : ซากุระโกะผู้ที่มีคะแนนสูงสุดของชั้น A Class และหวังที่จะเป็นผู้สืบทอดของอาจารย์ดัน โมริฮิโกะ ความจริงแล้วเธอเป็นหลานของอาจารย์ดันแต่ไม่มีผู้ใดรู้ แม้แต่อาจารย์ภายใน DDS เองก็มีผู้ที่รู้เรื่องนี้เพียงส่วนน้อยเท่านั้น ซากุระโกะมีความสามารถในการสืบสวนสอบสวนมาก ได้เข้ามามีส่วนร่วมในการสืบสวนคดีต่างๆ ตั้งแต่เด็ก เคยได้รับรางวัลพิเศษจากผู้บัญชาการตำรวจตั้งแต่ประถม 3 และสามารถพูดได้ทั้งหมด 4 ภาษา- ชิชิโดะ ทาเคชิ (Shishido Takeshi) : ทาเคชิเป็นนักศึกษามหาวิทยาลัยฮาวาร์ด แต่ปัจจุบันพักการเรียนมาเรียนอยู่ที่ DDS เมื่ออายุ 20 ปีได้รับปริญญาเอกด้านจิตวิทยาอาชญากรรม- ชิรามิเนะ ฮายาโตะ (Shiramine Hayato) : ฮายาโตะผู้มีความสามารถทางด้านมายากล เมื่ออายุได้ 8 ขวบเคยได้เปิดการแสดงที่ลาสเวกัสและเป็นที่กล่าวขวัญกันมาก เนื่องจากเป็นผู้มีความสามารถทางด้านมายากล ทำให้สามารถมองเห็นจุดบอดต่างๆ ที่บุคคลทั่วไปไม่ได้ให้ความสนใจได้- โกดะ เคียวสุเกะ (Gouda Kyousuke) : เคียวสุเกะเป็นอัจฉริยะทางด้านคณิตศาสตร์ เคยได้ตำแหน่งระดับโลกในการแข่งขันโอลิมปิกคณิตศาสตร์ตอนอายุ 17 ปี มักจะใช้คณิตศาสตร์เข้ามาช่วยในการไขคดีต่างๆ- โทยะ คุนิโกะ (Kuniko Touya) : คุนิโกะเป็นผุ้ต้องสงสัยในคดี "คอลเล็กเตอร์ฆาตกร" และภายหลังได้เข้าสอบใน DDS จนสามารถเข้ามาอยู่ในชั้น A Class ได้ สาเหตุที่เข้ามาเรียนใน DDS คาดว่ามาจากคุนิโกะต้องการอยู่กับริวมากกว่าคณาจารย์และเจ้าหน้าที่คณาจารย์และเจ้าหน้าที่. - ดัน โมริฮิโกะ (Morihiko Dan) เกิดเมื่อปี 1936 เป็นผู้ก่อตั้งและอาจารย์ใหญ่ของโรงเรียนนักสืบแห่งนี้ นอกจากนี้เขายังเป็นนักสืบที่โด่งดัง คลี่คลายคดีต่างๆ นับไม่ถ้วน และยังเป็นนักสืบเอกชนที่ได้รับการอนุญาตให้ถือปืนและมีอำจาจจับกุมโดยถูกต้องตามกฎหมายคนเดียวในญี่ปุ่น ดัน โมริฮิโกะได้ลาออกจากการเป็นตำรวจเพื่อเปิด สำนักงานนักสืบเอกชน แห่งหนึ่ง โดยมีคู่หูของเขา เรนโจ ซาโตรุ (พ่อของคิว) ที่ไม่ได้เป็นเพียงแค่คู่หู แต่ยังเป็นทั้งศิษย์และผู้ช่วยมือขวาของเขา ทั้งสองคนได้ช่วยกันสร้าง DDC (Dan Detective Company) และ DDS สำนักงานนักสืบมือหนึ่ง ในหมู่ผู้ก่ออาชญากรรม มีองค์กรลับใต้ดินนาม Pluto ทั้งสองได้เอาชนะ Pluto ได้ จึงทำให้องค์กรลับแห่งนี้แตกไปเพียงชั่วคราว ก็อย่างที่ผู้คนว่าไว้ ไม่มีของฟรีในโลก ได้อะไรมาก็ต้องเสียอะไรไป โมริฮิโกะได้เสียเพื่อนรักของเขา คู่หู และศิษย์รัก ซาโตรุ เรนโจในศึกครั้งนั้น และตัวเองต้องตกอยู่ในสภาพนั่งรถเข็นไปตลอดชีวิต หลายปีต่อมา องค์กร Pluto ได้ถือกำเนิดขึ้นอีกครั่ง โมริฮิโกะจึงได้ตัดสินใจเปิด คลาส Q ขึ้น ซึ่งประกอบด้วยนักเรียนที่มีความโดดเด่นเหนือจินตนาการ เขาจึงฝึกฝนเด็กเหล่านี้เพื่อที่ห้องนี้จะได้กลายผู้สืบทอดเจตนารมณ์ของเขารายชื่อตอน</doc>
