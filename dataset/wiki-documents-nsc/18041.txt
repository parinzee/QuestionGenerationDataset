<doc id="18041" url="https://th.wikipedia.org/wiki?curid=18041" title="ไวรัส A2K.Damcor">ไวรัส A2K.Damcor A2K.Damcor เป็นไวรัสคอมพิวเตอร์ ที่เขียนด้วย มาโคร ซึ่งอยู่ในรูปแบบไฟล์ .mdb และกระจายตนเองผ่านการแนบไฟลของ ไมโครซอฟท์ เอาท์ลุค. หนอนตัวนี้ถูกเขียนด้วยสคริป์ ไมโครซอฟท์ Visual Basicระบบปฏิบัติการที่มีผลกระทบระบบปฏิบัติการที่มีผลกระทบ. - Windows 2000 - Windows 95 - Windows 98 - Windows Me - Windows NT - Windows Server 2003 - Windows XPข้อมูลทางเทคนิค ข้อมูลทางเทคนิค. เมื่อ A2k.Damcor ทำงาน, มันจะทำงานดังนี้: 1. เปิด ไมโครซอฟท์ แอคเซส เพื่อสั่งให้รหัสมาโครทำงาน. 2. เปิดใช้งาน ไมโครซอฟท์ เอาท์ลุค เพื่อส่ง อีเมลตัวเองที่มีอยู่ใน Address book. 3. อีเมลดังกล่าวจะมีรูปแบบดังนี้: 4. ปิดไมโครซอฟท์ แอคเซส.</doc>
