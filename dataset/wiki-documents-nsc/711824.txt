<doc id="711824" url="https://th.wikipedia.org/wiki?curid=711824" title="ราชสหพันธ์ฟุตบอลสเปน">ราชสหพันธ์ฟุตบอลสเปน ราชสหพันธ์ฟุตบอลสเปน () หรือ เอร์เรเฟฟ () เป็นหน่วยงานที่ทำหน้าที่ดูแลกิจการฟุตบอลต่าง ๆ ในประเทศสเปน ก่อตั้งขึ้นเมื่อปี ค.ศ. 1913 โดยสืบทอดองค์กรมาจากสหพันธ์สโมสรฟุตบอลสเปน () ซึ่งได้รับการก่อตั้งในปี ค.ศ. 1909 ปัจจุบันสหพันธ์มีสำนักงานใหญ่อยู่ที่นครฟุตบอลในเมืองลัสโรซัส ทางทิศตะวันตกของกรุงมาดริด ราชสหพันธ์ฟุตบอลสเปนเป็นสมาชิกสหพันธ์ฟุตบอลระหว่างประเทศ, สหภาพสมาคมฟุตบอลยุโรป และคณะกรรมการโอลิมปิกสเปน มีหน้าที่จัดการแข่งขันในระดับชั้นที่ 3 ของระบบลีกฟุตบอลสเปน และดูแลการแข่งขันในระดับชั้นที่ 1, ระดับชั้นที่ 2 (จัดโดยสันนิบาตฟุตบอลอาชีพแห่งชาติ) และระดับชั้นที่ 4 (จัดโดยสหพันธ์ฟุตบอลของแต่ละแคว้นและนครปกครองตนเอง) นอกจากนี้ยังเป็นผู้รับผิดชอบการบริหารจัดการฟุตบอลทีมชาติสเปน (ชาย), ฟุตบอลหญิงทีมชาติสเปน, ฟุตบอลเยาวชนทีมชาติสเปน รวมทั้งฟุตซอลทีมชาติสเปนอีกด้วยการแข่งขัน การแข่งขัน. นอกจากการแข่งขันต่าง ๆ ดังกล่าวแล้ว ราชสหพันธ์ฟุตบอลสเปนยังเป็นผู้จัดการแข่งขันต่อไปนี้- ถ้วยรางวัลระดับชาติ :- โกปาเดลเรย์ - ซูเปร์โกปาเดเอสปาญา - โกปาเฟเดราซีออน - การแข่งขันฟุตบอลหญิง :- ปรีเมราดีบีซีออนเฟเมนีนา - เซกุนดาดีบีซีออนเฟเมนีนา - โกปาเดลาเรย์นา - การแข่งขันฟุตบอลเยาวชน :- โกปาเดกัมเปโอเนสคูเบนิล - โกปาเดลเรย์คูเบนิล - ดีบีซีออนเดโอนอร์คูเบนิล - ลีกานาซีโอนัลคูเบนิลหน่วยงานระดับท้องที่ปกครองตนเอง หน่วยงานระดับท้องที่ปกครองตนเอง. ราชสหพันธ์ฟุตบอลสเปนประกอบด้วยสหพันธ์ฟุตบอลในระดับภูมิภาค 19 สหพันธ์ แต่ละสหพันธ์เป็นตัวแทนของแคว้นและนครปกครองตนเองต่าง ๆ ในประเทศสเปน- กันตาเบรีย : สหพันธ์ฟุตบอลกันตาเบรีย () - กัสตียา-ลามันชา : สหพันธ์ฟุตบอลกัสตียา-ลามันชา () - กัสตียาอีเลออน : สหพันธ์ฟุตบอลกัสตียาอีเลออน () - กาตาลุญญา : สหพันธ์ฟุตบอลกาตาลุญญา () - กาลิเซีย : สหพันธ์ฟุตบอลกาลิเซีย () - เซวตา : สหพันธ์ฟุตบอลเซวตา () - นาบาร์รา : สหพันธ์ฟุตบอลนาบาร์รา () - บาเลนเซีย : สหพันธ์ฟุตบอลแห่งแคว้นบาเลนเซีย () - ประเทศบาสก์ : สหพันธ์ฟุตบอลบาสก์ (; ) - ภูมิภาคมูร์เซีย : สหพันธ์ฟุตบอลแห่งภูมิภาคมูร์เซีย () - มาดริด : ราชสหพันธ์ฟุตบอลมาดริด () - เมลียา : สหพันธ์ฟุตบอลเมลียา () - ลารีโอคา : สหพันธ์ฟุตบอลลารีโอคา () - หมู่เกาะคะแนรี : สหพันธ์ฟุตบอลคะแนรี () - หมู่เกาะแบลีแอริก : สหพันธ์ฟุตบอลหมู่เกาะแบลีแอริก () - อันดาลูซีอา : สหพันธ์ฟุตบอลอันดาลูซีอา () - อัสตูเรียส : ราชสหพันธ์ฟุตบอลแห่งราชรัฐอัสตูเรียส () - อารากอน : สหพันธ์ฟุตบอลอารากอน () - เอซเตรมาดูรา : สหพันธ์ฟุตบอลเอซเตรมาดูรา ()</doc>
