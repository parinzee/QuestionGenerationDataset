<doc id="50306" url="https://th.wikipedia.org/wiki?curid=50306" title="C">C C (ตัวใหญ่:C ตัวเล็ก:c) เป็นอักษรละติน ลำดับที่ 3ชื่อเรียกชื่อเรียก. - ในภาษาอังกฤษ เรียกว่า "ซี" ([siː]) - ในภาษาเยอรมัน ฝรั่งเศส และสเปน เรียกว่า "เซ" และผู้พูดโดยมากในประเทศสเปนมักเรียก "เธ" (/θ/) - ในภาษาอิตาลี เรียกว่า "ชิ"ความหมายของ C และ cความหมายของ C และ c. - ในภาษาโปรแกรม ซี ซีพลัสพลัส ซีชาร์ป - C เป็นสัญลักษณ์ของธาตุคาร์บอน - รหัสสีคอมพิวเตอร์ CMYK จะใช้ C แทน สีไซอัน (สีฟ้าสว่าง) - ในการเขียนเลขโรมัน C มีค่าเท่ากับ 100 - ในกีฬาบาสเกตบอล C ใช้แทน เซ็นเตอร์ - ในกีฬาฟุตบอล CB ใช้แทน เซนเตอร์แบ็ก - ในหน่วยเงิน c หรือ ¢ ใช้แทน เซนต์อื่นๆอื่นๆ. - ภาษามลายูใช้แทนเสียง /จ/ - ภาษาบาลี ใช้แทนเสียงไม่ก้อง ไม่มีลม วรรคจะ ตรงกับรูปอักษรไทย จ และเสียงอักษรไทย /จ/ - С ในอักษรซีริลลิก ที่เขียนคล้ายกัน - "soft c" ซึ่งออกเสียง /s/ ในภาษาอังกฤษ เมื่อถอดรูปอักษรเป็นอักษรไทย ในคำทับศัพท์หรือศัพท์ยื้ม ส่วนใหญ่ย่อมใช้ ซ เช่น ออนซ์ (ounce), ฟัสซิสต์ (Fascist), เซนติเมตร (centimeter), เซล (cell) เป็นต้น และ ศ เช่น ไอศกรีม (ice cream), ออฟฟิศ (office), ศิวิไลซ์ (civilize) เป็นต้น</doc>
