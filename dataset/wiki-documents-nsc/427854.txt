<doc id="427854" url="https://th.wikipedia.org/wiki?curid=427854" title="Usavich ซี้ป่วนก๊วนกระต่าย">Usavich ซี้ป่วนก๊วนกระต่าย Usavich ซี้ป่วนก๊วนกระต่าย (; ) ชื่อเรื่องมาจากคำว่า อุซะงิ ( "กระต่าย") กับคำว่า Vich ที่ทำให้ดูเหมือนภาษารัสเซีย เป็นอะนิเมะสั้นของญี่ปุ่น ความยาวตอนละ 1 นาทีเศษ แต่เดิมสร้างขึ้นเพื่อบริการโทรศัพท์มือถือเอ็มทีวีญี่ปุ่น "Flux" และถูกสร้างขึ้นออกอากาศทางเอ็มทีวีญี่ปุ่น โดยมีซะโตะชิ โทะมิโอะกะและสตูดิโอกะนะบังกราฟิกส์ของเขาเป็นผู้สร้าง เนื้อเรื่องเป็นเรื่องเกี่ยวกับชีวิตของกระต่ายนักโทษสองตัว 2 ตัวกับการดำรงชีวิตในคุกโซเวียตช่วงฤดูกาลแรก และฤดูกาลต่อ ๆ มาจะเป็นเรื่องราวของการผจญภัยและหลบหนีตำรวจ มีทั้งหมด 5 ฤดูกาลเรื่องย่อเรื่องย่อ. - ฤดูกาลที่ 1 - ฤดูกาลที่ 2 - ฤดูกาลที่ 3 - ฤดูกาลที่ 4 - ฤดูกาลที่ 5ตัวละครตัวละครหลักตัวละคร. ตัวละครหลัก. - คีเรเนนโก (; ) กระต่ายที่ใส่เสื้อหมายเลข 04 สีขาวสลับแดง อดีตหัวหน้าแก๊งมาเฟีย ซึ่งต่อมาเข้าติดคุกรอโทษประหาร และร่วมห้องเดียวกับปูติน เขาเป็นคนที่ไม่พูดมาก แต่หากโกรธก็จะทำลายทุกอย่างจนราบเป็นหน้ากลอง และหากเขาโกรธยิ่งขึ้นไปอีกก็จะกลายเป็นปีศาจ เขาคลั่งไคล้รองเท้าผ้าใบมาก และโปรดปรานการอ่านหนังสือ ทั้งนี้นิสัยดังเดิมของเขาคือบุคลิกดุดัน แต่หลังจากการผ่าตัดกับคีลล์เนนโก ผู้เป็นฝาแฝดซึ่งเป็นคนที่นิ่งๆ ทำให้เขากลายเป็นบุคคลสองบุคลิก - ปูติน (; ) กระต่ายที่ใส่เสื้อหมายเลข 541 สีขาวสลับเขียว เขารักการเต้นโคซาชอก () และเคยละเมอเต้นด้วย อดีตเขาเป็นคนงานที่ซื่อสัตย์ แต่พลาดงานเพราะเมาค้าง จึงถูกตัดสินให้จำคุกสามปีโดยติดคุกร่วมห้องกันกับคีเรเนนโก ปูตินฉลาดในเรื่องการประดิษฐ์หุ่นยนต์, ดัดแปลงรถ และแยกชิ้นส่วนปืน แต่เขามักผิดพลาด ทำตัวเฉิ่ม และตื่นตกใจง่าย - เลนินกราด (; ) เป็นกบที่คีเรเนนโกและปูตินพบจากในชักโครก ขณะที่คีเรเนนโกใช้เบ็ดที่ติดโคมาเนตซึนลงไปเก็บลูกปิงปองในตอนที่ 4 เลนินกราดกินโคมาเนตซึนเข้าไปบ่อย ๆ และถ่ายมูลออกมาเป็นลูกเจี๊ยบ เมื่อเขาส่งเสียงร้องปูตินจะมีอาการหอบเป็นจังหวะเพลง ทั้งนี้เลนินกราดมีแม่ชื่อ ชาราโปวา (; ) อาศัยอยู่กับสามีชื่อมีฮาอิล (; ) ซึ่งเป็นกบตัวใหญ่ด้วยกันในป่าอันห่างไกล - โคมาเนตซึน (; ) ลูกเจี๊ยบกะเทยที่ปรากฏตัวออกมาในตอนที่ 2 ขณะที่ปูตินกำลังคัดแยกเพศของลูกเจี๊ยบ แม่ของโคมาเนตซึน () เคยมาเยี่ยมโคมาเนตซึนในคุกครั้งหนึ่ง แต่ถูกคีเรเนนโกจับไปกินเสีย ทั้งนี้โคมาเนตซึนมักตายบ่อย ๆ โดยถูกเลนินกราดกิน แต่ก็กลับฟื้นขึ้นมาได้ตลอด ต่อมาได้ตายเพราะสูดดมสารพิษกลายเป็นวิญญาณในตอนที่ 28 และสามารถสิงร่างของซูร์โซลอฟได้ โคมาเนตซึนโตขึ้นในฤดูกาลที่ 4 และมีลูกชื่อ โคมาเนตซึนจูเนียร์ - เมคาเนนโก (; ) หุ่นยนต์ที่ปูตินสร้างขึ้นมาในตอนที่ 35 ซึ่งตอนนั้นคีเรเนนโกตายและยังไม่ฟื้น ทั้งนี้ปูตินได้ดัดแปลงจากตู้ควบคุมไฟฟ้ามาเป็นเมคาเนนโกโดยถอดแบบมาจากคีเรเนนโก ซึ่งมีประสิทธิภาพในการทำลายล้าง และมักก่อเรื่องวุ่น ๆ ในช่วงฤดูกาลที่ 4 และท้ายฤดูกาลนั้นเมคาเนนโกถูกดัดแปลงเป็นรถยนต์ใช้หนีตำรวจไปในป่าจนถึงฤดูกาลที่ 5ตัวละครรองตัวละครรอง. - ผู้คุมนักโทษ เป็นผู้คุมที่จะเข้ามาตรวจในห้องของคีเรเนนโกและปูติน โดยจะเป็นลักษณะเหมือนประตู แต่จะมองเห็นคีเรเนนโกและปูตินได้ไม่ชัดนัก อย่างไรก็ตามหลังประตูดังกล่าวก็พบว่าผู้คุมนั้นเป็นกระต่ายอยู่หลังประตูในตอนที่ 3 มีทั้งหมด 4 ตัวได้แก่- คันชคอฟ (; ) – เจ้าหน้าที่เบ็ดเตล็ด ตั้งชื่อมาจากคำญี่ปุ่นว่า คังชุ () แปลว่า "ผู้คุม" - รูดอฟ (; ) – หัวหน้าเจ้าหน้าที่ ตั้งชื่อมาจากคำญี่ปุ่นว่า โรโด () แปลว่า "แรงงาน" - เซนีรอฟ (; ) – พนักงานการเงิน ตั้งชื่อมาจากคำญี่ปุ่นว่า เซะนิ () แปลว่า "เงินสด" - โชเคสกี (; ) – เพชฌฆาต ตั้งชื่อมาจากคำญี่ปุ่นว่า โชะเกะ () แปลว่า "ประหาร" ผู้คุมนักโทษทั้งหมดจะปรากฏเฉพาะในฤดูกาลที่ 1 เท่านั้น และจะพบอีกครั้งในท้ายฤดูกาลที่ 3 บนรถของตำรวจ- บอริส (; ) และคอปเซฟ (; ) เป็นตำรวจที่ตามจับคีเรเนนโก โดยบอริสจะเป็นผู้ไล่ยิง และคอฟเซฟจะเป็นผู้ขับรถให้ และปรากฏตัวเป็นอริหลักในฤดูกาลที่ 2 ที่พยายามจับคีเรเนนโกแต่ล้มเหลวมาโดยตลอด - ซูร์โซลอฟ (; ) เป็นเจ้าของคาสิโน ปรากฏเป็นคู่ปรับคนสำคัญของคีเรเนนโกในฤดูกาลที่ 3 โดยเขาจะนั่งแนบเนื้ออยู่กับสองสาวฝาแฝด เขามีนิสัยโลภ, หลงตัวเอง และชอบโชว์ ตลอดฤดูกาลที่ 3 เขาพยายามฆ่าคีเรเนนโกเพื่อเอาเงิน ทั้งนี้ซูร์โซลอฟเป็นที่ต้องการของตำรวจมีค่าหัวถึงหนึ่งล้านรูเบิล ก่อนหน้านี้ซูร์โซลอฟเคยพยายามฆ่าคีเรเนนโกและคีลล์เนนโกโดยใช้รองเท้าผ้าใบเป็นเหยื่อ แต่คีเรเนนโกก็มิได้มีทีท่าว่าจำเรื่องดังกล่าวได้เลย - โซยา (; ) และซีลียา (; ) สองสาวฝาแฝดที่อยู่กับซูร์โซลอฟ โซยาจะมีตัวเป็นสีเหลือง ส่วนซีลียาจะมีตัวเป็นสีแดง ภายหลังได้หนีซูร์โซลอฟในตอนที่ 39 เพราะกลัวคีเรเนนโก - ซามูชาบิน () เป็นตัวละครที่ปรากฏตัวสั้น ๆ เป็นหนึ่งในกลุ่มอันธพาลหน้าลานเต้นรำบนชั้นห้าในกาสิโนของซูร์โซลอฟ ซึ่งโดดเด่นกว่าตัวละครอันธพาลตัวอื่นตรงที่สวมแว่นตาสีดำ และสวมหมวกที่มีสัญลักษณ์ "w" ซึ่งเป็นสัญลักษณ์ของ "w-inds." วงบอยแบนด์ของญี่ปุ่น โดยสัญลักษณ์ดังกล่าวเคนตะ ทะชิบะนะ สมาชิกของวงเป็นผู้ออกแบบ - คีลล์เนนโก (; ) ฝาแฝดของคีเรเนนโก ซึ่งเป็นหัวหน้ามาเฟียคนปัจจุบัน คลั่งไคล้ในรองเท้าผ้าใบเช่นเดียวกับคีเรเนนโก เนื่องจากทั้งสองจะถูกซูร์โซลอฟฆ่าด้วยการระเบิดโดยใช้รองเท้าผ้าใบเป็นเหยื่อล่อ ภายหลังร่างของทั้งสองได้รับการผ่าตัดสลับกัน (สังเกตได้จากใบหน้าของคีเรเนนโกที่มีรอยเย็บและสีผิวไม่เสมอกัน) จึงทำให้ทั้งคีลเนนโกและคีเรเนนโกเป็นบุคคลสองบุคลิก ทั้งนี้คีลเนนโกปรากฏตัวในฤดูกาลที่ 4 ผ่านจอเมคาเนนโกเพื่อดูรองเท้าผ้าใบสีแดงคู่ใหม่ของคีเรเนนโกที่ได้จากซูร์โซลอฟ และปรากฏตัวอีกครั้งใน ตอนที่ 59 หลังจากที่ พ่อกบตัวใหญ่ได้กินคิเรเนนโกเข้าไปแล้วปูตินทำให้พ่อกบจามออกมาก ปรากฏว่ามี คีลล์เนนโก โผล่มาด้วย - เคดามูสกี () ปรากฏตัวครั้งแรกในตอนที่ 53 ของฤดูกาลที่ 5 เป็นกระต่ายป่าตัวใหญ่สีเทา มีลายจุดสีดำ ชอบแสยะยิ้มยิงฟัน เขามักลอบขโมยกระเป๋าส่วนตัวของคีเรเนนโกเพื่อที่จะกินแครอทขนาดใหญ่ที่อยู่ในนั้น เคดามูสกีแม้จะมีรูปร่างใหญ่แต่ก็วิ่งหนีคีเรเนนโกได้อย่างรวดเร็ว และเขามักกินอาหารอย่างตะกละตะกลามเช่นเดียวกับปูตินเพราะว่าเขาอาศัยอยู่ในป่าเป็นเวลานานรายชื่อตอน รายชื่อตอน.      </doc>
