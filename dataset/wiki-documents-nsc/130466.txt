<doc id="130466" url="https://th.wikipedia.org/wiki?curid=130466" title="ไอไอเอส">ไอไอเอส อินเทอร์เน็ตอินฟอร์เมชันเซอร์วิซซิส หรือ ไอไอเอส () ชื่อเดิมว่า อินเทอร์เน็ตอินฟอร์เมชันเซิร์ฟเวอร์ () เป็นโปรแกรมให้บริการเว็บเซิร์ฟเวอร์ เพื่อให้สามารถ Run โปรแกรมผ่านเครื่องของเราได้ ซึ่งส่วนใหญ่จะหาได้จากแผ่น Windows อยู่แล้ว โดยขั้นตอนการติดตั้งให้ไปที่ หาก Windows เป็น Windows NT 4.0 อาจจะไม่สามารถรันได้ให้ Download ASP.exe ในการลง ซึ่งแถมมากับ Windows 98</doc>
