<doc id="98826" url="https://th.wikipedia.org/wiki?curid=98826" title="วินโดวส์เซิร์ฟเวอร์ 2008">วินโดวส์เซิร์ฟเวอร์ 2008 วินโดวส์เซิร์ฟเวอร์ 2008 (Windows Server 2008) เป็นชื่อของระบบปฏิบัติการเซิร์ฟเวอร์รุ่นต่อไปจากไมโครซอฟท์ เป็นรุ่นถัดจากวินโดวส์เซิร์ฟเวอร์ 2003 เดิมทีมีรหัสในการพัฒนาว่า Windows Server "Longhorn" วินโดวส์เซิร์ฟเวอร์ 2008 เป็นระบบปฏิบัติการเซิร์ฟเวอร์ที่นำคุณสมบัติหลายอย่างมาจากวินโดวส์วิสตา ความสัมพันธ์ลักษณะเดียวกับวินโดวส์เซิร์ฟเวอร์ 2003 และวินโดวส์เอกซ์พี ปัจจุบันวินโดวส์เซิร์ฟเวอร์ 2008 ออกรุ่นทดสอบมาแล้ว 3 รุ่น ได้แก่ เบต้า 1 (27 กรกฎาคม ค.ศ. 2005) เบต้า 2 (23 พฤษภาคม ค.ศ. 2006) และเบต้า 3 (25 เมษายน ค.ศ. 2007) รุ่นสุดท้ายมีกำหนดออกในครึ่งหลังของปี 2007รุ่น รุ่น. วินโดวส์เซิร์ฟเวอร์ 2008 จะมีทั้งที่เป็นรุ่นสำหรับ 32 บิต (x86) และ 64 บิต (x86-64) เช่นเดียวกับวินโดวส์เซิร์ฟเวอร์ 2003 และยังแบ่งเป็นรุ่นสำหรับการใช้งาน ดังนี้- Web Edition - Standard Edition - Enterprise Edition - Datacenter Edition</doc>
