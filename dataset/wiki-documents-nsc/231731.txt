<doc id="231731" url="https://th.wikipedia.org/wiki?curid=231731" title="ร็อคแมน X (ตัวละคร)">ร็อคแมน X (ตัวละคร) ร็อคแมน X (; ) หรือถูกเรียกย่อๆว่า X เป็นชื่อตัวละครในเกมซีรีส์ร็อคแมน X ซีรีส์ประวัติ ประวัติ. เอ็กซ์ถูกสร้างขึ้นมาโดยผู้สร้างคือโทมัส ไรท์ ที่สร้างขึ้นในบั้นปลายชีวิตของเขาโดยใส่ข้อมูลความสามารถหลักๆของร็อคแมนเป็นอิเรกูลาร์ฮันเตอร์ สำหรับในตอนนั้น X นับได้ว่าเป็นหุ่นยนต์รูปแบบใหม่ซึ่งต่างจากหุ่นยนต์ทั่วๆไปด้วยระบบความสามารถในการตัดสินใจด้วยตนเองอันยอดเยี่ยม แม้ว่าการสร้างเอ็กซ์จะเสร็จสมบูรณ์แล้ว แต่ถึงกระนั้น ไลท์ก็ยังคงคาดการณ์เอาไว้ว่า มีความเป็นไปได้ที่เอ็กซ์จะหันกลับมาทำการต่อต้านและทำร้ายมนุษย์ในภายหลัง ด้วยความกลัว ไลท์จึงได้ผนึกเอ็กซ์เอาไว้ในแคปซูล และทำการวิเคราะห์ผลต่อไปอีก 30 ปีเพื่อที่จะมั่นใจได้ว่าเอ็กซ์จะไม่เลือกหันมาทำร้ายมนุษย์และดำเนินชีวิตไปในทางที่เลวร้าย ในระหว่างนั้น ไม่มีใครรู้ว่าเกิดอะไรขึ้นกับดร.ไลท์ ในตลอดเวลาประมาณ 30 ปี และแล้ว แคปซูลของเอ็กซ์ก็ถูกเปิดขึ้นโดยนักโบราณคดีที่มีชื่อว่า "ดร. เคน" หลังจากการสร้าง X ขึ้นมาแล้วนับร้อยปีบทบาทในแต่ละซีรีส์The Day Of Σ บทบาทในแต่ละซีรีส์. The Day Of Σ. ใน ค.ศ. 21XX ได้มีคดีอีเรกูลาร์ก่ออาลาวาดขึ้นในเมืองซึ่งมีซิกม่าเป็นหัวหน้าหน่วยที่ 17 โดย ซีโร่ เป็นอีเร็กกูล่าฮันเตอร์ระดับ A และ เอ็กซ์เป็นอิเรกูลาร์ฮันเตอร์ระดับ B ซึ่งเป็นรุ่นน้องซึ่งเหตุการณ์ขับขันซึ่ง X ไม่ได้ทำอะไรไม่มากจนมีอีกคดีซึ่งลึกลับมีเรฟลิบรอยจำนวนมากถูกฆ่าอย่างปริศนา ซึ่ง X ไม่สงสัยในตัวซิกม่าแต่สุดท้ายซิกม่าได้หักหลัง X แต่ไม่ถึงกับฆ่า แต่ซิกม่าได้ประกาศเป็นศัตรูกับมนุษย์จน X เขาแลกกับดาบเซเบอร์ของซิกม่า จน X จำความเมื่อ 100 ปีก่อนแต่เขาต้องเสียความทรงจำไปเมื่อพลังงานตัวเองหมดไปจน ซีโร่ อยู่ในเหตุการณ์ช่วย X ไว้ได้ร็อคแมน X ร็อคแมน X. เป็นซีรีส์แรกที่ออกมาซึ่งภาคนี้เกิดใน ค.ศ. 21XX "เอ็กซ์"ได้ออกตามล่าซิกม่าซึ่งเป็นกบฏของมนุษย์ ได้ไปต่อสู้ทั่วทวีปกับอดีตอีเร็กกูล่าฮันเตอร์ระดับ A จนผ่านไปทุกที่และได้พลังมาจากศัตรูความสามารถนั้นนอกเหนือจากศัตรูและเป็นจุดอ่อนในกับศัตรู และ X ได้เจอกับดร.ไลท์ในแคปซูลแต่ไม่รู้จักจนได้เกราะอาร์เมอร์มาทั้งหมด 3 ส่วน จนสุดท้ายได้ข่าวว่าซีโร่ไปฐานของซิกม่าจนถูกวาว่าจับตัวไป X ช่วยอะไรไม่ได้จนซีโร่ต้องช่วย X เองจนตายไปและได้เกราะมาคือเกราะแขนชนะวาว่าได้และพอต่อสู้กับซิกม่าพลังไม่เทียบกับเขาและได้บอกว่ามนุษย์เป็นจุดด่อยกว่าหุ่นยนต์ถึงกับซิกม่ามีความทะเยอนทะยานกำจัดมนุษย์จน X ไม่เชื่อถือในตัวซิกม่าและได้สู้กับซิกม่าจนได้สู้อีกครั้งในร่างใหม่และได้ชนะและทำลายฐาน และสาบานว่าจะไม่ต่อสู้อีกร็อคแมน X2 ร็อคแมน X2. ต่อมาผ่านไปหลายเดือน X ผู้สาบานว่าจะไม่ทำสงครามอีก จึงได้ผนึกบัสเตอร์ไว้ แต่ทว่าคำสาบานนั้นไม่มีผล เพราะซิกม่าได้ฟื้นคืนชีพและได้ผลิตชิพซิกม่าและแพร่กระจายออกไป X จึงตัดสินใจปลดผนึกออกและได้ทำการต่อสู้อีกครั้งจนได้เจอกับ Dr.เคน ผู้ที่เคยปลุก X ให้ตื่นขึ้นมาได้คอยช่วยเหลือ X และ Dr.เคน ได้ไปพบกับ ซีโร่ ซึ่ง Dr.เคนเจอที่ญี่ปุ่นนั้นเองเขากำลังทำการซ่อมแซม แต่ทว่า 3 ลูกน้อง ของซิกม่า อาจิน,ซาเกลและไวโอเล็ต ได้ลักพาตัวซีโร่และดัดแปลงร่างกายใหม่จนหลังจากตอนที่สู้กับ Morph H Moth ได้เจอกับซีโร่ร่างสีดำและใช้ดาบฟัน X จนตกลงไปและได้เกราะอกมาแต่ทว่าไม่สามารถใช้บัสเตอร์ได้แต่ทว่า ในด่านของ Wheel Gator X ได้รับปรอกแขนและได้ใช้จนได้ท่าใหม่ว่า ดับเบอร์บัสเตอร์ เอาชนะมาได้และก็ไปฐานของซิกม่า ได้สู้กับ ไวโอเล็ต(ตัวแรก),ซาเกลกับอาจินจนได้รับชัยชนะมา (ในมังงะ:สู้กันที่เดียว 3 ตัว) แต่ต้องสู้กับซีโร่ที่ยังมีชิพซิกม่าอยู่ในหมวกแต่ก็เอาชนะมาได้และตกลงไปใต้ดินเจอกับซิกม่าแต่ X ไม่สามารถชนะพลังของซิกม่าได้แต่ซีโร่มาช่วยเอาไว้ แต่นี้คือร่างหลอกที่ X กับ ซีโร่ สู้กับ ร่างจริงคือ ดามี่ จน X กับ ซีโร่ ไม่มีพลังเหลือ แต่ทว่าวิญญานของ ดร.ไลท์ ช่วยเอาไว้และฟื้นพลังจนชนะซิกม่าได้และก็หนีออกมาจากฐานได้ในที่สุดร็อคแมน X3 ร็อคแมน X3. หลังจากที่ซิกม่าได้ถูกฆ่าไปแล้ว 2 ครั้งและพอผ่านไปได้ไม่นาน 2 ปีเหล่าอิเรกุราห์ฮันเตอร์ต้องประสบเคราะห์ร้ายด้วยไวรัสซิกม่าที่ยังคงรอยนวนอยู่ในโลก และไวรัสซึกม่าซึงได้ควบคุม ดร.ดอร์ปเปอร์ ที่ถูก ดร.เชน์ ตั้งองกรให้ และพอต่อมาไวรัสซิกม่าเริ่มมีความร้ายกาจ เอ็กซ์นั่นจึงไม่รีบอยู่เฉยจึงพยายามต่อต้านเหล่า ทหารของดอร์ปเปอร์ ที่กำลังโจมตีอิเรกุราห์ฮันเตอร์ และซีโร่ กับต้องเจอกับวาว่าอีกครั้งร็อคแมน X4 ร็อคแมน X4. หลังจากที่เอ็กซ์ได้หายตัวไป จากการต่อสู้กับซิกม่าในครั้งที่ 3 เอ็กซ์ก็ได้กลับมา แล้วในคืนนั้นก็พวก ดร.เคน และ ซีโร่ได้ฉลองร่วมกันให้กับ เอ็กซ์ เอ็กซ์ และ ซีโร่ จึงออกไปกำจัดหุ่นตัวนั้น หลังจากที่เอ็กซ์กำจัด Sky Dragoon ได้แล้ว คาร์เนล [หุ่นยนต์พันเอก ของ เรปลิฟรอส] ได้มารับตัวน้องสาวคืน พวกของซีโร่ได้ทำการสืบค้นซากศพของหุ่นที่อาละวาดในเมืองก็พบ ชิ้นส่วนของ เรปลิฟรอส ซีโร่ถึงถาม โคโลเนล แต่โคโลเนลปฏิเสธ แล้วไปหนีหายไป และแล้ว เวลา 13.00น. "นายพล เจเนอรัล" ก็ได้ประกาศยุติตัดขาด ความสัมพันธุ์ของมนุษย์ เอ็กซ์ก็จะไปห้ามปราม แต่ก็พบซีโร่ ซึ่งกับกำลังหาตัว โคโลเนลอยู่ ซีโร่ได้คลั่ง อาละวาดฟันพวกอิเล็กกูล่าห์ ฮันเตอร์ ด้วย เอ็กซ์จึงมาห้าม แต่ซีโร่ไม่ยอมฟัง แล้วได้ฟันตัว เอ็กซ์ แล้วก็จากหายไปร็อคแมน X5ร็อคแมน X5. - เอ็กซ์ ซึ้งเขายังไม่รู้เรื่องความลับของ ซีโร่ และที่ผ่าน ๆ มาเอ็กซ์ได้รวมมือกับ ซีโร่มาตลอด แต่ภาคนี้ ซีโร่มีท่าทางแปลก ซึ่งมีอาการติดไวรัส ตั้งแต่ชนะซิกม่าที่ยานอวกาศแห่งหนึ่ง แล้วก็ได้กลับไปทำงานที่ หน่วยสำนักงานใหญ่ อีเล็กกูร่าฮันเตอร์ และมีเพื่อนมากมาย แต่พอเหตุการณ์เริ่มไปกันใหญ่ ซีโร่ ได้ติดไวรัสของ ดร.ไวลี่ ที่เคยอยู่ในร่างกายมาก่อน เอ็กซ์อาสาไปนำตัว ซีโร่ กลับมา แต่กลับต้องสู้กัน ในที่สุดซีโร่หายไปพร้อมกับซิกม่า สุดท้าย ซีโร่เหลือตัวแค่ครึ่งตัวระหว่างอก เอ็กซ์ เข้าไปหา แต่ไม่รู้ตัวว่า ซิกม่าโจมตีใส่ เลเซอร์ที่ซิกม่ายิง ทำให้ทั้ง 2 ทะลุทั้ง 2 ข้าง และเอ็กซ์ ก็หมดสติหลังจากที่ ซีโร่ พูดเรื่องราวของตัวเขาจบ และร่างวิณญานของ ดร.ไลย์ ก็เข้ามาทำการพื้นฟู (อีกนัยคือในเกมถ้าซีโร่ติดไวรัสก็จะไม่ได้เลยตัวซีโร่และเอ็กซ์ก็โดนระเบิดของซิกม่าทำลายร่างของเขา) และเอ็กซ์หลังจากฟื้นขึ้นจาก 3 ปีต่อมา ก็ได้เจอชื้นส่วนของซีโร่แค่ดาบเซเบอร์สุดรักของซีโร่มาใช้งานแทนร็อคแมน X6 ร็อคแมน X6. หลังจากเซโร่ได้ถูกทำลายไปเมื่อ 3 สัปดาห์ก่อนเอ็กซ์จึงต้องจับดาบของเซโร่แล้วพูดว่า"ฉันจะจับดาบแทนนายเอง"และการที่ต้องต่อสู้กับความฝันร้าย แต่เอ็กซ์นั่นยังคงเป็นหน่วยอิเรกุราห์ฮันเตอร์ หลังจากนั้นไม่นานเขาก็ได้พบเซโร่เพื่อนของอีกครั้งแล้วจับมือสัญญาว่าจะต่อสู้ไปด้วยกันเสมอไปร็อคแมน X7 ร็อคแมน X7. หลังจากผ่านไปเป็นเวลาหลายปีโลกที่ไร้ซิกม่ามาหลายๆปี และหลังจากนั้นก็ได้มีองค์กรหนึ่งที่ชื่อว่า Red Alert(ไม่ใช้ Command and Conquer)และหลังจากนั้นก็ได้มีเรพลอยด์หนุ่มคนนึงที่มีนามว่า Axl(เอ็กเซล)เขาได้ถูกรับเลี้ยงโดยพ่อบุณธรรมที่ชื่อว่า Red(เรด)เขาได้รับการเลี้ยงดูจาก Red มาเป็นอย่างดีแต่พอไปได้ไม่นาน Axl ก็เริ่มเบื่อกับการฆ่าหุ่นยนต์อันโหดเหี่ยมของ Red Alert เขาจึงคิดที่จะหนีออกมาแต่ด้วยความสามารถในการคัดลอด DNA ของเรพลอยด์ได้เขาจึงพยายามเอาตัวรอดขาก Red ให้ได้และหลังจากนั้น Zero กำลังกลับมาดูที่ๆเขาเคยมาเมื่อตอนนี้ที่เขาเห็นเอ็กซ์ครั้งแรก หรือ High Way เขาก็ได้พบกับเด็กหนุ่มที่ชื่อว่า Axl และจากนั้นเขาก็จับตัว Axl ไปยังอิเรกุราห์ฮันเตอร์และหลังจากนั้น Axl ก็ได้พบกับ X และ X ก็ไม่ค่อยที่จะชอบ Axl มากเสียเท่าไหร่จึงได้ออกไปจากอิเรกุราห์ฮันเตอร์และปล่อยให้ Axl และ Zero ทำหน้าที่แทน หลังจากนั้น Zero และ Axl ก็ได้ X คืนมาอีกครั้งหลังจากที่ X ยอมรับ Axl แล้วร็อคแมน X8สงครามเอลฟ์ร็อคแมน ซีโร่ร็อคแมน ZXร็อคแมน ZXAตอนถูกสร้างร็อคแมน ZXA. ตอนถูกสร้าง. - X ถูกสร้างจาก ร็อคแมน ซึ่งถูกปรับปรุงใหม่โดย ดร.ไลท์ (ร็อคแมน & ฟอร์เต้) - X เป็นหุ่นยนต์ตัวใหม่ของ ดร.ไลท์ เพราะร็อคแมนตัวเก่าถูกดร.ไวลลี่ดัดแปลงเป็นควินท์ (ร็อคแมนเวิลด์ 5)</doc>
