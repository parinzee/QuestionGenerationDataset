<doc id="13532" url="https://th.wikipedia.org/wiki?curid=13532" title="ภาษาเบสิก">ภาษาเบสิก ภาษาเบสิก (BASIC programming language) เป็นภาษาโปรแกรมที่ออกแบบมาให้ใช้งานได้ง่าย และยังได้รับความนิยมมาจนถึงทุกวันนี้ เบสิกออกแบบมาให้ใช้กับคอมพิวเตอร์ตามบ้าน ชื่อภาษาเบสิก หรือ BASIC ย่อมาจาก Beginner's All-purpose Symbolic Instruction Code ต้องเขียนด้วยตัวพิมพ์ใหญ่เสมอ บริษัทไมโครซอฟท์ได้นำภาษาเบสิกมาปรับปรุงให้ทันสมัย และพัฒนาเครื่องมือพัฒนาโปรแกรม Visual Basic ทำให้เบสิกได้รับความนิยมในการพัฒนาโปรแกรมยุคใหม่ รุ่นล่าสุดของวิชวลเบสิกเรียกว่า VB.NETประวัติของภาษา Basic ประวัติของภาษา Basic. ภาษา Basic ถูกคิดค้นเมื่อปี 1963 โดย นาย John และ นาย Thomas Kurtz ณ Dartmouth College และบรรดานักเรียนนักศึกษาในความดูแลของพวกเขา ซึ่งหลายปีต่อมา ภาษา Basic ฉบับนี้ได้ชื่อเรียกอีกอย่างหนึ่งว่า Dartmouth BASIC. ท่านสามารถดูการพัฒนา จาก Basic สู่ VB.NETภาษา Basic พื้นฐาน ภาษา Basic พื้นฐาน. ภาษา Basic นั้นเป็นโปรแกรมที่เป็นพื้นฐานสำหรับหัดเขียนโปรแกรมก็ว่าได้ เนื่องจากเป็นภาษาที่เป็นคำพูดของคนเราทั่วไป เช่น ดังนี้ การใช้คำสั่ง Print เครื่องจะประมวลผลและแสดงผลออกมาเป็นข้อความ codice_1 ออกมาทางหน้าจอประเภทของค่าในภาษา Basic ประเภทของค่าในภาษา Basic. ! : Single-precision1. : Double-precision $ : String % : Integer & : Longเครื่องหมายทางคณิตศาสตร์เครื่องหมายทางคณิตศาสตร์. - : เครื่องหมายคูณ - : เครื่องหมายลบ</doc>
