<doc id="150912" url="https://th.wikipedia.org/wiki?curid=150912" title="คริปทูล">คริปทูล คริปทูล (CrypTool) คือ freeware program ที่สามารถใช้และวิเคราะห์ cryptographic mechanism โดย cryptool นั้นสามารถใช้เข้ารหัสข้อมูลได้ทั้งในระบบ classic อย่าง Caesar cipher หรือแบบ modern อย่าง RSA เป็นต้น นอกจากนี้ยังสามารถทดลองทำ digital signature ได้จากหลายๆ algorithm เช่น MD5 โดยทุกๆรูปแบบของการทำงานนั้นสามารถแยกแสดงขั้นตอนต่างๆให้ดูได้อย่างละเอียดเป็นแบบ animationที่คล้ายกับ flash game เหมาะกับการนำไปใช้ในการศึกษาระบบการเข้ารหัสได้อย่างดี จุดเด่นอีกอย่างของ programนี้คือมี online help ที่อธิบายระบบการเข้ารหัสได้อย่างละเอียด นอกจากนี้ cryptool ยังสามารถที่จะใช้ในการวิเคราะห์ cipher บางอย่างได้ด้วยเช่น sequence analysis เป็นต้น Cryptool เป็น open source software ซึ่งทำงานอยู่บน window โดยพัฒนาด้วยภาษา c++ บน platform window โดยในตอนแรกนั้นพัฒนาโดย German companies and universities และมีจุดประสงค์หลักคือการแสดงให้คนเห็นถึงความสำคัญของการเข้ารหัสและ concept โดยเป็น e-learning ประเภทหนึ่ง ในขณะนี้มี project ที่จะ port ลงไปใน linux edition และ เขียนใหม่ลงใน c# โดยใช้เทคโนโลยี WPF (window presentation fundamental) ที่ใช้แสดงผลในด้านกราฟิกได้อย่างดี CrypTool ได้รับรางวัลในฐานะของ educational software หลากหลายรางวัล เช่น (TeleTrusT Special Award, EISA 2004, IT Security Award NRW 2004). ในตอนนี้ได้ถูกพัฒนาขึ้นจากหลายองค์กร แต่ยังคงไว้ซึ่งจุดประสงค์เดิม คือเพื่อการศึกษาหลักการเข้ารหัสและอักกอริทึมต่างๆ และส่วนมากถูกใช้ประกอบการสอนในโรงเรียนและสถาบันการศึกษาต่างๆSee alsoSee also. - Asymmetric key algorithm - CryptosystemExternal linksExternal links. - www.cryptool.com The official site - www.cryptoolinux.net CrypTooLinux a clone of CrypTool on Linux</doc>
