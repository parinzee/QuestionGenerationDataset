<doc id="95939" url="https://th.wikipedia.org/wiki?curid=95939" title="อีลีตบีตเอเจินส์">อีลีตบีตเอเจินส์ อีลีตบีตเอเจินส์ () หรือชื่อย่อ EBA (อีบีเอ) เป็นเกมดนตรีของเครื่องนินเทนโด ดีเอส ซึ่งเกมนี้คือเกมโอ๊ทส์! ทาทาคาเอะ! โอเอนดันที่ถูกสร้างขึ้นมาใหม่โดยวางจำหน่ายในประเทศสหรัฐอเมริกา, ยุโรปและออสเตรเลีย ซึ่งได้มีการเปลี่ยนแปลงเนื้อหาทั้งหมดของเกมให้สอดคล้องกับวัฒนธรรมตะวันตก ตัวเกมผลิตโดยนินเทนโดและ iNiS ภายหลังจากที่เกมโอ๊ทส์! ทาทาคาเอะ! โอเอนดันได้รับความนิยมอย่างสูงไปทั่วโลก รองประธานของบริษัท iNiS จึงได้ตัดสินใจนำเกมนี้มาวางจำหน่ายในประเทศฝั่งตะวันตก แต่ทว่าเนื่องด้วยเกมโอ๊ทส์! ทาทาคาเอะ! โอเอนดันเป็นเกมที่อิงกับวัฒนธรรมญี่ปุ่นมาก จึงตัดสินใจเปลี่ยนเนื้อหาของเกมให้สอดคล้องกับฝั่งตะวันตก โดยเปลี่ยนจากทีมเชียร์นักเรียนมัธยมปลายของญี่ปุ่นให้กลายมาเป็นกลุ่มสมาชิกขององค์กรดนตรี และเปลี่ยนจากการเต้นเชียร์มาเป็นการร้องและเต้นให้เข้าจังหวะแทน เนื้อเรื่องของอีลีตบีตเอเจินส์จะเป็นเรื่องขององค์กรรัฐบาลที่มีชื่อว่า อีลีตบีตเอเจินส์ (ตามชื่อเกม) โดยเมื่อมีผู้คนที่ตกอยู่ในความสิ้นหวัง พวกเขาจะตะโกนมาว่า Helppppppppp!!!! (คล้ายๆกับคำว่า โอเอนดัน! ในเกมก่อนหน้า) ซึ่งหลังจากที่ผู้การคาห์น หัวหน้าของ EBA พบเป้าหมายพวกเขาจะทำการส่งตัวละครที่ผู้เล่นเลือก โดยใช้คำว่า Agents are...Go! ซึ่งจากนั้นตัวละครก็จะปรากฏต่อหน้าคนที่ขอความช่วยเหลือและเริ่มร้องเพลงเพื่อปลุกใจพวกเขาระบบของเกม ระบบของเกม. ระบบโดยรวมของเกมจะคล้ายๆกับเกมโอ๊ทส์! ทาทาคาเอะ! โอเอนดัน โดยก่อนจะเริ่มเข้าแต่ละด่าน ผู้เล่นจะอยู่ตรงฉากแผนที่ ซึ่งจะมีจุดต่างๆในแผนที่เพื่อแสดงว่ามีด่านอะไรและเพลงอะไรให้เล่นบ้าง และเมื่อเข้าไปในด่านนั้นๆ เพลงก็จะถูกบรรเลงขึ้นมาหน้าที่ของผู้เล่นคือกดปุ่มต่างๆให้เข้ากับจังหวะในขณะที่วงกลมรอบนอกจะถูกบีบลงเข้ามาในปุ่ม ซึ่งแต่ละปุ่มจะมีตัวเลขกำกับไว้ซึ่งก็คือลำดับการกด โดยปุ่มจะมีสีต่างกันออกไป ซึ่งผู้เล่นต้องกดปุ่มตามลำดับของสีนั้นๆถึงจะข้ามไปสีต่อไปได้- ปุ่ม - เป็นลักษณะปุ่นกดครั้งเดียว เมื่อกดแล้วจะหายไปโดยผู้เล่นจะได้คะแนนแปรตามจังหวะที่กด (ช้าหรือเร็วเกินไป 50 คะแนน, ช้าหรือเร็วกว่าปกตินิดนึง 100 คะแนน, พอดี 300 คะแนน) - ลาก - ตัวลากนั้นจะมีหัวและหางเป็นปุ่มเหมือนปกติ โดยในแถบลากนั้นจะมีจุดเล็กๆอยู่ข้างใน เมื่อผู้เล่นเริ่มลากจะมีลูกบอลวิ่งไปตามแถบลาก โดยเมื่อโดนจุดเล็กๆจะได้ 10 คะแนน และเมื่อถึงจุดปลายถ้าลากครบโดนทุกจุดในแถบลากจะได้ 300 คะแนน แต่ถ้าโดนทุกปุ่มครบแต่ปล่อยก่อนที่จะถึงปลายหรือพลาดในจุดเล็กๆบางจุดแต่ปล่อยที่ปลายพอดีจะได้ 100 คะแนน และถ้าพลาดจุดเล็กๆมากเกินไปจะได้ 50 คะแนน โดยปกติถ้าผู้เล่นลากไม่โดนจุดเล็กๆจะถือว่าผิดคอมโบจะลดลงหมด แต่จะไม่นับว่าผิด (แต่จะไม่ได้ Perfect) - หมุน - จะเป็นลักษณ์วงกลม เมื่อผู้เล่นเริ่มหมุน แถบที่อยู่ข้างๆจะเริ่มเพิ่มปริมาณขึ้นจนเต็ม และเมื่อเต็มแล้วยังหมุนต่อไปจะได้คะแนนเพิ่มรอบละ 1000 คะแนน ถ้าหมุนครบรอบพอดีเมื่อเวลาหมดจะได้ 300 คะแนน แต่ถ้าหมุนได้เกือบๆ จะได้ 50 คะแนน ในระหว่างเล่นผู้เล่นจะสะสม คอมโบ ซึ่งคอมโบนั้นจะทำหน้าที่ทวีคณูคะแนนไปเรื่อยๆ นอกจากนั้นเมื่อผู้เล่นกดครบจำนวนของสีนั้นๆ ถ้าในระหว่างกดสีนั้นผู้เล่นได้ 300 คะแนนตลอดผู้เล่นจะได้ Elite Beat ซึ่งคะแนนจะทวีคูณขึ้นไปอีก ถ้าได้ 100 เป็นบางตัวจะได้ Beat และถ้าได้ 50 คะแนนในสีนั้นๆจะไม่ได้รับทั้ง Elite Beat และ Beat ในระหว่างเล่นเกมผู้เล่นต้องรักษาเกจกำลังใจด้านบนไม่ให้ลดลงจนหมด เมื่อผู้เล่นเล่นด่านนั้นๆไปซักพักหนึ่งเกมจะตัดเข้าสู่เนื้อเรื่องของด่านนั้นๆ ถ้าเกจกำลังใจของผู้เล่นเกินครึ่ง ตัวละครจะกระทำการสำเร็จ แต่เกจลดลงมากเกินไป ตัวละครจะกระทำการนั้นๆล้มเหลว ซ่งเกจกำลังใจโดยปกติจะลดลงอยู่ตลอดเวลา (โดยความเร็วขึ้นอยู่กับระดับความยากของเกมและด่าน) ถ้ากดปุ่มผิดจังหวะจะทำให้เสียเกจมากขึ้น แต่ในทางตรงกันข้ามถ้ากดจังหวะถูกต้องก็จะได้เกจกำลังใจเพิ่มขึ้น นอกเหนือจากระบบทั่วๆไปในโอ๊ทส์! ทาทาคาเอะ! โอเอนดัน เกมนี้ยังได้เพิ่มระบบต่างๆขึ้นมาอีก- ข้ามจุดเริ่มเพลงไปช่วงนับ 3, 2, 1 - เมื่อแพ้สามารถดูช่วงที่แพ้ได้ - เซฟรีเพลย์, ดูรีเพลย์และแจกจ่ายรีเพลย์ - Ghost Mode เล่นแบบผู้เล่นหลายคนแต่จะให้เอารีเพลย์มาเป็นผู้เล่นอีกคนแทน - เพลงลับซึ่งจะเล่นได้เมื่อสะสมคะแนนถึงปริมาณหนึ่ง - ดาวน์โหลดตัวอย่างให้กับเครื่องที่ไม่มีตลับ - ภาพจบ 2 จอเมื่อสามารถผ่านเนื้อเรื่องทั้งหมดในเพลงนั้นๆได้ - ตอนจบที่ต่างออกไป ขึ้นอยู่กับว่าสามารถผ่านเนื้อเรื่องทั้งหมดในเพลงนั้นๆได้หรือไม่ - แผนที่ 3 มิติ - สามารถดูภาพคั่นฉากในเนื้อเรื่องได้ตัวละครในเกม ตัวละครในเกม. เช่นเดียวกับโอ๊ทส์! ทาทาคาเอะ! โอเอนดัน ผู้เล่นจะได้เลือกตัวละครจากคนของ Elite Beat Agent ซึ่งตัวละครทั้ง 4 จะแทนตามระดับความยาก (ยากจะเล่นได้ต้องจบง่ายและธรรมดาก่อน ยากมากจะได้เมื่อรับระดับยาก)- Agent BA-5 สปิน (Spin) - ระดับ "ง่าย" มีจุดเด่นที่ผิวสีและสวมหูฟังตลอดเวลา - Agent BA-2 เจ (J) - ระดับ "ธรรมดา" มีจุดเด่นที่ทรงผมสีเหลืองซึ่งม้วนเป็นเกลียว เขาเป็นตัวละครที่ปรากฏบนกล่องเกมและหน้าเมนูเกมหลัก - Agent BA-1 ชิฟเทน (Chieftain) - ระดับ "ยาก" มีจุดเด่นที่ใบหน้าที่ไม่ยิ้มกับหมวกและเครา - Agent BA-3 มอริส (Moris) และ Agent BA-4 เดเรค (Derek) เป็นตัวประกอบซึ่งจะอยู่ข้างๆตัวละครชายทั้ง 3 มอริสมีจุดเด่นที่ผิวสี, เครา, แว่นตาดำและหมวก ในขณะที่เดเรคจะมีจุดเด่นที่ผมทรงแอฟโฟรสีแดง - Elite Beat Divas - ระดับ "ยากมาก" สมาชิกหญิงของทีม ประกอบด้วย Agent BD-1 สตาร์ (Starr) ผู้นำ, Agent BD - 2 ฟ็อกซ์ (Foxx) ผมยาวสีขาว, Agent BD-3 มิสสี้ (Missy) ผมเปียสีเหลืองใส่แว่น - ผู้การคาห์น (Commander Kahn) สามารถเลือกเล่นได้โดยจบครบทุกเพลงในทุกระดับความยาว เขาจะปรากฏตัวมาแทนที่สตาร์เมื่อผู้เล่นเลือกทีม Elite Beat Divasเพลงที่ใช้ในเกม เพลงที่ใช้ในเกม. เช่นเดียวกับเกมโอ๊ทส์! ทาทาคาเอะ! โอเอนดัน Elite Beat Agent ได้มีการเปลี่ยนคนร้องในแต่ละเพลง รวมทั้งตัดหรือเพิ่มเนื้อเร้องในบางจุด ซึ่งเพลงที่ใช้จะเป็นเพลงทั้งป็อบและร็อกโดยแต่ละด่านก็จะมีเรื่องราวที่สอดคล้องกับวัฒนธรรมและประเพณีของฝั่งตะวันตกเช่นกัน สามเพลงสุดท้าย (ABC, Believe, Survivor) จะไม่อ้างอิงในระดับความยากใด ซึ่งผู้เล่นจะได้เล่นเมื่อทำคะแนนสะสมได้จำนวนหนึ่ง (แต่ผู้เล่นต้องผ่านด่านบางด่านก่อนถึงจะเข้าเล่นสามเพลงนั้นได้)1.
