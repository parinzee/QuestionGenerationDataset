<doc id="138789" url="https://th.wikipedia.org/wiki?curid=138789" title="ร็อคแมน X ซีรีส์">ร็อคแมน X ซีรีส์ ร็อคแมน X () หรือ เมกาแมน X (Mega Man X) เป็นซีรีส์ภาคต่อชุดที่สองของเกมชุดร็อคแมน ซึ่งได้รับสัมปทานการสร้างภาคต่อจากแคปคอม ออกวางตลาดครั้งแรกที่ญี่ปุ่นในวันที่ 19 ธันวาคม พ.ศ. 2536 และออกวางตลาดครั้งแรกที่อเมริกาในช่วงเดือนกุมภาพันธ์ พ.ศ. 2537 โดยดำเนินเนื้อเรื่องต่อเนื่องจาก ร็อคแมนซีรีส์ โดยดำเนินในช่วงเวลา 100 ปีเนื้อเรื่อง เนื้อเรื่อง. ณ เวลาในปี 21xx มนุษย์ในโลกได้อยู่ร่วมกับหุ่นยนต์และมีมนุษย์กึ่งหุ่นยนต์เรียกว่าเรพริรอยด์ (Repliroid, レプリロイド, Reploid ในภาษาอังกฤษ) แต่ว่าได้เกิดการก่อการร้ายภายใต้ของซิกม่าอดีตอิเรกูลาร์ฮันเตอร์ได้นำกลุ่มอิเรกูลาร์ (Maverick ในภาษาอังกฤษ) มาทำลายโลก ในขณะเดียวกันได้มีอิเรกูลาร์ฮันเตอร์ที่มีชื่อว่า เอ็กซ์ ได้ต่อกรกับอีเรกูลาร์เพื่อที่จะปกป้องโลกและเรพริรอยด์ไปด้วยซีรีส์เกมซีรีส์หลักซีรีส์เกม. ซีรีส์หลัก. - ร็อคแมน X (SFC) - ร็อคแมน X2 (SFC) - ร็อคแมน X3 (SFC, Windows, PS และ SS) - ร็อคแมน X4 (PS และ SS, Windows) - ร็อคแมน X5 (PS และ Windows) - ร็อคแมน X6 (PS และ Windows) - ร็อคแมน X7 (PS2 และ Windows) - ร็อคแมน X8 (PS2 และ Windows)ซีรีส์เสริมซีรีส์เสริม. - ร็อคแมน X ไซเบอร์มิชชั่น (GBC) - ร็อคแมน X โซลอีเรเซอร์ (GBC) - ร็อคแมน X คอมมานด์มิชชั่น (PS2 และ GC) - อิเรกูลาร์ฮันเตอร์ X (PSP)ตัวละครเรพลิรอยด์ตัวละคร. เรพลิรอยด์. - เอ็กซ์ (エックス, X) จากนั้นพอผ่านไปเกือบศตวรรษแคปซูลของ X ก็ถูกเปิดขึ้นโดยนักโบราณคดีที่มีชื่อว่า "ดร. เคน" (Dr.Cain) ทำให้ X ตื่นขึ้นมา เคนและ X ได้ช่วยกันสร้างเหล่าหุ่นยนต์โดยดัดแปลงมาจากความสามารถของ X ซึ่งเป็นต้นแบบและได้ถูกเรียกว่า "Replicate Android" หรือ "เรพริรอยด์" (Repliroid, レプリロイド, Reploid ในภาษาอังกฤษ) เป็นหุ่นยนต์ที่มีความคิดเหมือนมนุษย์ พอเรพริรอยด์อาลวาดทำร้ายมนุษย์จนถูกเรียกว่าอิเรกูลาร์ X จึงเข้าร่วมกับอิเรกูลาร์ฮันเตอร์ จนเป็นระดับ B เมื่อซิกม่าก่อสงคราม X จึงต้องหยุดยั้งความทะเยอทะยานหลายครั้งตลอด พอจบ X1 ทำให้ X เป็นผู้นำอิเรกูล่าฮันเตอร์หน่วย 17 ในX2 ต้องกลับมาสู้กับซิกม่ากับรวบรวมชิ้นส่วนของซีโร่ ในX3 ร่วมมือกับซ๊โร่หาทางยับยั้งไวรัสซิกม่าจากด็อปเลอร์ ในX4 Xกับซ๊โร่แยกกัน ยับยั้งเรปพิฟอร์ซ ในX5 ร่วมมือกับซีโร่รวบรวมชิ้นส่วนสร้างอาวุธทำลายไวรัสซิกม่าจากอวกาศ ในX6 X พก Z-Saber ของซีโร่เป็นอาวุธประจำตัวกับของดูต่างหน้าซีโร่ เพื่อกำจัด Nightmare ไวรัสจากเกม แล้วซีโร่ตัวจริงมาช่วย ในX7 X ลาออกจากฮันเตอร์มาเป็นคนสั่งการกับสนับสนุน เพราะคิดว่าจะเป็นทางที่ดี แต่พอแอ็กเซลมาก็ต้องทำให้ X กลับมาเป็นฮันเตร์อีกครั้ง ในX8 X ซีโร่ แอ็กเซล ทั้ง 3 ร่วมมือกันปราบพวก New Generation Reploid- ซีโร่ (ゼロ, Zero) เมื่อซิกม่าก่อสงคราม ช่วย X จากวาว่า แล้วก็บอกถึงเรื่องสถานที่ของอดีตอีเรกูล่าฮันเตอร์ระดับ A ทั้ง 8 แห่งที่ก่อจลาจล ส่วนซีโร่ไปสืบหาที่กบดานของซิกม่า แต่ถูกวาว่าเล่นงานจนบาดเจ็บสาหัส พอ X มาช่วย ซีโร่ระเบิดตัวเองเพื่อฆ่าวาว่าแต่ล้มเหลวจนตาย แต่ก่อนตายได้มอบบัสเตอร์ให้X ใน X2 ชิ้นส่วนของซีโร่ทั้ง 3 ส่วน คือหัว ลำตัว ขา ถูกพวก X Hunter ขโมยไปเพื่อสร้างขึ้นมาใหม่เพื่อสู้กับ X แต่พอ X ช่วยซีโร่ได้ ซีโร่ร่วมมือกับ X กำจัดซิกม่าโดยใช้ Z-Saber เป็นอาวุธประจำตัว แล้วค่อยๆเลิกใช้บัสเตอร์ไป ใน X3 ซีโร่ได้มาร่วมมือต่อสู้ร่วมกับ X ใน X4 ซีโร่ได้หลงรักไอริส แต่โครโน่ไม่ยอมรับ ทั้งคู่สู้กัน จนไอริสต้องมาห้ามกันทุกครั้ง แล้วซีโร่ไปหาวัคซีนรักษาไอริสแต่ก็ฆ่าโครโน่จนไอริสไปรวมตัวกับชิ้นส่วนของโครโน่ จนซีโร่ฆ่าไอริส กับเศร้า ใน X5 ซีโร่ค้นพบว่าตัวเองติดไวรัสจากตอนต่อสู้กับซิกม่าใน X4 ซีโร่จึงปฏิบัติภารกิจ อาสาเป็นนักบินไประเบิดยานขนส่งอวากาศที่เต็มไปด้วย ไวรัสซิกม่า ไประเบิดด้วยตัวเองจนกลับกลายเป็นระเบิดยานไม่ได้และไวรัสที่ฝังอยู่ในชิฟ ได้กระจายออกสู่ร่างกาย ไปท้าสู้กับ X จนซีโร่กลับมาอีกครั้ง แต่ซีโร่ก็ช่วย X เอาไว้จากซิกม่าทำให้ไม่มีใครพบซีโร่อีก แต่ความจริงซีโร่นั้นเจ็บหนักปางตายโดยร่างนั้นก็เหลือเพียงแค่ลำตัวกับแขนข้างเดียวแอบไปซ่อมตัวเองเงียบๆ ใน X6 ปรากฏตัวอีกครั้งหลังจาก X กำจัด Nightmare ซีโร่ มาช่วย X ร่วมมือกำจัดซิกม่าจนสำเร็จ ใน X7 ปฏิบัติภารกิจสืบผู้ซื้อขายไวรัสซิกม่า จนได้พบกับแอ็กเซลที่หนีจากแมชชีนรอยด์ ทั้งคู่ร่วมมือกันกำจัดได้ แล้วแอ็กเซลถูกจับโดยซีโร่ ทางซีโร่ได้ช่วยแก้ต่างให้พ้นผิด แล้วร่วมมือกันในการกำจัดกลุ่มเรดอเลิร์ต กับเป็นรุ่นพี่คอยสอนหน้าที่ของฮันเตอร์ให้แอ็กเซล จน X มาช่วยอีกที ทั้งสามร่วมมือกันกำจัดซิกม่า ใน X8 ทั้ง 3 ร่วมมือกันกำจัดซิกม่า กับคิดเกี่ยวกับเรื่องอนาคต- แอ็กเซล (アクセル, Axl) ตัวจริงของแอ็กเซลนั้น เจ้าตัวก็จำไม่ได้ว่าตนเป็นใครมาจากไหนหรือใครสร้าง แต่ก็มีแนวโน้วว่าผู้สร้างนั้นต้องเป็นนักวิทยาศาสตร์ที่อัจฉริยะระดับพอๆกับไรท์หรือไวลี่ ซึ่งมีเพียงคนเดียวทที่ทำได้คือโคแซค ไม่ก็ลูกหลานเท่านั้น แต่ไม่มีการเปิดเผยว่าจริงหรือไม่ ต่อให้จริงก็คงสร้างมาเพื่อให้เป็นเหมือนไรท์ ม่ก็เพื่อนสัติทางวงการอื่นๆ- เอเลีย (エイリア, Alia) - ซิกม่า (シグマ) ในเรื่องครั้งที่ซิกม่าถูกทำลาย ก็ถูกเคนท์เตอร์ฮันเตอร์(X2) ดร.ดอปเลอร์(X3) X7 ซิกม่าถูกแอ็กเซลยิงตกจากยอดตึกจนร่างกายแหลกไปหมด แต่ถูกลูมินี่รักษากับซ่อมแซมจนมีร่างเป็นคอร์ลแกนกลางที่ควบคุมยึดเหนี่ยวร่างด้วยมวลพลังงาน พอมาถึง X8
