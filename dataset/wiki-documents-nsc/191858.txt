<doc id="191858" url="https://th.wikipedia.org/wiki?curid=191858" title="วิดีโอเกมโอเพนซอร์ซ">วิดีโอเกมโอเพนซอร์ซ เกมโอเพนซอร์ส () คือซอฟต์แวร์เกมที่ถูกพัฒนาและแจกจ่าย ตามสัญญาอนุญาตแจกจายซอฟต์แวร์แบบโอเพนซอร์ส ได้แก่ GNU ,BSD หรือสัญญาอนุญาตแจกจ่ายและแกไขซอฟต์แวร์อื่นๆที่นับว่าเป็นโอเพนซอร์ส ตามที่ Open Source Initiative ได้ระบุไว้ที่มา ที่มา. การที่มีการพัฒนาซอฟต์แวร์แบบโอเพนซอร์สเกิดขึ้น เช่นระบบ ลินุกซ์ ,โอเพนออฟฟิศดอตอ็อก เพื่อตอบสนองความต้องการของผู้ใช้งานโอเพนซอร์สที่ชื่นชอบในการเล่นเกม จึงได้มีการพัฒนาซอฟต์แวร์ เกมเอนจิน เพื่อพัฒนาเกมส์ในแบบโอเพนซอร์ส แล้วนำไปแจกจ่ายให้ผู้ที่สนใจ นำไปใช้ในการพัฒนาเกมส์ ส่วนหนึ่งถูกสร้างเป็นเกมเพื่อการค้า แต่อีกส่วนก็ถูกนำออกมาเป็นเกมเพื่อการแจกจ่ายให้เล่นได้ฟรี ทั้งยังสามารถแก้ไขซอร์สโค้ด เพื่อพัฒนาให้ดียิ่งขึ้นได้ ในลิขสิทธิ์โอเพนซอร์สบางประเภท ยังสามารถนำไปใช้ในทางพาณิชย์ได้อีกด้วยผู้เริ่มต้นการพัฒนาเกมส์โอเพนซอร์ส ผู้เริ่มต้นการพัฒนาเกมส์โอเพนซอร์ส. เดิมทีนั้น ซอฟต์แวร์พัฒนาเกมส์แบบโอเพนซอร์ส จะเป็นการสร้างในรูปแบบของ SDK หรือ Software Development Kit ซึ่งก็เป็นการเพิ่มคำสั่งหรือคลาสให้กับตัว Compiler ของภาษาคอมพิวเตอร์ทั่วๆไป ซึ่งผนวกการใช้ API ที่มีอยู่เช่น DirectX หรือ OpenGL แต่หลังจาก John D. Carmack เกมโปรแกรมเมอร์ผู้พัฒนาเกมสามมิติชื่อดังอย่างเกม Quake และ Doom ได้นำ Game Engine ที่เคยมีอยู่มาเผยแพร่เป็นโอเพนซอร์สภายใต้สัญญาอนุญาตแจกจ่ายแบบ GNU ซึ่งทำให้นักพัฒนาเกมส์ทั่วโลกที่เคยซื้อเกมเอนจินอย่าง Id Tech มาให้ในราคาหลายล้านก็ต้องตกใจกัน เพระราคาสูงแพงหายไปเหลือเพียงของฟรีเท่านั้น แต่ก็ทำให้เกมโอเพนซอร์สได้เพิ่มเทคนิคขึ้นมาอีกหลายอย่าง ทำให้เกิดการนำโอเพนซอร์สไปใช้ในอุตสาหกรรมเกมมากขึ้นซอฟต์แวร์พัฒนาเกมส์โอเพนซอร์ส ซอฟต์แวร์พัฒนาเกมส์โอเพนซอร์ส. ซอฟต์แวร์ในการใช้พัฒนาเกมส์โอเพนซอร์ส แบ่งออกเป็นสามกลุ่ม คือ เกมเอนจิน1.
