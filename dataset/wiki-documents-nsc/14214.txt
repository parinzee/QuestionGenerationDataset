<doc id="14214" url="https://th.wikipedia.org/wiki?curid=14214" title="เว็บเซิร์ฟเวอร์">เว็บเซิร์ฟเวอร์ เว็บเซิร์ฟเวอร์ สามารถมีได้ 2 ความหมาย คือ1. เครื่องคอมพิวเตอร์ซึ่งให้บริการเว็บไซต์ ผู้ใช้เรียกชมหน้าเว็บไซต์ได้โดยใช้โพรโทคอล HTTP ผ่านทางเว็บเบราว์เซอร์ 2. โปรแกรมคอมพิวเตอร์ ที่ทำงานตาม คุณสมบัติข้างต้นการใช้งานการใช้งาน. 1. เมื่อผู้ใช้ป้อนยูอาร์แอล (URL) ในโปรแกรมเว็บเบราว์เซอร์ 2. เครื่องไคลแอนท์จะแปลงชื่อโฮสต์ ภายในยูอาร์แอลเป็นไอพีแอดเดรส 3. เครื่องไคลแอนท์ติดต่อกับเครื่องเว็บเซิร์ฟเวอร์ โดยปรกติจะใช้โพรโทคอล TCP พอร์ต 80 4. เมื่อทำการเชื่อมต่อเสร็จ จะใช้โพรโทคอล HTTP ในการเรียกใช้ข้อมูลที่ต้องการซอฟต์แวร์ ซอฟต์แวร์. ซอฟต์แวร์เว็บเซิร์ฟเวอร์ที่ได้รับความนิยมสูงสุด 4 อันดับแรก คือ- Apache HTTP Server จาก Apache Software Foundation - Internet Information Server (IIS) จากไมโครซอฟท์ - Sun Java System Web Server จากซัน ไมโครซิสเต็มส์ (เดิมชื่อ Sun ONE Web Server, iPlanet Web Server และ Netscape Enterprise Server) - Zeus Web Server จาก Zeus Technology สถิติการใช้งานเว็บเซิร์ฟเวอร์ สามารถดูได้จาก Netcraft Web Server Survey</doc>
