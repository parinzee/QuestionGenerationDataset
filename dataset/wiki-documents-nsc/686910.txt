<doc id="686910" url="https://th.wikipedia.org/wiki?curid=686910" title="รหัสปริศนา Code Hunter">รหัสปริศนา Code Hunter รหัสปริศนา Code Hunter เป็นละครซิทคอม แนวแอ็คชั่น คอเมดี้ สืบสวนสอบสวน ออกอากาศ วันเสาร์-วันอาทิตย์ เวลา 9.00-9.45 น. รวมตอนทั้งหมดทั้งสิ้น 12 ตอน บทประพันธ์โดย พฤกษ์ เอมะรุจิ กำกับการแสดงโดย พฤกษ์ เอมะรุจิ ผลิตโดย บริษัท เซ้นส์ เอ็นเตอร์เทนเมนท์ จำกัด นำแสดงโดย อาทิตย์ ตั้งวิบูลย์พาณิชย์, ซอนญ่า สิงหะ, อนุสรณ์ มณีเทศ, กัญญา รัตนเพชร์ เป็นต้นนักแสดงนักแสดง. - อาทิตย์ ตั้งวิบูลย์พาณิชย์ รับบทเป็น ร.ต.อ. ศรุต - อนุสรณ์ มณีเทศ (โย่ง อาร์มแชร์) รับบทเป็น นิมิตร - ซอนญ่า สิงหะ รับบทเป็น กีรติ - กัญญา รัตนเพชร์ รับบทเป็น หมอพิยะดา - เดวิด อัศวนนท์ รับบทเป็น จ่าแม่น - ธีรภัทร์ แย้มศรี รับบทเป็น หมวดทะนง - สุเมธ องอาจ รับบทเป็น ผู้กำกับศานติ - ประกาศิต โบสุวรรณ รับบทเป็น นาวา - จุ๊บจิ๊บ เชิญยิ้ม รับบทเป็น จ่าเป้ง - อัฒรุต คงราศรี รับบทเป็น จ่าตัน - ธานัท ชินธนาณรงค์ รับบทเป็น หมวดเขี้ยว - ชาลินี บุนนาค รับบทเป็น หมอฤทัย</doc>
