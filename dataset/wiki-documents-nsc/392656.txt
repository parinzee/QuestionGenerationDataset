<doc id="392656" url="https://th.wikipedia.org/wiki?curid=392656" title="Ubuntu Software Center">Ubuntu Software Center โปรแกรมศูนย์ซอฟต์แวร์ Ubuntu (Ubuntu Software Center) (ชื่อดั้งเดิม AppCenter) เป็น โปรแกรมคอมพิวเตอร์ สำหรับค้นหา ติดตั้ง และถอดถอนโปรแกรมบนระบบปฏิบัติการ Ubuntu พัฒนาบนพื้นฐานของ GNOME และเป็นส่วนติดต่อผู้ใช้ของ Advanced Packaging Tool เป็นส่วนหนึ่งของระบบจัดการแพคเกจ Debian Ubuntu Software Center สามารถเพิ่ม repositories โปรแกรมศูนย์ซอฟต์แวร์ Ubuntu เขียนขึ้นด้วยภาษา Python programming language</doc>
