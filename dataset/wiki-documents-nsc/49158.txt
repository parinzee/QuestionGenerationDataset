<doc id="49158" url="https://th.wikipedia.org/wiki?curid=49158" title="ทรงกลม n มิติ">ทรงกลม n มิติ ในทางคณิตศาสตร์ ทรงกลม n มิติ หมายถึงทรงกลมในมิติใดๆ โดยสมการของทรงกลม n มิติ สามารถเขียนได้ในรูปของ formula_1 ดังนี้ปริมาตรของทรงกลม n มิติ ปริมาตรของทรงกลม n มิติ. ปริมาตรของทรงกลม n มิติคือ เมื่อ formula_4 คือ ฟังก์ชันแกมม่า (สำหรับ n ที่เป็นจำนวนคู่ formula_5, สำหรับ n ที่เป็นจำนวนคี่ formula_6 โดยที่ formula_7 หมายถึง ดับเบิลแฟกทอเรียล)ตัวอย่างปริมาตรของทรงกลม n มิติ ตัวอย่างปริมาตรของทรงกลม n มิติ. สำหรับค่า n ที่มีค่าไม่มาก ทรงกลม n มิติที่มีรัศมี 1 หน่วย จะมีปริมาตรดังนี้</doc>
