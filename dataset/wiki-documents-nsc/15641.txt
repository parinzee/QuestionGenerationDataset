<doc id="15641" url="https://th.wikipedia.org/wiki?curid=15641" title="แคสเคดดิงสไตล์ชีตส์">แคสเคดดิงสไตล์ชีตส์ แคสเคดดิงสไตล์ชีตส์ () เป็นภาษาสไตล์ชีตใช้ในการจัดรูปแบบของเอกสารที่เขียนในภาษามาร์กอัป CSS เป็นภาษาที่สำคัญตัวหนึ่งในการเขียนเว็บเพจ ซึ่งเขียนในภาษา HTML และ XHTML แต่ก็ยังสามารถประยุกต์ใช้กับใน XML ซึ่งรวมถึง SVG และ XUL ด้วย มาตรฐาน CSS นั้น สร้างโดยกลุ่ม World Wide Web Consortium (W3C) แคสเคดดิงสไตล์ชีตส์ มักเรียกโดยย่อว่า สไตล์ชีต แต่ความจริงแล้วคำนี้อาจหมายถึงภาษาสไตล์ชีตอื่นก็ได้ระดับของ CSS ระดับของ CSS. ปัจจุบัน CSS แบ่งเป็น 4 รุ่น- CSS 1 ยอมรับเป็นมาตรฐาน ธันวาคม ค.ศ. 1996 - CSS 2 เริ่มร่าง ค.ศ. 1997 ยอมรับเป็นมาตรฐาน พฤษภาคม ค.ศ. 1998 - CSS 3 เริ่มร่าง ค.ศ. 1998 ยอมรับเป็นมาตรฐาน พฤศจิกายน ค.ศ. 2011 - CSS 4 เริ่มร่าง 29 กันยายน ค.ศ. 2009 ปัจจุบันยังไม่มีเว็บเบราว์เซอร์ที่รองรับ CSS 4ตัวอย่าง ตัวอย่าง. ตัวอย่างโค้ดของ CSS</doc>
