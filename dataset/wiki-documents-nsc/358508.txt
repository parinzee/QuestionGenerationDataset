<doc id="358508" url="https://th.wikipedia.org/wiki?curid=358508" title="สวีตพรีเคียว♪">สวีตพรีเคียว♪ สวีตพรีเคียว (Suite Precure) เป็น อะนิเมะ ในซีรีส์ของ พริตตี้เคียว ลำดับที่ 8 ผลิตโดย Toei Animation "สวีตพรีเคียว" เริ่มออกอากาศใน ประเทศไทย ในวันที่ 11 มิถุนายน พ.ศ. 2560 จนถึงวันที่ 9 ธันวาคม พ.ศ. 2560 ทางช่อง โมเดิร์นไนน์ทีวีเนื้อเรื่อง เนื้อเรื่อง. กาลครั้งหนึ่งที่เมเจอร์แลนด์ดินแดนศักดิ์สิทธิ์แห่งดนตรีได้มีคอนเสิร์ตเพื่อที่จะเล่นเมโลดี้แห่งความสุขให้แก่ชาวเมือง ซึ่งจัดขึ้นโดยราชินีแห่งเมเจอร์แลนด์ อะโฟรดิเต้และเธอได้กำหนดให้ภูติแห่งการร้องเพลงที่มีลักษณะคล้ายแมวอย่างฮามี่เป็นดีวาสำหรับการร้องเมโลดี้แห่งความสุขครั้งนี้แต่ทว่าได้มีการบุกรุกจากไมเนอร์แลนด์อย่างกะทันหัน โดยเมฟิสต์ได้เข้ามาในคอนเสิร์ตเพื่อที่จะปั่นป่วนชาวเมืองด้วยการเปลี่ยนสกอร์ทั้งหมดให้กลายเป็นเมโลดี้แห่งความโศกเศร้า ซึ่งร้องโดยดีวาที่ชื่อว่าเซเรน ซึ่งเธอเคยเป็นดีวาของเมเจอร์แลนด์มาก่อนแต่ยังไม่ทราบว่า ทำไมเธอถึงได้ไปร่วมมือกับไมเนอร์แลนด์ เธอเริ่มร้องเมโลดี้ที่ถูกปรับเปลี่ยน ซึ่งทำให้ทุกคนที่เข้าชมคอนเสิร์ตครั้งนี้มีจิตใจโศกเศร้าและทรมานมาก อะโฟรดิเต้จึงได้รวบรวมตัวโน้ตในตำนานทั้งหมดที่อยู่ในสกอร์แล้วขอให้เหล่าตัวโน้ตจงไปซ่อนตัวในสถานที่ต่างๆ ของโลกมนุษย์ เมฟิสต์ไม่ยอมแพ้จึงได้ส่งเซเรนและ 3 สหายไมเนอร์ไปตามหาตัวโน้ต ในขณะเดียวกันอะโฟรดิเต้ได้ส่งฮามี่และแฟรี่โทนไปตามหานักรบในตำนานพริตตี้เคียวเพื่อที่จะให้เหล่านักรบช่วยรวบรวมตัวโน้ตเพื่อปกป้องโลกมนุษย์และเมเจอร์แลนด์จากไมเนอร์แลนด์ให้ได้ตัวละครหลัก ตัวละครหลัก. เคียวเมโลดี้ (Cure Melody) / ซูเปอร์ เคียวเมโลดี้ (Super Cure Melody) / เครสเชนโด เคียวเมโลดี้ (Crescendo Cure Melody) / โฮโจ ฮิบิกิ (Hojo Hibiki) "จงดีดสะบัดท่วงทำนองอันเกรี้ยวกราด เคียวเมโลดี้!" สีประจำตัว : "สีชมพู" ( ), "สีขาว" เด็กสาวผู้ร่าเริง มีความสามารถด้านกีฬาสูง เกิดในครอบครัวนักดนตรี โดยคุณพ่อมีอาชีพเป็นคุณครูสอนดนตรี ส่วนคุณแม่เป็นนักไวโอลินที่มีชื่อเสียง "ท่าไม้ตาย" "Precure Passionato Harmony!" "Precure Suite Session Ensemble Crescendo!" เคียวมิวส์ (Cure Muse) / เครสเชนโด เคียวมิวส์ (Crescendo Cure Muse) / ชิราเบะ อาโกะ (Shirabe Ako) "จงดีดสะบัดท่วงทำนองแห่งเทพธิดา เคียวมิวส์!" สีประจำตัว : "สีดำ Masked Form" ( ), "สีเหลือง ( ), "สีชมพู" ( ) เด็กสาวผู้เงียบขรึมและแสดงสีหน้าเรียบเฉยตลอดเวลาแต่ความจริงเป็นคนร่าเริงสดใส รักเพื่อน เมื่อในยามต่อสู้มักปรากฏตัวในฐานะหน้ากากพริตตี้เคียวปริศนาพร้อมกับแฟรี่โทนโดโดรี่ทุกครั้งและจะสื่อสารกับพริตตี้เคียวคนอื่นๆ ผ่านทางแฟรี่โทนโดโดรี่ทุกครั้ง ในตอนที่ 35 ได้เผยโฉมชุดที่แท้จริงออกมาเพื่อเปิดเผยความจริงและหาทางช่วยเหลือเมฟิสต์ให้ลบล้างพลังด้านมืดออกไป สถานะที่แท้จริงของอาโกะคือเจ้าหญิงแห่งเมเจอร์แลนด์และเหตุที่แสดงสีหน้าที่เคร่งขรึม เนื่องจากปกปิดสถานะตนเองในโลกมนุษย์ "ท่าไม้ตาย" "Precure Shining Circle!" "Precure Suite Session Ensemble Crescendo!"ผู้ช่วยของพรีเคียว ผู้ช่วยของพรีเคียว. ฮามี่ (Hummy) ภูติแห่งเมเจอร์แลนด์ มีลักษณะคล้ายกับแมวขนสีขาว มีความสามารถในการร้องเพลง "เมโลดี้แห่งความสุข" เดินทางมายังโลกมนุษย์เพื่อตามหาและรวบรวม "ตัวโน้ตในตำนาน" และคอยเป็นที่ปรึกษาให้กับพริตตี้เคียว เมื่ออยู่บนโลกมนุษย์ ฮามี่จะสามารถสื่อสารกับแมวบนโลกมนุษย์ได้ แฟรี่โทน (Fairy Tone) เหล่าผู้ช่วยภูติที่ติดตามมาพร้อมกับฮามี่ในการตามหา "ตัวโน้ตในตำนาน" สามารถแปรสภาพเป็นพลังในการแปลงร่างหรือพลังในการโจมตีให้กับพริตตี้เคียวและรวมไปถึงการเก็บตัวโน้ตที่ถูกแปรสภาพเป็นเมเจอร์โทน โดรี่ (Dory) แฟรี่โทนสีชมพู ทำให้ฮิบิกิแปลงร่างเป็นเคียวเมโลดี้และทำให้เคียวเมโลดี้ใช้มิราเคิลเบอร์เทียร์เบลล์โหมดได้ มักพูดว่า "โดโด" เรรี่ (Rery) แฟรี่โทนสีขาว ทำให้คานาเดะแปลงร่างเป็นเคียวริซึ่มและทำให้เคียวริซึ่มใช้ฟานทาสติคเบอร์เทียร์เบลล์โหมดได้ มักพูดว่า "เรเร" มีรี่ (Miry) แฟรี่โทนสีส้ม ทำให้เคียวเมโลดี้ใช้มิราเคิลเบอร์เทียร์สติ๊กโหมดและเบลล์โหมดได้ มักพูดว่า "มีมี" ฟารี่ (Fary) แฟรี่โทนสีเหลือง ทำให้เคียวริซึ่มใช้ฟานทาสติคเบอร์เทียร์สติ๊กโหมดและเบลล์โหมดได้ มักพูดว่า "ฟาฟา" ซอลรี่ (Sory) แฟรี่โทนสีเขียว ทำให้เคียวบีทใช้โซลร็อดได้ มักพูดว่า "ซอลซอล" ลารี่ (Lary) แฟรี่โทนสีฟ้า ทำให้เอเลนแปลงร่างเป็นเคียวบีทและทำให้เคียวบีทใช้ร็อดโหมดได้ มักพูดว่า "ลาลา" ทีรี่ (Tiry) แฟรี่โทนสีน้ำเงิน ทำให้เคียวมิวส์ใช้ท่าไม้ตายได้ มักพูดว่า "ทีที" โดโดรี่ (Dodory) แฟรี่โทนสีม่วง ทำให้อาโกะแปลงร่างเป็นเคียวมิวส์ได้ มักพูดว่า "โดโด" เช่นเดียวกับโดรี่ เครสเชนโดโทน (Crescendo Tone) แฟรี่โทนสีทองที่เพิ่มพลังให้กับพริตตี้เคียวตัวละครอื่นๆ ตัวละครอื่นๆ. โฮโจ ดัน (Hojo Dan) คุณพ่อของฮิบิกิ โฮโจ มาเรีย (Hojo Maria) คุณแม่ของฮิบิกิ มินามิโนะ โซสึเกะ (Minamino Sotsuke) คุณพ่อของคานาเดะ มินามิโนะ มิโซระ (Minamino Misora) คุณแม่ของคานาเดะ มินามิโนะ โซตะ (Minamino Souta) น้องชายของคานาเดะ ชิราเบะ โอโตคิจิ (Shirabe Otokichi) คุณตาของอาโกะ เป็นช่างซ่อมกลไกในเมือง มักจะเคลิบเคลิ้มเมื่อเอ่ยถึงหลานสาวของตนเอง
