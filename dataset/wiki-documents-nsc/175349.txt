<doc id="175349" url="https://th.wikipedia.org/wiki?curid=175349" title="ฟังก์ชันเครื่องหมาย">ฟังก์ชันเครื่องหมาย ในทางคณิตศาสตร์ ฟังก์ชันเครื่องหมาย () คือฟังก์ชันทางคณิตศาสตร์อย่างหนึ่งที่ดึงเครื่องหมายออกมาจากจำนวนจริง เขียนแทนด้วย sgn และเพื่อไม่ให้สับสนกับฟังก์ชันไซน์ (sine) ซึ่งออกเสียงเหมือนกันในภาษาอังกฤษ ฟังก์ชันนี้จึงเรียกอีกชื่อหนึ่งว่า ซินยุม หรือ ซิกนัม (signum) มาจากภาษาละตินนิยาม นิยาม. นิยามของฟังก์ชันเครื่องหมายมีดังนี้ เมื่อ x เป็นจำนวนจริงสมบัติต่างๆ สมบัติต่างๆ. สำหรับจำนวนจริง x ใดๆ สามารถแสดงให้อยู่ในรูปผลคูณระหว่างค่าสัมบูรณ์กับฟังก์ชันเครื่องหมาย จากสมการดังกล่าว เราจะได้ความหมายของฟังก์ชันเครื่องหมายอีกอย่างหนึ่ง เมื่อ x ไม่เท่ากับ 0 ฟังก์ชันเครื่องหมายคืออนุพันธ์ของฟังก์ชันค่าสัมบูรณ์ (ซึ่งประเมินค่าไม่ได้ที่ 0) ฟังก์ชันเครื่องหมายสามารถหาอนุพันธ์ได้ทุกจุดยกเว้นจุด 0 แต่สำหรับการหาอนุพันธ์ในทฤษฎีการกระจาย อนุพันธ์ของฟังก์ชันเครื่องหมายมีค่าเป็นสองเท่าของฟังก์ชันเดลตาของดิแร็ก (Dirac delta function) ฟังก์ชันเครื่องหมายมีความสัมพันธ์กับฟังก์ชันขั้นบันไดของเฮฟวีไซด์ (Heaviside step function) H(x) นั่นคือ เมื่อเลข 1/2 ของฟังก์ชันขั้นบันไดหมายถึง H(0) = 1/2 ฟังก์ชันเครื่องหมายยังสามารถเขียนโดยใช้สัญกรณ์วงเล็บเหลี่ยมของอีเวอร์สัน (Iverson bracket) ดังนี้ สำหรับ k ≫ 0 การประมาณค่าโดยละเอียดของฟังก์ชันขั้นบันไดดังกล่าวหาได้จากฟังก์ชันบนจำนวนเชิงซ้อน ฟังก์ชันบนจำนวนเชิงซ้อน. ฟังก์ชันเครื่องหมายสามารถอธิบายบนจำนวนเชิงซ้อน z ใดๆ ยกเว้น 0 ได้ดังนี้ ซึ่งจะให้ผลลัพธ์เป็นจุดจุดหนึ่งบนวงกลมหนึ่งหน่วยที่อยู่ใกล้กับ z มากที่สุดบนระนาบเชิงซ้อน นั่นคือ โดยที่ arg z คืออาร์กิวเมนต์จำนวนเชิงซ้อนของ z เนื่องจากเหตุผลของความสมมาตร และเพื่อรักษานัยทั่วไปที่สมบูรณ์ของฟังก์ชันเครื่องหมายบนจำนวนจริง ดังนั้นบนจำนวนเชิงซ้อนก็มีการกำหนดให้ sgn&nbsp;0 = 0 ด้วย การวางนัยทั่วไปอีกแบบหนึ่งของฟังก์ชันเครื่องหมายสำหรับทั้งจำนวนจริงและจำนวนเชิงซ้อนคือ csgn ซึ่งนิยามโดย ซึ่งเราจะได้สมบัติดังนี้ (ยกเว้นค่า z = 0)ฟังก์ชันเครื่องหมายแบบนัยทั่วไป ฟังก์ชันเครื่องหมายแบบนัยทั่วไป. ที่จำนวนจริง x เราสามารถสร้างฟังก์ชันเครื่องหมายในรูปแบบของฟังก์ชันนัยทั่วไป (generalized function) คือ formula_13 โดยนิยามให้ formula_14 บนทุกๆ ค่าของ x รวมทั้งจุดที่ x = 0 (ซึ่งต่างกับ sgn คือ formula_15) และถึงแม้ว่าฟังก์ชันนัยทั่วไปนี้สามารถทำให้เกิดพีชคณิตของฟังก์ชันได้ แต่จะเสียสมบัติการสลับที่ไป โดยเฉพาะฟังก์ชันเดลตาของดิแร็กที่เป็นคู่ต่างสลับที่ของฟังก์ชันนี้ นอกจากนั้น formula_17 ไม่สามารถประเมินค่าได้ที่ x = 0 ดังนั้นความหมายของ formula_18 จึงสำคัญที่จะแยกแยะออกจากฟังก์ชัน sgn (นั่นคือ
