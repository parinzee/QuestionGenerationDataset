<doc id="756301" url="https://th.wikipedia.org/wiki?curid=756301" title="นังเหมียวย้อมสี">นังเหมียวย้อมสี นังเหมียวย้อมสี เป็นละครแนวคอมเมดี้-ดราม่า ค่ายโพลีพลัส ออกอากาศวันศุกร์-วันเสาร์-วันอาทิตย์ ในปี พ.ศ. 2545 ทางสถานีโทรทัศน์ไทยทีวีสีช่อง 3 นักแสดงนำ ภัทรพล ศิลปาจารย์, บัวชมพู ฟอร์ด, อภินันท์ ประเสริฐวัฒนกุล, ภัคจีรา วรรณสุทธิ์, สมพล ปิยะพงศ์สิริ และ ซอนย่า คูลลิ่งเรื่องย่อ เรื่องย่อ. พัดยศ วรรณรักษ์ ( ภัทรพล ศิลปาจารย์ ) เพิ่งจะเดินทางกลับมาเมืองไทย หลังจากที่เขาได้สำเร็จการศึกษาจากอเมริกา พัดยศ สนิทกับ คุณหญิงพิมพ์แข ( พิศมัย วิไลศักดิ์ ) ผู้เป็นย่ามาก เนื่องจากท่านเป็นคนหัวสมัยใหม่เวลาพัดยศมี ปัญหาอะไรก็จะปรึกษาคุณย่าตลอด พัดยศได้โทรหาอินทุอร ( สมาพร เขียวบวร ) แฟนสาว แต่ไม่พบจึงชวนเพื่อน ๆ ไปดูที่ดินของตระกูลที่หัวหิน และที่นั่นเขาก็ได้พบกับเหมียว ( บัวชมพู ฟอร์ด ) ที่งานวัดแห่งหนึ่งซึ่งรับจ้างเป็นสาวน้อยตกน้ำอยู่ที่นั่นเพื่อนำเงินมา เลี้ยงยายมุก ( พิมพ์พรรณ บูรณะพิมพ์ ) และแม่เม้า ( วิยะดา อุมารินทร์ ) ขี้เมา พัดยศลองปาไปมั่ว ๆ แต่ถูกเป้า ทำให้เหมียวตกน้ำท่ามกลางเสียงเฮดังสนั่น ระหว่างขับรถกลับบ้านรถของพัดยศเกิดน้ำมันหมด เหมียวนั่งรถผ่านมาพอดีจึงเข้ามาช่วยเหลือ พัดยศขอบคุณโดยการให้เงินเหมียวแต่เหมียวไม่รับ พัดยศรู้สึกผิดจึงกลับไปขอบคุณ และขอโทษโดยยื่นนามบัตรของเขาให้เหมียวแต่เหมียวไม่สนใจ แม่จึงเก็บไว้เอง คืนนั้นยายไม่สบายมากต้องผ่าตัด แต่เหมียวไม่ไม่มีเงินจึงโทรไปขอความช่วยเหลือจากพัดยศ พัดยศเห็นสภาพครอบครัวเหมียวแล้วทำให้เขาอยากจะช่วยเหลือเธอมา คุณหญิงย่าเห็นด้วยจึงชวนเหมียวมาอยู่กรุงเทพฯ โดยคุณหญิงย่าจะอุปการะทุกอย่าง เหมียวเป็นที่ถูกอกถูกใจของคุณหญิงย่ามาก ทำให้ระรินทิพย์ ( อุทุมพร ศิลาพันธ์ ) ชวน ลูกท้อ ( สุวัจนี ไชยมุสิก ) หลานสาวมาเอาใจคุณย่า เพราะกลัว ท่านจะหลงเหมียวจนยกสมบัติให้ อินทุอร โทรมาตัดสัมพันธ์พัดยศทำให้เขาเสียใจมากขับรถเหม่อจนชนจักรยานเหมียวที่สวน มา พัดยศพาเหมียวไปโรงพยาบาล และทั้งคู่ก็นั่งรถออกไปนอกเมืองด้วยกัน ทำให้คนที่บ้านวรรณรักษ์วุ่นวายไปหมด เมื่อเห็นพัดยศ และเหมียวหายไปจนกระทั่งทั้งคู่กลับถึงบ้านเหตุการณ์วุ่นวายทุกอย่างจึงสงบ ลง คุณหญิงพาครอบครัวไปเที่ยวหัวหินเพื่อน ๆ พัดยศขอตามไปด้วย ส่วนทีน่า ( ภัคจีรา วรรณสุทธิ์ ) เพื่อนของพัดยศ คิดหาเรื่องเหมียวโดยแกล้งทำสร้อยข้อมือหายแล้วโยนความผิดให้เหมียว เหมียวไม่โกรธแต่น้อยใจที่พัดยศไม่เชื่อเธอพัดยศรู้สึกผิด และขอโทษเหมียวเมื่อรู้ความจริง พัดยศให้สร้อยคอเหมียวเป็นรางวัลที่เธอสามารถวางตัวอยู่ในสังคมได้เป็นอย่าง ดี ในวันงานสืบสานลูกหลานไฮโซ งานแฟชั่นโชว์ที่คุณหญิงอมรา ( อมรา อัศวนนท์ ) จัดขึ้นเพื่อหารายได้เข้าการกุศล ทีน่า ลูกท้อ รวมทั้งเหมียวก็ได้ร่วมเดินแบบในงานนี้ด้วยแม้ว่า เหมียว จะถูกกลั่นแกล้งต่าง ๆ นานา แต่ เหมียว ก็ผ่านเหตุการณ์ต่าง ๆ มาได้ จนทำให้ เหมียว นั้นมีโอกาสได้เป็นนางเอกละครที่สร้างพล็อตเรื่องจากชีวิตของ เหมียว เอง ซึ่ง เหมียว ก็ได้รับรางวัลดารานำหญิงดีเด่น หลังจากประกาศผลในคืนนั้น เหมียว รีบไปที่บ้าน พัดยศ เพื่อที่จะขอบคุณ แต่เธอก็ต้องรู้ความจริงบางอย่างว่าตัวเธอนั้นเป็นตัวเดิมพัน ของพัดยศและ เพื่อน ๆ โดยการทำให้เหมียวโสโครกตัวนี้เป็นนังเหมียวย้อมสี เหมียวโกรธมากเมื่อรู้ความจริง เธอกระชากสร้อยที่พัดยศให้เขวียงใส่หน้าเขาแล้วจากไปทันที พัดยศ ออกตามหา เหมียว แต่ไม่พบ เขาเสียใจมาก และอยากเจอ เหมียว เพื่อสารภาพรักและขอโทษเธอ ทั้งเขาและเธอจะลงเอยกันอย่างไร เหมียว
