<doc id="205522" url="https://th.wikipedia.org/wiki?curid=205522" title="โครโนสกรุป">โครโนสกรุป โครโนสกรุป () เป็นองค์กรที่ตั้งขึ้นโดยใช้เงินสนับสนุนจากสมาชิกในกลุ่มอุตสาหกรรม เพื่อเป็นตัวกลางในการรวมรวมจัดตั้งคณะทำงานเพื่อสร้างมาตรฐานเปิดสำหรับ API ต่างๆคณะทำงานคณะทำงาน. - OpenGL, API สำหรับประมวลผลคอมพิวเตอร์กราฟิกส์ - OpenCL, ภาษาโปรแกรมเพื่อใช้งาน GPU - COLLADA - glFX - OpenGL SC - OpenKODE - OpenGL ES - OpenVG, API เพื่อสนับสนุนการประมวลผลกราฟิกส์สองมิติ หรือ vector graphics - OpenMAX - OpenSL ES - Embedded Graphics Library - OpenWF - OpenML</doc>
