<doc id="23980" url="https://th.wikipedia.org/wiki?curid=23980" title="สคริปต์ด้านเซิร์ฟเวอร์">สคริปต์ด้านเซิร์ฟเวอร์ สคริปต์ด้านเซิร์ฟเวอร์ () เป็นเทคโนโลยีที่สคริปต์ทำงานบนบนฝั่งเซิร์ฟเวอร์ ซึ่งแตกต่างกับสคริปต์ด้านไคลเอนต์ อย่างJavascriptที่ทำงานบนฝั่งไคลเอนต์ และยังใช้สำหรับสร้างเว็บเพจแบบมีการตอบสนอง (dynamic) ซึ่งอยู่ในรูปแบบเว็บแอปพลิเคชัน ซึ่งมีใช้อยู่แพร่หลายในปัจจุบัน เช่น Web-base อีเมล การทำรายการบัญชีออนไลน์ (online banking) รายงานข่าว หรือ Search Engineรายชื่อเทคโนโลยีที่ใช้สคริปต์ด้านเซิร์ฟเวอร์ รายชื่อเทคโนโลยีที่ใช้สคริปต์ด้านเซิร์ฟเวอร์. ทุกภาษาสามารถสร้างเว็บเว็บเพจได้โดยใช้โพรโทคอล Common Gateway Interface (CGI) รายชื่อนี้นั้นสำหรับเทคโนโลยีที่ออกแบบและรองรับสคริปต์ด้านเซิร์ฟเวอร์โดยเฉพาะ:- ASP - ใช้ภาษาสคริปต์อย่าง VBScript หรือ JScript พัฒนาโดยไมโครซอฟท์ - PHP - ใช้ภาษา PHP ของมันเองในการพัฒนา ซึ่งเป็นโอเพนซอร์ส - JSP - ใช้โค้ด Java ของมันเองในการพัฒนา พัฒนาโดยซัน ไมโครซิสเต็มส์ - Python - ใช้โค้ดภาษา Python พัฒนา</doc>
