<doc id="634629" url="https://th.wikipedia.org/wiki?curid=634629" title="วินโดวส์เมสเซนเจอร์">วินโดวส์เมสเซนเจอร์ วินโดวส์เมสเซนเจอร์ () เป็นโปรแกรมที่ใช้สนทนาแบบเรียลไทม์ระหว่างผู้ใช้ ซึ่งเป็นชุดโปรแกรมที่ถูกรวมไว้ในระบบปฏิบัติการ Windows XP ตัวโปรแกรมถูกออกแบบสำหรับผู้ใช้ในองค์กรธุรกิจมากกว่าผู้ใช้ในบ้าน ต่อมาถูกปรับปรุงในปี ค.ศ. 2001 และถูกปรับปรุงอีกครั้งในปี ค.ศ. 2004 โดยถูกทำให้ใช้งานได้ใน Windows 2000 และ Windows Server 2003 ซึ่งเป็น Windows สำหรับเครื่อง Server สุดท้าย Windows Messenger ถูกพัฒนาขึ้นใหม่และเปลี่ยนชื่อเป็น Windows Live Messenger โดยมีความสามารถใหม่ที่เพิ่มขึ้น นั่นคือมีการเชื่อมต่อกับ Microsoft Messenger service และ Microsoft Lync เพื่อการทำงานในบริษัทที่ดียิ่งขึ้น</doc>
