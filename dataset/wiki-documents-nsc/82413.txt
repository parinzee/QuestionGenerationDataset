<doc id="82413" url="https://th.wikipedia.org/wiki?curid=82413" title="รายนามประธานรัฐสภาไทย">รายนามประธานรัฐสภาไทยรายนามประธานรัฐสภาไทยอดีตประธานรัฐสภาที่ยังมีชีวิตอยู่อดีตประธานรัฐสภาที่ยังมีชีวิตอยู่. - มารุต บุนนาค (เกิด ) - พิชัย รัตตกุล (เกิด ) - ชัย ชิดชอบ (เกิด ) - อุกฤษ มงคลนาวิน (เกิด ) - มีชัย ฤชุพันธุ์ (เกิด ) - อุทัย พิมพ์ใจชน (เกิด ) - วันมูหะมัดนอร์ มะทา (เกิด ) - พรเพชร วิชิตชลชัย (เกิด ) - โภคิน พลกุล (เกิด ) - สมศักดิ์ เกียรติสุรนนท์ (เกิด ) - ยงยุทธ ติยะไพรัช (เกิด )</doc>
