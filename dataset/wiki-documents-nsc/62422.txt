<doc id="62422" url="https://th.wikipedia.org/wiki?curid=62422" title="ไมโครซอฟท์ เอกซ์เพรสชันเว็บ">ไมโครซอฟท์ เอกซ์เพรสชันเว็บ ไมโครซอฟท์ เอกซ์เพรสชันเว็บ () คือ HTML editor สำหรับออกแบบเว็บเพจพัฒนาขึ้นโดยไมโครซอฟท์ โดยเป็นส่วนหนึ่งของ Microsoft Expression ซึ่งจะมาแทนที่ Frontpage ที่ได้หยุดพัฒนาไปแล้ว เอกซ์เพรสชันเว็บนั้นได้ถูกพัฒนาขึ้นมาใหม่ ซึ่งมีเป้าหมายสำหรับผู้ออกแบบเว็บเพจในการพัฒนาเว็บไซต์ที่มีคุณภาพสูงและตามมาตรฐานเว็บสากล โดย Expression Web ได้รองรับ XML CSS 2.1 ASP.NET 2.0 XHTML จาวาสคริปต์ และเทคโนโลยีอื่นๆ สำหรับการเขียนเว็บแบบไดนามิค และเข้าถึงได้ เอกซ์เพรสชันเว็บได้ออกรุ่นแรกวันที่ 4 ธันวาคม พ.ศ. 2549</doc>
