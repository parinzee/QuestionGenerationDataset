<doc id="730583" url="https://th.wikipedia.org/wiki?curid=730583" title="เขตควังจิน">เขตควังจิน เขตควังจิน (; ฮันกึล: ; ฮันจา: ) เป็นหนึ่งใน 25 เขตที่อยู่ในกรุงโซล ประเทศเกาหลีใต้ ตั้งอยู่ทางด้านทิศเหนือของแม่น้ำฮัน เขตนี้มีพื้นที่ทั้งหมด 23.92 กม. และมีประชากรทั้งสิ้น 368,021 คน เขตควังจินก่อตั้งขึ้นในปี ค.ศ. 1995 โดยแยกออกจากเขตซ็องดง เขตควังจินเป็นที่ตั้งของมหาวิทยาลัยค็อนกุกและมหาวิทยาลัยเซจงเขตการปกครองเขตการปกครอง. - แขวงควังจัง (ฮันกึล: ; ฮันจา: ) - แขวงคุนจา (ฮันกึล: ; ฮันจา: ) - แขวงคูอึย (ฮันกึล: ; ฮันจา: ) - แขวงฮวายัง (ฮันกึล: ; ฮันจา: ) - แขวงโมจิน (ฮันกึล: ; ฮันจา: ) - แขวงชายัง (ฮันกึล: ; ฮันจา: ) - แขวงชุงกก (ฮันกึล: ; ฮันจา: ) - แขวงนึง (ฮันกึล: ; ฮันจา: ) - แขวงโนยู (ฮันกึล: ; ฮันจา: )การคมนาคมทางรถไฟการคมนาคม. ทางรถไฟ. - โซล เมโทร- รถไฟใต้ดินโซล สาย 2- (เขตซ็องดง) ← มหาวิทยาลัยค็อนกุก — คูอึย — คังบย็อน → (เขตซงพา)  - องค์การรถไฟฟ้ามหานครโซลคอร์พอเรชัน- รถไฟใต้ดินโซล สาย 5- (เขตทงแดมุน) ← คุนจา — อาชาซัน — ควังนารู → (เขตคังดง) - รถไฟใต้ดินโซล สาย 7- (เขตชุงนัง) ← ชุงกก — คุนจา — ชิลเดรนส์แกรนด์พาร์ก — มหาวิทยาลัยค็อนกุก — ตุกซ็อมริซอร์ต → (เขตคังนัม)เมืองพี่เมืองน้องเมืองพี่เมืองน้อง. - โพอึน, ประเทศเกาหลีใต้ - โพรย็อง, ประเทศเกาหลีใต้ - เอเรลี, ประเทศตุรกี - ฝังชาน, ประเทศจีน - อินเจ, ประเทศเกาหลีใต้ - มุนกย็อง, ประเทศเกาหลีใต้ - ย็องกวัง, ประเทศเกาหลีใต้</doc>
