<doc id="29277" url="https://th.wikipedia.org/wiki?curid=29277" title="ด็อม">ด็อม ด็อม ( แบบจำลองอ็อบเจกต์เอกสาร) เป็นวิธีการอธิบายว่าข้อมูลต่างๆ ในเอกสาร HTML หรือ XML จัดเรียงตัวแบบเชิงวัตถุ (object oriented) อย่างไร ด็อมเป็นวิธีในการสร้าง API ให้สามารถควบคุมเนื้อหา โครงสร้าง และรูปแบบของเอกสารได้ เดิมทีนั้นเว็บเบราว์เซอร์แต่ละค่ายมีการพัฒนาด็อมโดยใช้จาวาสคริปต์ที่แตกต่างกันออกไป ทาง W3C จึงได้ร่างมาตรฐานกลางที่เรียกว่า W3C Document Object Model (W3C DOM) โดยไม่ขึ้นกับภาษาโปรแกรมมิ่งภาษาใดภาษาหนึ่ง การจัดข้อมูลในรูปแบบด็อมนั้นใช้แผนภูมิต้นไม้ในการบอกความสัมพันธ์ระหว่างวัตถุแต่ละชิ้น อย่างไรก็ตาม ด็อมมีจุดอ่อนในเรื่องประสิทธิภาพเมื่อต้องประมวลผลเอกสารขนาดใหญ่ ซึ่งด็อมจำเป็นต้องอ่านเอกสารให้ครบทั้งหมดก่อน จึงจะเริ่มประมวลผล ข้อเสียนี้จึงเกิดการออกแบบ SAX ที่ใช้วิธีอ่านเอกสารทีละส่วนแทน ตามมาตรฐานของ W3C นั้น ได้แบ่งด็อมเป็นระดับชั้น (level) ที่มีข้อกำหนดแตกต่างกันออกไป ปัจจุบันมี Level 0-3</doc>
