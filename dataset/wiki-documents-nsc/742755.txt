<doc id="742755" url="https://th.wikipedia.org/wiki?curid=742755" title="สถานีรถไฟนาพู่">สถานีรถไฟนาพู่ สถานีรถไฟนาพู่ เป็นสถานีรถไฟชั้น 3 ตั้งอยู่ที่บ้านนาพู่ ตำบลนาพู่ อำเภอเพ็ญ จังหวัดอุดรธานี อยู่ห่างจากสถานีรถไฟกรุงเทพเป็นระยะทาง 580 กิโลเมตรตารางเวลาการเดินรถเที่ยวขึ้นเที่ยวล่องข้อมูลจำเพาะตารางเวลาการเดินรถ. ข้อมูลจำเพาะ. - รหัส : 2201 - ชื่อภาษาไทย : นาพู่ - ชื่อภาษาอังกฤษ : Na Phu - ชื่อย่อภาษาไทย : นพ. - ชื่อย่อภาษาอังกฤษ : NP. - ชั้นสถานี : สถานีรถไฟชั้น 3 - ระบบอาณัติสัญญาณ : ระบบอาณัติสัญญาณป้ายเขตสถานี - พิกัดที่ตั้ง : - ที่อยู่ :ตำบลนาพู่ อำเภอเพ็ญ จังหวัดอุดรธานี - ขบวนรถ/วัน : 4ขบวน ขบวนรถเร็ว ขบวนรถธรรมดา ขบวนรถชานเมือง - ปริมาณผู้โดยสาร/วัน : - นายสถานี : นายสุเทพ พิทักษ์กุล - ผู้ช่วยนายสถานี : - ผู้ช่วยนายสถานี :' - ผู้ช่วยนายสถานี : - สถานีก่อนหน้า(ที่ไม่ใช่ป้ายหยุดรถ) : สถานีรถไฟอุดรธานี - สถานีถัดไป(ที่ไม่ใช่ป้ายหยุดรถ) : สถานีรถไฟนาทา - ห่างจากสถานีกรุงเทพ : 593.00 กิโลเมตร</doc>
