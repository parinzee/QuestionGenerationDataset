<doc id="102214" url="https://th.wikipedia.org/wiki?curid=102214" title="Multilayer Switch">Multilayer Switch Multilayer Switch อ้างถึงอุปกรณ์ Switch ที่ทำงานในหลายๆ Layer ตั้งแต่ Data Link Layer (Layer 2) และ Network Layer (Layer 3) ขึ้นไป โดยในบางอุปกรณ์นั้นสามารถทำงานได้ถึง Application Layer (Layer 7) ด้วย เช่นอุปกรณ์ Load Balancer โดยทั่วไปแล้วอุปกรณ์ Multilayer Switch ที่นิยมใช้กันอย่างแพร่หลายจะทำงานที่ Data Link Layer และ Network Layer ดังนั้นอุปกรณ์ดังกล่าวจะมีฟังก์ชันการทำงานที่สำคัญอันได้แก่ VLAN Assignment และ Routing ManagementVLAN Assignment VLAN Assignment. เป็นการกำหนด VLAN ให้กับ Interface ต่างๆของ Multilayer Switch เพื่อให้สามารถรับและส่งข้อมูลไปยังคอมพิวเตอร์ปลายทางภายในเครือข่ายเดียวกัน (Same Network) ได้อย่างถูกต้อง รวมไปถึงการกำหนด parameter ต่างๆ (เช่น Spanning Tree เป็นต้น) ของ VLAN เพื่อให้ Multilayer Switch สามารถทำงานร่วมกับอุปกรณ์ Switch อื่นๆที่เชื่อมต่อเข้ามาRouting Management Routing Management. เพื่อให้ Multilayer Switch สามารถที่จะรับและส่งข้อมูลไปยังคอมพิวเตอร์ปลายทางที่อยู่ต่างเครือข่าย (Different Network) และทำงานร่วมกับอุปกรณ์ในระดับ Network Layer ที่เชื่อมต่อเข้ามาได้อย่างถูกต้อง</doc>
